# 上传回单文件

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/receipt-file/upload 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/receipt-file/upload 
 ## 接口描述 
支付回单文件上传，并返回文件唯一编号，可结合绑定回单文件接口使用。
上传附件单个大小不超过10M。
只允许上传文件类型为PNG/JPG/JPEG/OFD/PDF的文件。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json

## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
