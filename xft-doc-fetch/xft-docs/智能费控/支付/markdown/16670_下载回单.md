# 下载回单

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/receipt-file/download 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/receipt-file/download 
 ## 接口描述 
根据回单文件名称下载回单文件。
回单文件名称fileName对应回单文件查询接口返回值data中fileName值。
返回文件内容：ResponseEntity<byte[]>(data, headers, HttpStatus.OK)；data为文件字节数组，即byte[] bytes = response.body().bytes()就能拿到文件。 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json
