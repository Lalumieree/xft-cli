# 分页查询支付明细

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/pay-detail/page-query 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/pay-detail/page-query 
 ## 接口描述 
通过单据编号，支付状态，支付场景（对公，对私），支付完成时间范围，发起支付时间范围，分页查询费控支付工作台的支付明细数据，主要包括明细状态、金额、支付完成时间（如果已经支付成功）和收、付款账户以及支付发送时间、支付类型和拓展字段等信息 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
