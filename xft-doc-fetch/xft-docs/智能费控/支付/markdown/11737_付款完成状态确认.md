# 付款完成状态确认

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/oboPayConfirm 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/oboPayConfirm 
 ## 接口描述 
对外部支付成功或支付失败的单据支付明细，可以通过回调进行付款成功或失败的状态确认。
- 如果是单据编号，则会把单据下的所有待支付的明细进行付款状态变更；
- 如果是明细号，则会把此付款明细进行付款状态变更。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json

## 接口调用示例 
 ### 请求示例
{BaseURL}/api/itrip-gateway/v1/bills/oboPayConfirm?OPAUID=XFT00001
```json
{
    "billId": 2022000000000000,
    "payFinishTime": "2022-10-10 12:23:12",
    "empId":"U0000",
    "empNumber":"A1234",
    "empPhone":"18888888888"
    "operationFlag":"SUCCESS"
}
```
### 返回示例
```json
{
    "data": {
        "total": 1,
        "successCount": 1,
        "detail": [
            {
                "oboSeqId": 1,
                "billId": 2022000000000000
                "payIndex": 1,
                "success": true,
                "message": "确认成功"
            }
        ]
    },
    "success": true,
    "message": "成功"
}
```
