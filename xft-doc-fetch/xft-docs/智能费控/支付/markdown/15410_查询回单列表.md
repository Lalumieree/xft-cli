# 查询回单列表

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/receipt-file/query 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/receipt-file/query 
 ## 接口描述 
回单列表查询。
支付回单查询单据id数量不可超过50。
下载回单使用的fileName为返回值列表data中的fileName值。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
