下载回单

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/receipt-file/download

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/receipt-file/download

接口描述

根据回单文件名称下载回单文件。

回单文件名称fileName对应回单文件查询接口返回值data中fileName值。

返回文件内容：ResponseEntity<byte[]>(data, headers, HttpStatus.OK)；data为文件字节数组，即byte[] bytes = response.body().bytes()就能拿到文件。

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
参数类型必填说明示例
rootOBJECTquery根节点
root.fileNameSTRING是对应回单文件查询接口返回值data中fileName值97845d94-fd50-4581-b4fd-a06052

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.codeSTRING下载回单文件编码200
root.messageSTRING错误提示下载回单失败

接口调用示例

请求调用


{BaseUrl}/v1/receipt-file/download?fileName=97845d94-fd50-4581-b4fd-a06052

响应报文


{
    "code": 400,
    "message": "{\"returnCode\":\"XFFB000\",\"errorMsg\":\"未查询到相应的回单文件，请确认后重试\",\"body\":null}"
}

错误码说明

错误码错误信息解决方案
400下载回单失败
