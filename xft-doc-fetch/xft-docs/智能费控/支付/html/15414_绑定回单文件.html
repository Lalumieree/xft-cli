绑定回单文件

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/receipt-file/bind

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/receipt-file/bind

接口描述

回单文件绑定接口，传入文件唯一编号，可结合回单文件上传接口使用。

支持传入billId+payIndex 或者 只传 oboSeqId，作为明细的匹配值,

billId+payIndex 和 oboSeqId 都传时，明细匹配值要保持一致。

最多支持传入100条回单文件key。

多次调用绑定接口，回单文件绑定为累加，原有绑定的回单文件有效。

回单文件绑定时，无效文件key会自动清除，只保留有效文件key。

支付明细已绑定的文件key再次进行绑定不进行重复绑定。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.billIdLONG支持传入billId+payIndex 或者 只传 oboSeqId，作为明细的匹配值2025031804139
root.payIndexINTEGER支持传入billId+payIndex 或者 只传 oboSeqId，作为明细的匹配值1
root.oboSeqIdLONG支持传入billId+payIndex 或者 只传 oboSeqId，作为明细的匹配值1270
root.receiptListARRAY是最多支持传入100条回单文件key[ "903e38be-01c7-4419-992d-18092db585b0" ]
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN是是否成功true
root.errorMessageSTRING错误信息未找到对应的支付信息，请确认后重试
