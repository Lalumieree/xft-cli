查询回单列表

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/receipt-file/query

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/receipt-file/query

接口描述

回单列表查询。

支付回单查询单据id数量不可超过50。

下载回单使用的fileName为返回值列表data中的fileName值。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.billIdListARRAY是支付回单查询单据id数量不可超过50[ "2025041404177979" ]
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataARRAY是返回数据
root.data.seqIdLONG回单自增ID8618
root.data.billIdLONG单据ID2025041404177979
root.data.oboSeqIdLONG支付明细自增ID129789
root.data.fileNameSTRING文件名称不带后缀，uuid32e177f4-2d4d-4d3a-8745-55b076bcea25
root.data.fileShowNameSTRING文件原始名称单据下载流程.png
root.data.receiptBusinessNumberSTRING回单业务参考号2025041404177979-1
root.data.receiptNumberSTRING回单编号991B720015684
root.data.receiptAccountNumberSTRING回单付款账号755937155610003
root.data.bankTypeSTRING银行类型CMB
root.data.transactionDateSTRING交易日期2025-01-22
root.data.bankTransactionNumberSTRING银行流水号C0446VC00008JOZ
root.data.checkCodeSTRING对账码DZM202411210000004069
root.data.amountBIGDECIMAL整数15位，小数两位1.00
root.data.sourceFlagSTRINGMANUAL-手动上传，SYNC-同步MANUAL
root.successBOOLEAN是是否成功true
root.messageSTRING否返回报文调用成功
