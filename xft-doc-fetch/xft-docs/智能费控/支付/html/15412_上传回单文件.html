上传回单文件

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/receipt-file/upload

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/receipt-file/upload

接口描述

支付回单文件上传，并返回文件唯一编号，可结合绑定回单文件接口使用。

上传附件单个大小不超过10M。

只允许上传文件类型为PNG/JPG/JPEG/OFD/PDF的文件。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT否请求体根节点
root.fileFILE是文件大小不得超过10M，只允许上传文件类型为PNG/JPG/JPEG/OFD/PDF的文件
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN是回单文件上传是否成功true
root.errorMessageSTRING否错误信息"只允许上传文件类型为PNG/JPG/JPEG/OFD/PDF的文件"
root.fileIdSTRING否回单文件上传Keyd05c0eb4-efcd-4616-a4da-718cc2fe9732

接口调用示例

请求报文

{BaseURL}/v1/receipt-file/upload

注意文件上传，参数file

响应报文


{
    "success": true,
    "errorMessage": null,
    "fileId": "d05c0eb4-efcd-4616-a4da-718cc2fe9732"
}
