# 新建差旅申请单（草稿、审批通过、审批中）

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/apply/travel/create 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/apply/travel/create 
 ## 接口描述 
支持调用此接口新增草稿、审批通过的差旅申请单

自定义字段传值格式和说明如下：
