# 新建用车申请单（草稿、审批通过、审批中）

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/apply/use-car/create 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/apply/use-car/create 
 ## 接口描述 
支持系统调用接口新增草稿、审批通过的用车申请单

## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
