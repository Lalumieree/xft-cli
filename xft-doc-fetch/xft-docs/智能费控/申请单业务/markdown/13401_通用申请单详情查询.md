# 通用申请单详情查询

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/apply/common/detail 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/apply/common/detail 
 ## 接口描述 
支持调用此接口查询单个通用申请单的单据详情
