# 批量作废申请单

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/apply/batch-discard 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/apply/batch-discard 
 ## 接口描述 
**批量根据单据号作废申请单接口**
接口支持部分成功，返回成功的单据编号集合和失败的单据编号集合
**允许作废单据 ：**
1、通用申请单
2、差旅申请单
3、费用申请单
4、招待申请单
5、用车申请单
 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.returnCode | STRING | 是 | 业务执行返回码 | SUC0000 |
| root.success | BOOLEAN | 是 | 业务是否执行成功 | true |
| root.message | STRING | 否 | 系统提示信息 | 成功 |
| root.body | OBJECT | 否 | 返回参数体 |  |
| root.body.successBillIdList | ARRAY | 否 | 执行作废成功的单据编号集合 | [2025010100000001] |
| root.body.failedBillList | ARRAY | 否 | 执行作废失败的单据编号和原因集合 |  |
| root.body.failedBillList.billId | LONG | 是 | 单据编号 | 2025010100000002 |
| root.body.failedBillList.message | STRING | 是 | 具体文本可展示，如果触发了多个校验，会拼接在一起 | 此申请单已产生订单，并且没有退票，不允许废弃！ |
 
## 接口调用示例 
 ### 请求报文
```json
{
  "applyBillDiscardItemList": [
    {
      "billId": 2025010100000001
    },
    {
      "billId": 2025010100000002
    }
  ]
}
```
### 响应报文
```json
{
  "returnCode": "SUC0000",
  "success": true,
  "message": "成功",
  "body": {
    "successBillIdList": [
      2025010100000001,
      2025010100000002
    ]
  }
}
```
### 部分失败响应报文
```json
{
  "returnCode": "SUC0000",
  "success": true,
  "message": "成功",
  "body": {
    "successBillIdList": [
      2025010100000001
    ],
    "failedBillList": [
      {
        "billId": 2025010100000002,
        "message": "此申请单已产生订单，并且没有退票，不允许废弃！"
      }
    ]
  }
}
```
