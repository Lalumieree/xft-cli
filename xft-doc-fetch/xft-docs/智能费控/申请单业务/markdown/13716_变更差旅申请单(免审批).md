# 变更差旅申请单(免审批)

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/apply/travel/change 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/apply/travel/change 
 ## 接口描述 
支持调用此接口变更审批通过的差旅申请单
**请注意：api变更不受表单配置->单据管控中的允许变更申请单和编辑不可变更字段配置管控**
自定义字段编码取值来源: 单据设置中的自定义字段编码

自定义字段传值格式和说明如下：

| 数字 | 小数位和整数位以自定义字段配置为准，value以字符串格式传入 | 8.6 |
