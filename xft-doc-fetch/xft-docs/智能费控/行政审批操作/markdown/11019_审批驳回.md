# 审批驳回

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/approve/reject 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/approve/reject 
 ## 接口描述 
审批驳回 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.businessList | ARRAY | 否 | 单次最多处理50条审批信息 |  |
| body.businessList.approveCategory | STRING | 否 | 目前支持单据(BILL)和财务凭证(FNC_CERTIFICATE),默认不传为单据 | BILL |
| body.businessList.approveOpinion | STRING | 否 | 该值根据审批节点的审批意见配置进行校验<br>当审批意见配置为：不必填时，该值不必填，其他配置都为必填 | OK |
| body.businessList.businessId | LONG | 否 | 单据ID/流程编号 | 2025021047250831 |
| body.empId | STRING | 是 | 发起审批操作人的员工编号 | U0000 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | 薪福通企业号 | XFT00001 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.error | STRING | 否 | 记录失败的数据, 值为key:错误原因。当数据整体错误时,返回 errorMessage:错误原因 | null |
| body.success | BOOLEAN | 否 | 执行结果布尔值:<br>成功为true;有一条失败则为false | true |
| body.successKey | ARRAY | 否 | 成功的数据会返回成功的唯一标识key | [2022062101309617] |
 
## 接口调用示例 
 ###请求示例
` {BaseURL}/api/v1/approve/reject?OPAUID=XFT00001
```json
{
  "businessList": [
    {
      "approveCategory": "BILL",
      "approveOpinion": "拒绝",
      "businessId": 2025021047250831
    }
  ],
  "empId": "U0000"
}
```
### 返回示例
```json
{
	"error": null,
	"success": true,
	"successKey": [2025021047250831]
}
```
