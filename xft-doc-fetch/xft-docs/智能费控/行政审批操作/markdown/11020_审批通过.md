# 审批通过

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/approve/pass 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/approve/pass 
 ## 接口描述 
单据审批通过 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.businessList | ARRAY | 是 | 单据/流程总业务信息 |  |
| body.businessList.approveCategory | STRING | 否 | 目前支持单据(BILL)和财务凭证(FNC_CERTIFICATE),默认不传为单据 | BILL |
| body.businessList.approveOpinion | STRING | 否 | 根据审批节点审批意见配置进行校验，当审批意见配置为所有操作必填时，该值必填 | OK |
| body.businessList.businessId | LONG | 是 | 单据ID/流程编号 | 2025021047250831 |
| body.empId | STRING | 是 | 薪福通企业用户号可以通过智能费控-系统设置-档案导出页面，导出员工信息，其中"员工号"即为薪福通企业用户号 | U0000 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | 薪福通企业号 | XFT00001 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.error | STRING | 否 | 记录失败的数据, 值为key:错误原因。当数据整体错误时,返回 errorMessage:错误原因 | null |
| body.success | STRING | 否 | 执行结果布尔值:<br>成功为true;有一条失败则为false | true |
| body.successKey | ARRAY | 否 | 成功的数据会返回成功的唯一标识key | [2022062101309617] |
 
## 接口调用示例 
 ### 请求示例
{BaseURL}/api/v1/approve/pass?OPAUID=XFT00001
```json
{
    "businessList": [
        {
            "approveCategory": "BILL",
            "approveOpinion": "同意",
            "businessId": 2025021047250831
        }
    ],
    "empId": "U0000"
}
```
### 返回示例
```json
{
    "error": "",
    "success": true,
    "successKey": [
        2025021047250831
    ]
}
```
