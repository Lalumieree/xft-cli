审批通过

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/approve/pass

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/approve/pass

接口描述

单据审批通过

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.businessListARRAY是单据/流程总业务信息
body.businessList.approveCategorySTRING否目前支持单据(BILL)和财务凭证(FNC_CERTIFICATE),默认不传为单据BILL
body.businessList.approveOpinionSTRING否根据审批节点审批意见配置进行校验，当审批意见配置为所有操作必填时，该值必填OK
body.businessList.businessIdLONG是单据ID/流程编号2025021047250831
body.empIdSTRING是薪福通企业用户号可以通过智能费控-系统设置-档案导出页面，导出员工信息，其中"员工号"即为薪福通企业用户号U0000
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是薪福通企业号XFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.errorSTRING否记录失败的数据, 值为key:错误原因。当数据整体错误时,返回 errorMessage:错误原因null
body.successSTRING否执行结果布尔值:
成功为true;有一条失败则为false		true
body.successKeyARRAY否成功的数据会返回成功的唯一标识key[2022062101309617]

{BaseURL}/api/v1/approve/pass?OPAUID=XFT00001
