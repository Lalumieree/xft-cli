审批退回

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/approve/back

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/approve/back

接口描述

单据审批退回

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.empIdSTRING是员工号信息可以通过智能费控-系统设置-档案导出，导出员工，其中的员工号进行对应U0000
body.businessListARRAY是审批退回目前仅支持单个处理
body.businessList.businessIdLONG是单据id不允许重复2025021047250831
body.businessList.approveCategorySTRING否目前支持单据(BILL),默认不传为单据BILL
body.businessList.approveOpinionSTRING否根据审批节点的审批意见是否必填配置进行校验
审批意见配置为：所有操作必填或者驳回退回必填时，该值必填，否则非必填		退回
body.businessList.backTypeFlagSTRING0：退回后无需走已审批节点，1：需重走全部节点
默认：0		0
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是薪福通企业号XFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.successBOOLEAN否执行结果布尔值:
成功为true;有一条失败则为false		true
body.successKeyARRAY否执行结果布尔值:
成功为true;有一条失败则为false		[2022062101309617]
body.errorSTRING否记录失败的数据, 值为key:错误原因。当数据整体错误时,返回 errorMessage:错误原因null

接口调用示例

###请求示例

{BaseURL}/api/itrip-gateway/v1/approve/back?OPAUID=XFT00001


{
  "empId": "U0000",
  "businessList":[
  	{
		"businessId": 2022062101309617,
		"approveCategory": "BILL",
		"approveOpinion": "审批通过"
  	}
  ]
}

返回示例


{ 
	   "success":  true,
	   "successKey": [2022062101309617]
	   "error":  null
}
