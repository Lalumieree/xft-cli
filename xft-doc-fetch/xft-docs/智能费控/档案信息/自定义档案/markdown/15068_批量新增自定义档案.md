# 批量新增自定义档案

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/custom-file/batch-add 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/custom-file/batch-add 
 ## 接口描述 
批量新增自定义档案接口

接口所属功能路径：
智能费控 - 系统设置 - 档案设置 - 自定义档案
<img src="https://s3gw.cmbchina.com/lw36.01-public-prd/public-cbdf247a-d330-4e9a-b2ba-8aae14064ba4" alt="示例图片" width="300" height="200">

接口调用注意事项：
* 接口调用单次最大支持50条；
* 接口不支持部分成功，单次接口调用全量成功/失败；
 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | ARRAY |  | 请求体根节点 |  |
| root.customFileId | STRING | 是 | 档案编码 | CF100000 |
| root.customFileName | STRING | 是 | 档案名称 | 自定义档案 |
| root.status | STRING | 是 | * 1-启用<br>* 0-停用 | "1" |
| root.customFileType | STRING | 是 | * MULTI_LEVEL-多层级<br>* SINGLE_LEVEL-无层级 | MULTI_LEVEL |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.success | BOOLEAN | 是 | true-成功 | true |
| root.message | STRING | 是 | 接口调用信息 | 调用成功！ |
| root.returnCode | STRING | 是 | 接口调用结果描述信息 |  |
 
## 接口调用示例 
 ### 请求
```json
[
    {
        "customFileId": "CF10000",
        "customFileName": "档案1号",
        "status": "1",
        "customFileType": "MULTI_LEVEL"
    },
    {
        "customFileId": "CF20000",
        "customFileName": "档案2号",
        "status": "1",
        "customFileType": "SINGLE_LEVEL"
    }
]
```
### 返回
  
```json
{  
    "returnCode": "SUC0000",  
    "success": true,  
    "message": "成功"  
}  
```
```json
{  
    "returnCode": "EROOR1",  
    "success": false,  
    "message": "不存在的档案编码【CUS1000】请核对后重试"  
}  
```   
## 错误码说明 
| 错误码| 错误信息|解决方案|
  | --- | --- | --- |
  | SUC0000| 成功| 成功|
  | EROOR1| 系统异常| 系统错误，联系管理员处理|
  | EROOR2| 业务执行错误| 业务执行错误按提示修改入参后重试|
