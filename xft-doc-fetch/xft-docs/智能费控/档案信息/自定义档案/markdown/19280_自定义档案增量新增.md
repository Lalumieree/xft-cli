# 自定义档案增量新增

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/custom-file/incr-add 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/custom-file/incr-add 
 ## 接口描述 
增量新增自定义档案选项接口，通过档案编码定位档案位置，通过optionId来判断选项的层级，以及新增的选项信息来实现增量新增自定义档案选项信息，单多层级选项总数均不可超过20000，多层级每层选项总数量不可超过1000。
接口所属功能路径：
智能费控 - 系统设置 - 档案设置 - 自定义档案 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT | 是 | 请求体根节点 |  |
| root.customFileId | STRING | 是 | 不可为空 | code20250422 |
| root.options | ARRAY | 是 | 该值为增量新增的自定义档案选项信息，单多层级选项总数均不可超过20000，多层级每层选项总数量不可超过1000。 |  |
| root.options.optionId | STRING | 否 | 1，定义层级为哪一层（下方newOptionId，newOptionName，serialNumber，applyCompanies，status等信息为本optionId子节点信息）；<br> 2，单层级时可以不传optionId，多层级时optionId不填就默认层级为1。 | 4135 |
| root.options.newOptionId | STRING | 是 | 必填 | 4136 |
| root.options.newOptionName | STRING | 是 | 必填 | 选项名称4136 |
| root.options.serialNumber | INTEGER |  | 默认值为0，最大值为20000 | 0 |
| root.options.applyCompanies | STRING |  | 多个适用公司编码之间用,分隔 | 73299 |
| root.options.status | STRING |  | 1：已启用；0，已禁用。 | 1 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
