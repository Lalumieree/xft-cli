# 查询自定义档案详情

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/customFile/detail 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/customFile/detail 
 ## 接口描述 
获取自定义档案详情 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.id | LONG | 是 | 档案编码 | 10 |
