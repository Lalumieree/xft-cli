# 更新自定义档案

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/customFile/update 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/customFile/update 
 ## 接口描述 
更新自定义档案 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
