# 获取档案列表

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/customFile/list 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/customFile/list 
 ## 接口描述 
获取自定义档案列表，分页接口，一次最多查询100条 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.customFileId | STRING | 否 | 自定义档案编码 |  |
| body.customFileName | STRING | 否 | 自定义档案名称 |  |
| body.status | STRING | 否 | 0-关闭，1-开启 | 1 |
| body.start | INTEGER | 是 | 最大不能超过100条 | 0 |
| body.limit | INTEGER | 是 | 最大不能超过100条 | 10 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
