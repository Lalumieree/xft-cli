# 更新自定义档案状态

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/customFile/updateStatus 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/customFile/updateStatus 
 ## 接口描述 
更新自定义档案的状态，0-关闭，1-开启 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.status | STRING | 是 | 0-关闭，1-开启 | 0 |
| query.customFileId | STRING | 是 | 自定义档案编码 | CUS0000 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.success | BOOLEAN | 否 | true-成功，false-失败 | true |
| body.message | STRING | 否 | 调用信息 | 调用成功！ |
 
## 接口调用示例 
 {BaseURL}/v1/customFile/updateStatus?status=string&customFileId=CUS0000
