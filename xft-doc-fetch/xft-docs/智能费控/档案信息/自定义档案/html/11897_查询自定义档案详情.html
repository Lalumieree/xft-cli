查询自定义档案详情

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/customFile/detail

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/customFile/detail

接口描述

获取自定义档案详情

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.idLONG是档案编码10

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.successBOOLEAN否true-成功，false-失败true
body.messageSTRING否调用信息调用成功！
body.dataOBJECT否数据
body.data.createTimeDATE否创建时间
body.data.customFileIdSTRING否自定义档案编码
body.data.customFileNameSTRING否自定义档案名称
body.data.customFileTypeSTRINGMULTI_LEVEL有层级
SINGLE_LEVEL无层级
body.data.statusSTRING0-关闭，1-开启
body.data.fromApiSTRING否0-不是 1-是
body.data.limitINTEGER否每页个数
body.data.optionsARRAY否选项
body.data.options.childrenARRAY否子选项数组
body.data.options.customFileSeqIdLONG否所属档案自增id
body.data.options.optionIdSTRING否选项编码
body.data.options.optionIdListARRAY否选项ID列表
body.data.options.optionLevelDOUBLE否选项层级
body.data.options.optionNameSTRING否选项名称
body.data.options.serialNumberINTEGER顺序号
body.data.options.statusSTRING0-关闭，1-开启
body.data.options.parentNameSTRING否上级选项名称
body.data.options.parentSeqIdLONG否上级选项自增Id
body.data.options.seqIdLONG否选项自增ID

接口调用示例

{BaseURL}/api/v1/customFile/detail?id=10
