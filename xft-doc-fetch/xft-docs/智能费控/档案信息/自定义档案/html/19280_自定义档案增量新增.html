自定义档案增量新增

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/custom-file/incr-add

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/custom-file/incr-add

接口描述

增量新增自定义档案选项接口，通过档案编码定位档案位置，通过optionId来判断选项的层级，以及新增的选项信息来实现增量新增自定义档案选项信息，单多层级选项总数均不可超过20000，多层级每层选项总数量不可超过1000。

接口所属功能路径：

智能费控 - 系统设置 - 档案设置 - 自定义档案

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT是请求体根节点
root.customFileIdSTRING是不可为空code20250422
root.optionsARRAY是该值为增量新增的自定义档案选项信息，单多层级选项总数均不可超过20000，多层级每层选项总数量不可超过1000。
root.options.optionIdSTRING否1，定义层级为哪一层（下方newOptionId，newOptionName，serialNumber，applyCompanies，status等信息为本optionId子节点信息）；
2，单层级时可以不传optionId，多层级时optionId不填就默认层级为1。		4135
root.options.newOptionIdSTRING是必填4136
root.options.newOptionNameSTRING是必填选项名称4136
root.options.serialNumberINTEGER默认值为0，最大值为200000
root.options.applyCompaniesSTRING多个适用公司编码之间用,分隔73299
root.options.statusSTRING1：已启用；0，已禁用。1
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN是否成功true
root.messageSTRING调用信息选项增量插入成功
