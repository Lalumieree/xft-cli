自定义档案批量删除

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/custom-file/batch-delete

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/custom-file/batch-delete

接口描述

自定义档案根据自定义档案编码批量删除接口

接口所属功能路径：

智能费控 - 系统设置 - 档案设置 - 自定义档案

<img src="https://s3gw.cmbchina.com/lw36.01-public-prd/public-713bf327-c73e-4057-992d-7be57e8fcbbd" alt="示例图片" width="300" height="200">

接口调用注意事项：

* 接口调用单次最大支持50条；

* 接口不支持部分成功，单次接口调用全量成功/失败；

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootARRAY请求体根节点
root.customFileCodeSTRING否字段在费控系统中的源：
系统设置 -> 档案设置 -> 自定义档案 -> 新增/编辑（档案编码）
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.returnCodeSTRING否返回码SUC0000
root.successBOOLEAN否是否调用成功true
root.messageSTRING否错误信息成功

接口调用示例

请求报文


[
    "CUSTOM0001",
    "CUSTOM0002"
]

响应报文


{  
    "returnCode": "SUC0000",  
    "success": true,  
    "message": "成功"  
}  


{  
    "returnCode": "EROOR1",  
    "success": false,  
    "message": "不存在的档案编码【CUS1000】请核对后重试"  
}

错误码说明

错误码错误信息解决方案
SUC0000成功成功
EROOR1系统异常系统错误，联系管理员处理
EROOR2业务执行错误业务执行错误按提示修改入参后重试
