批量修改自定义档案

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/custom-file/batch-update

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/custom-file/batch-update

接口描述

批量修改自定义档案接口

接口所属功能路径：

智能费控 - 系统设置 - 档案设置 - 自定义档案

<img src="https://s3gw.cmbchina.com/lw36.01-public-prd/public-fe51d2ed-b244-4967-b2b1-978b042c8af3" alt="示例图片" width="300" height="150">

接口调用注意事项：

* 接口调用单次最大支持50条；

* 接口不支持部分成功，单次接口调用全量成功/失败；

* 接口支持重复调用，按自定义档案编码修改档案信息，全量覆盖档案信息；

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootARRAY请求体根节点
root.customFileIdSTRING是档案编码CF10000
root.customFileNameSTRING是档案名称档案1号
root.statusSTRING是* 1-启用
* 0-停用		"1"
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN是true-成功true
root.messageSTRING是接口调用信息调用成功！
root.returnCodeSTRING是接口调用描述信息

接口调用示例

请求


[
    {
        "customFileId": "CF10000",
        "customFileName": "档案1号",
        "status": "1"
    },
    {
        "customFileId": "CF20000",
        "customFileName": "档案2号",
        "status": "1"
    }
]

返回


{  
    "returnCode": "SUC0000",  
    "success": true,  
    "message": "成功"  
}  


{  
    "returnCode": "EROOR1",  
    "success": false,  
    "message": "不存在的档案编码【CUS1000】请核对后重试"  
}

错误码说明

错误码错误信息解决方案
SUC0000成功成功
EROOR1系统异常系统错误，联系管理员处理
EROOR2业务执行错误业务执行错误按提示修改入参后重试
