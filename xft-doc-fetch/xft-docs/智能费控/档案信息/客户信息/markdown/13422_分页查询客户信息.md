# 分页查询客户信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/customer/queryPage 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/customer/queryPage 
 ## 接口描述 
分页查询客户信息 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.start | INTEGER | 是 | 例如：limit=10时，第一页的开始条数为0，第二页的开始条数为10 | 0 |
| body.limit | INTEGER | 是 | 最大支持每页100条 | 10 |
| body.query | OBJECT | 否 | 查询条件 |  |
| body.query.customerIdList | STRING |  | 客户编号列表 | ["CUS0000"] |
| body.query.customerStatus | STRING | 否 | 0：停用<br>1：启用 | 1 |
| body.query.globalSearch | STRING | 否 | 客户编码、客户名称模糊搜索 | 客户 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
