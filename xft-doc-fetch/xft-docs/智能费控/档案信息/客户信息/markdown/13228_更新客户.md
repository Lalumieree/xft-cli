# 更新客户

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/customer/update 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/customer/update 
 ## 接口描述 
根据客户名称，更新客户信息 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.customerTypeId | STRING | 是 | 客户类型编码 | CUS369 |
| body.customerStatus | LONG | 是 | 1:启用，0：禁用 | 1 |
| body.availableOrgType | LONG | 否 | 0：全部公司可用；1：部分公司可用,不传默认全部公司 | 1 |
| body.availableOrgCodeList | ARRAY | 否 | 法人公司编码 |  |
| body.customerCode | STRING | 是 | 客户唯一ID为空时，根据客户编码更新客户信息，客户编码不支持更新 | CUS20231129090546 |
| body.customerName | STRING | 是 | 客户名称 | 客户api测试3 |
| body.unifiedCreditCode | STRING | 否 | 统一信用代码 | 123456486785339896 |
| body.contactName | STRING | 否 | 联系人姓名 | 急急急 |
| body.contactPhone | STRING | 否 | 联系人电话 | 16000000000 |
| body.email | STRING | 否 | 邮箱地址 | 123@qq.com |
| body.companyWebsite | STRING | 否 | 公司官网 | baidu.com |
| body.companyAddress | STRING | 否 | 公司地址 | 浙江省杭州市 |
| body.faxNumber | STRING | 否 | 传真号码 | 123-456 |
| body.remark | STRING | 否 | 备注 | 这是一个测试客户 |
| body.customFieldValues | ARRAY | 否 | 自定义字段值 |  |
| body.customFieldValues.customFieldCode | STRING | 否 | 自定义字段编码 | INFINITY_SELECT_103720543 |
| body.customFieldValues.customFieldValue | STRING |  | 自定义字段填写值 | 自定义字段值 |
| body.customFieldValues.customFieldName | STRING |  | 自定义字段名称 | 这是一个自定义字段 |
| body.customFieldValues.multiValues | ARRAY | 否 | 多选值 |  |
| body.customFieldValues.multiValues.id | STRING | 否 | 多选值的key | 选项2 |
| body.customerSeqId | LONG |  | 客户唯一ID不为空时，根据唯一ID更新客户信息，客户编码支持更新 | 540 |
| body.applyAllUser | BOOLEAN |  | 1，true 全部 false 部分<br>2，如果适用人员传值全部并且适用人员用户组传值不为空时，报错 当前适用范围为全部，不支持用户组配置 | true |
| body.customerGroups | ARRAY |  | 合同适用范围的请求体 |  |
| body.customerGroups.type | STRING | 是 | 1，类型分为六种 EMPLOYEE 员工 DEPARTMENT 部门 POST 岗位 JOB 职位 ROLE 角色 RANK 职级<br>2，用户组类型不在六种类型范围时，报错 适用类型传值错误，请检查 | EMPLOYEE |
| body.customerGroups.typeIdName | STRING | 是 | 传值为对应类型的编码。eg：员工编码（type: EMPLOYEE） | 00210416 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
