# 批量删除客户

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/customer/batch-delete 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/customer/batch-delete 
 ## 接口描述 
客户档案根据客户档案编码批量删除接口

接口所属功能路径：
智能费控 - 系统设置 - 档案设置 - 客户

<img src="https://s3gw.cmbchina.com/lw36.01-public-prd/public-5898e3b4-1944-464a-b36c-cbaddce590cc" alt="示例图片" width="300" height="150">

接口调用注意事项：
* 接口调用单次最大支持50条；
* 接口不支持部分成功，单次接口调用全量成功/失败； 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json

## 接口调用示例 
 #### 请求报文
```json
[
    "CUSTOMER0001",
    "CUSTOMER0002"
]
```
#### 响应报文
```json
{
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}
```  
## 错误码说明 
| 错误码| 错误信息|解决方案|
  | --- | --- | --- |
  | SUC0000| 成功| 成功|
  | EROOR1| 系统异常| 系统错误，联系管理员处理|
  | EROOR2| 业务执行错误| 业务执行错误按提示修改入参后重试|
