更新客户

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/customer/update

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/customer/update

接口描述

根据客户名称，更新客户信息

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.customerTypeIdSTRING是客户类型编码CUS369
body.customerStatusLONG是1:启用，0：禁用1
body.availableOrgTypeLONG否0：全部公司可用；1：部分公司可用,不传默认全部公司1
body.availableOrgCodeListARRAY否法人公司编码
body.customerCodeSTRING是客户唯一ID为空时，根据客户编码更新客户信息，客户编码不支持更新CUS20231129090546
body.customerNameSTRING是客户名称客户api测试3
body.unifiedCreditCodeSTRING否统一信用代码123456486785339896
body.contactNameSTRING否联系人姓名急急急
body.contactPhoneSTRING否联系人电话16000000000
body.emailSTRING否邮箱地址123@qq.com
body.companyWebsiteSTRING否公司官网baidu.com
body.companyAddressSTRING否公司地址浙江省杭州市
body.faxNumberSTRING否传真号码123-456
body.remarkSTRING否备注这是一个测试客户
body.customFieldValuesARRAY否自定义字段值
body.customFieldValues.customFieldCodeSTRING否自定义字段编码INFINITY_SELECT_103720543
body.customFieldValues.customFieldValueSTRING自定义字段填写值自定义字段值
body.customFieldValues.customFieldNameSTRING自定义字段名称这是一个自定义字段
body.customFieldValues.multiValuesARRAY否多选值
body.customFieldValues.multiValues.idSTRING否多选值的key选项2
body.customerSeqIdLONG客户唯一ID不为空时，根据唯一ID更新客户信息，客户编码支持更新540
body.applyAllUserBOOLEAN1，true 全部 false 部分
2，如果适用人员传值全部并且适用人员用户组传值不为空时，报错 当前适用范围为全部，不支持用户组配置		true
body.customerGroupsARRAY合同适用范围的请求体
body.customerGroups.typeSTRING是1，类型分为六种 EMPLOYEE 员工 DEPARTMENT 部门 POST 岗位 JOB 职位 ROLE 角色 RANK 职级
2，用户组类型不在六种类型范围时，报错 适用类型传值错误，请检查		EMPLOYEE
body.customerGroups.typeIdNameSTRING是传值为对应类型的编码。eg：员工编码（type: EMPLOYEE）00210416
参数类型必填说明示例
queryOBJECTquery根节点

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.returnCodeSTRING否返回码SUC0000
body.successBOOLEAN否是否成功true
body.messageSTRING否错误信息成功
body.bodyOBJECT返回信息
body.body.customerSeqIdLONG客户唯一iD540
