新增供应商

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/supplier/add

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/supplier/add

接口描述

新增单个供应商数据

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.availableOrgCodeListARRAY否数据来源：薪福通子公司组织编码

若availableOrgIdList列表为空，则使用本Code列表对应的公司作为可用公司列表
body.availableOrgIdListARRAY否数据来源：薪福通子公司组织ID， 优先使用ID对应的可用公司列表
body.availableOrgTypeSTRING是1-部分公司 0-全部公司0
body.corOrgNbrSTRING否统一信用代码
body.customFieldValuesARRAY否自定义字段需要填入的值
body.customFieldValues.customFieldCodeSTRING否智能费控-档案设置=字段设置-供应商

字段编码		SELECT_140332273
body.customFieldValues.customFieldValueSTRING否自定义字段是单值情况下填入，如果是多值，无需填入
body.customFieldValues.multiValuesARRAY否自定义字段是多值情况下填入，如果是单值，无需填入
body.customFieldValues.multiValues.idSTRING自定义多选字段编码
body.customFieldValues.multiValues.nameSTRING自定义多选字段值
body.enableDateSTRING否格式：yyyy-mm-dd2023-11-11
body.recAccListARRAY否收款账户集合
body.recAccList.accountModeSTRING否账户类型:
BANK-银行账户
DIGITAL_YUAN-数字人民币		BANK
body.recAccList.acctNoSTRING是银行账号不能为空
银行账号需是不能超过30位的字母或数字		6217847384349543545
body.recAccList.acctTypeSTRING是1-对公账户 0-个人账户1
body.recAccList.collectingCompanyNameSTRING否数字人民币账户字段，银行账户无需处理测试中心
body.recAccList.depoBankSTRING是银行支行联行号，可通过公共接口中的银行接口获取105399100293
body.recAccList.depoNameSTRING是银行户名不能为空测试账户
body.recAccList.mainAcctSTRING是1-是 0-否1
body.recAccList.remarkSTRING否备注测试新增
body.recAccList.stateSTRING是1-是 0-否1
body.recAccList.wthrDigitalYuanMainAcctSTRING否数字人民币账户字段，银行账户无需处理

1-是，0-否		0
body.supIdSTRING否供应商编号必须是50位以内的字母或数字SUP20231110123456
body.supNamSTRING是供应商名称不能为空测试供应商
body.supStsSTRING是供应商状态
1-启用 0-停用		1
body.supTypeIdSTRING是智能费控-档案设置-字段设置-供应商-供应商类型s1
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是薪福通企业号XFT12345

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点

POST {BaseURL}/api/v1/supplier/add?OPAUID=XFT12345
