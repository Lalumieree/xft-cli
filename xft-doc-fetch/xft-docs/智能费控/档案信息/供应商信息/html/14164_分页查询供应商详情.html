分页查询供应商详情

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/supplier/page-for-detail

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/supplier/page-for-detail

接口描述

供应商详情分页查询

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.startINTEGER是起始行数0
root.limitINTEGER是最大1000条10
root.queryOBJECT否请求体
root.query.globalSearchSTRING否供应商名称或编码模糊搜索
root.query.supStsSTRING否1：启用； 0：停用1
root.query.availableOrgTypeSTRING否0：全部公司 1：部分公司0
root.query.availableOrgIdSTRING否法人公司idCORx80000
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.bodyOBJECT否返回体
root.body.totalINTEGER是不是list长度1
root.body.listARRAY否供应商信息列表
root.body.list.seqIdLONG是供应商自增ID27321
root.body.list.supIdSTRING是供应商编码SUP20230810104235
root.body.list.supNamSTRING是供应商名称供应商api添加测试1
root.body.list.supTypeIdSTRING是智能费控-档案设置-字段设置-供应商-供应商类型编码SUPTYPE0001
root.body.list.supplierTypeSTRING智能费控-档案设置-字段设置-供应商-供应商类型名称默认供应商类型
root.body.list.supStsSTRING是供应商状态
1-启用 0-停用		1
root.body.list.enableDateSTRING否启用日期2023-01-02
root.body.list.updateTimeSTRING是更新时间2023-08-10 11:09:02
root.body.list.corOrgNbrSTRING否统一信用代码123121234567890985
root.body.list.availableOrgTypeLONG否1-部分公司 0-全部公司1
root.body.list.availableOrgIdListARRAY否String列表["CORx60000"]
root.body.list.customFieldValuesARRAY否自定义字段值列表
root.body.list.customFieldValues.customFieldCodeSTRING否自定义字段编码INFINITY_SELECT_172921020
root.body.list.customFieldValues.customFieldValueSTRING否自定义字段填写值null
root.body.list.customFieldValues.customFieldNameSTRING否自定义字段名称下拉单选
root.body.list.customFieldValues.multiValuesARRAY否多选时的值
root.body.list.customFieldValues.multiValues.idSTRING多选的值 key选项1
root.body.list.customFieldValues.multiValues.nameSTRING多选的显示选项1
root.body.list.customFieldValues.multiValues.codeSTRING多选的 编码null
root.body.list.recAcctListARRAY否银行账户
root.body.list.recAcctList.acctTypeSTRING否1：对公账户，0：个人账户1
root.body.list.recAcctList.bankcodeSTRING银行名称代码C1030935001347
root.body.list.recAcctList.stateSTRING否1：是，0：否1
root.body.list.recAcctList.bankNameSTRING否银行名称中国人民人行
root.body.list.recAcctList.depoBankSTRING否联行号-开户行名称代码001199514244
root.body.list.recAcctList.depoBankNameSTRING否开户行名称人民银行开鲁县支行
root.body.list.recAcctList.depoNameSTRING否默认与供应商名称一致供应商名称
root.body.list.recAcctList.acctNoSTRING否银行账号123456111111
root.body.list.recAcctList.mainAcctSTRING否1：是，0：否1
root.body.list.recAcctList.depoCountrySTRING否开户国家中国
root.body.list.recAcctList.depoProvinceSTRING否开户省份代码150000
root.body.list.recAcctList.depoProvinceNameSTRING否开户省份名称内蒙古自治区
root.body.list.recAcctList.depoCitySTRING否开户城市代码150500
root.body.list.recAcctList.depoCityNameSTRING否开户城市名称通辽市
root.body.list.recAcctList.accountModeSTRING否BANK:银行账户，DIGITAL_YUAN：数字人民币BANK
root.body.list.recAcctList.wthrDigitalYuanMainAcctSTRING否数字人民币账户字段，银行账户无需处理

1-是，0-否		0
root.body.list.recAcctList.collectingCompanyNameSTRING否数字人民币账户字段，银行账户无需处理测试中心
root.body.list.recAcctList.remarkSTRING否备注测试新增
root.returnCodeSTRING是错误码SUC0000
root.successBOOLEAN是是否调用成功true
root.messageSTRING是错误信息成功
