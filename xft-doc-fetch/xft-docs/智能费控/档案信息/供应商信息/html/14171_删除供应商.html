删除供应商

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/supplier/delete

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/supplier/delete

接口描述

删除指定供应商

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.supIdsARRAY否String数组["SUP20230814152024","SUP20230810185817"]
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.bodySTRING否调用成功有此字段删除供应商成功！
root.returnCodeSTRING否错误码SUC0000
root.successBOOLEAN否是否删除成功true
root.messageSTRING否未删除成功的错误提示成功

接口调用示例

请求报文


{
    "supIds":["SUP20230810185817","SUP20230814152024"]
}

响应报文


{
    "body": "删除供应商成功！",
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}

错误码说明

错误码错误信息
EROOR2删除失败
SUC0000删除成功
