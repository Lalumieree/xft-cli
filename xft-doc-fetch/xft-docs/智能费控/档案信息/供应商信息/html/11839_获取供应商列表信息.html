获取供应商列表信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/common/getSupInfoList

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/common/getSupInfoList

接口描述

结算单新增时，外部系统需要根据费控的供应商查询到相应名称的编码信息，才能新增

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是格式为XFT00001的企业编号XFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.successBOOLEANtrue-成功,false-失败true
body.messageSTRING返回消息调用成功！
body.dataARRAY数据
body.data.supIdSTRING是供应商IDSUP20221013144327
body.data.supNamSTRING是供应商名称薪福通模拟供应商2
body.data.taxNumSTRING供应商税号123456789454dsad55
body.data.supStatusSTRING是状态:0停用1启用0

接口调用示例

{BaseURL}/api/v1/common/getSupInfoList?OPAUID=XFT00001
