# 批量更新供应商

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/supplier/batch-update 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/supplier/batch-update 
 ## 接口描述 
批量更新供应商，一个批次最大支持50个供应商进行批量修改，修改时，根据传入的uniqueKey去匹配已经存在的供应商编号  
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
