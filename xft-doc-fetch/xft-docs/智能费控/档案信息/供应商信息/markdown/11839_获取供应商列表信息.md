# 获取供应商列表信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/common/getSupInfoList 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/common/getSupInfoList 
 ## 接口描述 
结算单新增时，外部系统需要根据费控的供应商查询到相应名称的编码信息，才能新增 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | 格式为XFT00001的企业编号 | XFT00001 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.success | BOOLEAN |  | true-成功,false-失败 | true |
| body.message | STRING |  | 返回消息 | 调用成功！ |
| body.data | ARRAY |  | 数据 |  |
| body.data.supId | STRING | 是 | 供应商ID | SUP20221013144327 |
| body.data.supNam | STRING | 是 | 供应商名称 | 薪福通模拟供应商2 |
| body.data.taxNum | STRING |  | 供应商税号 | 123456789454dsad55 |
| body.data.supStatus | STRING | 是 | 状态:0停用1启用 | 0 |
 
## 接口调用示例 
 {BaseURL}/api/v1/common/getSupInfoList?OPAUID=XFT00001
