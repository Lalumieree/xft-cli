# 删除供应商

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/supplier/delete 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/supplier/delete 
 ## 接口描述 
删除指定供应商 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.supIds | ARRAY | 否 | String数组 | ["SUP20230814152024","SUP20230810185817"] |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.body | STRING | 否 | 调用成功有此字段 | 删除供应商成功！ |
| root.returnCode | STRING | 否 | 错误码 | SUC0000 |
| root.success | BOOLEAN | 否 | 是否删除成功 | true |
| root.message | STRING | 否 | 未删除成功的错误提示 | 成功 |
 
## 接口调用示例 
 #### 请求报文
```json
{
    "supIds":["SUP20230810185817","SUP20230814152024"]
}
```
#### 响应报文
```json
{
    "body": "删除供应商成功！",
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}
```  
## 错误码说明 
| 错误码| 错误信息|
  | --- | --- |
  | EROOR2| 删除失败|
|SUC0000|删除成功|
