# 分页查询供应商详情

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/supplier/page-for-detail 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/supplier/page-for-detail 
 ## 接口描述 
供应商详情分页查询 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
