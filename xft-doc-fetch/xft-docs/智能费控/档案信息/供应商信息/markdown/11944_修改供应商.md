# 修改供应商

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/supplier/update 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/supplier/update 
 ## 接口描述 
修改单个供应商数据 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.availableOrgCodeList | ARRAY | 否 | 数据来源：薪福通子公司组织编码<br> <br>若availableOrgIdList列表为空，则使用本Code列表对应的公司作为可用公司列表 |  |
