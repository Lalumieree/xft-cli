# 上传合同附件

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/archives/contract/attachment/upload 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/archives/contract/attachment/upload 
 ## 接口描述 
为新增/编辑合同准备，优先上传合同附件信息，获取附件的唯一ID和编码 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: form-data 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.file | FILE | 是 | 合同附件信息，上传附件大小不能超过50M | d1384b32d6994edfbeeacb236d7ba9da_IMAGE.png |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.EMPNUMBER | STRING | 是 | 操作员工的编码信息 | 00210416 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT | 是 | 响应体根节点 |  |
| body.body | OBJECT | 是 | 文件返回体 |  |
| body.body.seqId | LONG | 是 | 文件返回唯一ID信息 | 33238 |
| body.body.fileName | STRING | 是 | 文件返回唯一编码信息 | 4d88e028-b997-40e9-912a-fd56bbda457a |
| body.body.fileShowName | STRING | 是 | 文件原名称 | d1384b32d6994edfbeeacb236d7ba9da_IMAGE.png |
| body.body.success | BOOLEAN | 是 | 文件是否上传成功 | true |
| body.returnCode | STRING | 是 | 返回编码 | SUC0000 |
| body.success | BOOLEAN | 是 | 是否请求成功 | true |
| body.message | STRING | 是 | 明细信息 | 成功 |
 
## 接口调用示例 
 #### 请求报文
```json
{"file":[文件请求体]}
```
#### 响应报文
```json
{
    "body": {
        "seqId": 33238,
        "fileName": "4d88e028-b997-40e9-912a-fd56bbda457a",
        "fileShowName": "d1384b32d6994edfbeeacb236d7ba9da_IMAGE.png",
        "success": true
    },
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}
```
