# 获取合同台账

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/contract/getContractBook 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/contract/getContractBook 
 ## 接口描述 
获取合同台账分页查询接口
1. 提供合同编码和合同名称的模糊查询
2. 提供合同性质，合同状态，合同类型等匹配筛选
3. 返回合同基本信息，已报销金额明细，已支付金额明细，印花税金额明细，条款金额明细等信息 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
