# 新增合同档案

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/archives/contract/add 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/archives/contract/add 
 ## 接口描述 
在公司下创建合同档案，支持创建付款合同、框架合同。支持传入合同基本信息、合同适用范围、合同附件、付款合同付款条款、合同受益期、合同自定义字段等信息。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
