# 分页查询合同档案信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/archives/contract/page 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/archives/contract/page 
 ## 接口描述 
针对企业合同信息进行分页查询，支持基于合同类型编码、合同名称、合同编码、合同公司、合同供应商等参数进行搜索 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
