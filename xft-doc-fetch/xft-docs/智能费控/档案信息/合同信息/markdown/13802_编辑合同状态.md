# 编辑合同状态

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/archives/contract/status/update 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/archives/contract/status/update 
 ## 接口描述 
更新合同状态信息，支持针对合同状态进行变更为已暂挂，取消暂挂，终止状态，只有履行中、未履行、暂挂中的合同可以修改状态 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.EMPNUMBER | STRING | 是 | 操作人编码 | 00210416 |
| query.contractSeqId | LONG | 否 | 支持根据合同自增ID修改合同状态，合同自增ID和合同编码不能同时为空 | 1952 |
| query.contractNbr | STRING | 否 | 支持根据合同编码修改合同状态，合同编码和合同自增ID不能同时为空 | CON20240507101713 |
| query.status | STRING | 是 | 3：终止，5：解除暂挂，6：暂挂 | 5 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.body | BOOLEAN | 否 | 返回请求体 | true |
| body.returnCode | STRING | 否 | 返回编码 | SUC0000 |
| body.success | BOOLEAN | 否 | 是否请求成功 | true |
| body.message | STRING | 否 | 明细信息 | 成功 |
 
## 接口调用示例 
 #### 请求报文
```json
{BaseURL}/api/v2/archives/contract/status/update?EMPNUMBER=00210416&contractNbr=CON20240507101713&status=5
```
#### 响应报文
```json
{
    "body": true,
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}
```
