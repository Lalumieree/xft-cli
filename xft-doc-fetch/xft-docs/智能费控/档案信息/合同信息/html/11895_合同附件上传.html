合同附件上传

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/contractArchives/uploadAttachment

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/contractArchives/uploadAttachment

接口描述

用于上传合同附件，请求类型multipart/form-data，文件不能超过50MB

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.PRJCODSTRING是企业标识XFT00001
query.EMPNUMBERSTRING是花名册员工号U0000

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.dataOBJECT否返回数据
body.data.seqIdLONG否合同上传需要用这个作为参数13909
body.data.fileNamSTRING否文件名称(合同上传需要用这个作为参数)合同1
body.successBOOLEAN否true-成功，false-失败true
body.messageSTRING否返回消息调用成功！

接口调用示例

{BaseURL}/v1/contractArchives/uploadAttachment?PRJCOD=XFT00001&EMPNUMBER=U0000

注意文件上传，参数file

返回示例


{
    "data": {
        "seqId": 13909,
        "fileNam": "d5e3ee5b-07a7-461a-aba0-c63b3070a93e"
    },
    "success": true,
    "message": "调用成功！"
}
