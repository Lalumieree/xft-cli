单个查询合同明细信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/archives/contract/detail/query

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/archives/contract/detail/query

接口描述

根据合同唯一ID查询合同明细的信息，合同唯一ID需要根据分页查询合同接口获取

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.contractSeqIdLONG支持根据合同自增ID查询合同信息，合同自增ID和合同编码不能同时为空1845
query.contractNbrSTRING否支持根据合同编码查询合同信息，合同编码和合同自增ID不能同时为空CON20240507101713
