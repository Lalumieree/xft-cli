分页查询合同档案信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/archives/contract/page

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/archives/contract/page

接口描述

针对企业合同信息进行分页查询，支持基于合同类型编码、合同名称、合同编码、合同公司、合同供应商等参数进行搜索

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.contractLpcCodeSTRING否合同所屬公司编码
body.contractTypeCodeListARRAY否合同类型编码列表
body.contractSupplierNamSTRING否乙方公司（供应商名称）
body.contractStatusListARRAY否0：审批中
1：未履行
2：履行中
3：已终止
4：已过期
6：暂挂中
7：已完成
8：变更审批中
body.contractNameSTRING否合同名称
body.contractCodeSTRING否合同编码
body.startINTEGER是分页查询开始位置0
body.limitINTEGER是分页需要查询的数量50
参数类型必填说明示例
queryOBJECTquery根节点
