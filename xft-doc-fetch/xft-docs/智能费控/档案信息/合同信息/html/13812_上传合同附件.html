上传合同附件

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/archives/contract/attachment/upload

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/archives/contract/attachment/upload

接口描述

为新增/编辑合同准备，优先上传合同附件信息，获取附件的唯一ID和编码

调用方式

METHOD: POST

CONTENT-TYPE: form-data

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.fileFILE是合同附件信息，上传附件大小不能超过50Md1384b32d6994edfbeeacb236d7ba9da_IMAGE.png
参数类型必填说明示例
queryOBJECTquery根节点
query.EMPNUMBERSTRING是操作员工的编码信息00210416

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT是响应体根节点
body.bodyOBJECT是文件返回体
body.body.seqIdLONG是文件返回唯一ID信息33238
body.body.fileNameSTRING是文件返回唯一编码信息4d88e028-b997-40e9-912a-fd56bbda457a
body.body.fileShowNameSTRING是文件原名称d1384b32d6994edfbeeacb236d7ba9da_IMAGE.png
body.body.successBOOLEAN是文件是否上传成功true
body.returnCodeSTRING是返回编码SUC0000
body.successBOOLEAN是是否请求成功true
body.messageSTRING是明细信息成功

接口调用示例

请求报文


{"file":[文件请求体]}

响应报文


{
    "body": {
        "seqId": 33238,
        "fileName": "4d88e028-b997-40e9-912a-fd56bbda457a",

        "success": true
    },
    "returnCode": "SUC0000",

}
