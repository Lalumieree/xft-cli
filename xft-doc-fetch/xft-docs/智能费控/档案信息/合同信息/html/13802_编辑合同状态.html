编辑合同状态

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/archives/contract/status/update

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/archives/contract/status/update

接口描述

更新合同状态信息，支持针对合同状态进行变更为已暂挂，取消暂挂，终止状态，只有履行中、未履行、暂挂中的合同可以修改状态

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.EMPNUMBERSTRING是操作人编码00210416
query.contractSeqIdLONG否支持根据合同自增ID修改合同状态，合同自增ID和合同编码不能同时为空1952
query.contractNbrSTRING否支持根据合同编码修改合同状态，合同编码和合同自增ID不能同时为空CON20240507101713
query.statusSTRING是3：终止，5：解除暂挂，6：暂挂5

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.bodyBOOLEAN否返回请求体true
body.returnCodeSTRING否返回编码SUC0000
body.successBOOLEAN否是否请求成功true
body.messageSTRING否明细信息成功

接口调用示例

请求报文


{BaseURL}/api/v2/archives/contract/status/update?EMPNUMBER=00210416&contractNbr=CON20240507101713&status=5

响应报文


{
    "body": true,
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}
