获取合同台账

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/contract/getContractBook

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/contract/getContractBook

接口描述

获取合同台账分页查询接口

1. 提供合同编码和合同名称的模糊查询

2. 提供合同性质，合同状态，合同类型等匹配筛选

3. 返回合同基本信息，已报销金额明细，已支付金额明细，印花税金额明细，条款金额明细等信息

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.comIdSTRING企业ID1111
body.billIdSTRING合同创建单据编号ID202300000000000000
body.globalSearchSTRING全局查询：合同编码/合同名称测试
body.payStampTaxFlagSTRING是否开启了印花税配置1
body.contractAttrSTRING0-归档合同 1-付款合同 2-框架合同1
body.contractTypeSeqIdListSTRING合同类型自增ID1
body.depSeqIdListSTRING责任部门自增ID1
body.supSeqIdListSTRING乙方单位自增ID1
body.startINTEGER是查询起始0
body.limitINTEGER是分页大小10
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是新福通企业号XFT00001
