查询公司付款账户详情

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/pay/account/detail

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/pay/account/detail

接口描述

公司账户根据账号查询详情

调用方式

METHOD: POST

CONTENT-TYPE: application/json

参数类型必填说明示例
bodyOBJECT请求体根节点

query.OPAUIDSTRING企业号XFT00001
