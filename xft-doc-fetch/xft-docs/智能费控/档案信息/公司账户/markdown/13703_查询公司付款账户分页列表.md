# 查询公司付款账户分页列表

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/pay/account/page 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/pay/account/page 
 ## 接口描述 
公司账户查询分页列表 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
