修改项目信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/update

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/update

接口描述

修改项目信息

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT是请求体根节点
body.projectSeqIdLONG是项目自增ID239
body.comIdLONG否公司ID3100
body.projectIdSTRING否项目编码20211020094758b09bf
body.projectNameSTRING否项目名称API导入项目456
body.projectParentIdSTRING否上级项目编码202110200947588754d
body.projectParentNameSTRING否上级项目名称API导入项目123
body.projectStatusBOOLEAN否项目状态true
body.projectLevelLONG否项目层级2
body.projectTypeNameSTRING否项目类型名称API导入项目类型
body.projectTypeIdSTRING否项目类型编码PRJTYPE20220408112737
body.projectTypeOBJECT否项目类型
body.projectType.prjTypeIdSTRING否项目类型编码PRJTYPE20230602113257
body.projectRolesARRAY否项目角色
body.projectRoles.prjRoleSeqIdLONG否项目角色自增ID1388
body.projectRoles.prjRoleNamSTRING否项目角色名称demoData
body.projectRoles.userGroupARRAY否项目角色可见用户组
body.projectRoles.userGroup.prjTypeSeqIdLONG否项目类型自增ID1388
body.projectRoles.userGroup.typeSeqIdLONG否项目角色类型自增ID41856
body.projectRoles.userGroup.typeSTRING否类型EMPLOYEE
body.visibleTypeBOOLEAN否可见范围（true为全部可见）true
body.userGroupARRAY否可见用户组
body.userGroup.typeSTRING否类型EMPLOYEE
body.userGroup.staffOBJECT否员工信息
body.userGroup.staff.empIdSTRING否员工idU0000
body.userGroup.staff.empNumberSTRING否员工花名册编码A0001
body.userGroup.staff.empNamSTRING否员工名称员工名
body.userGroup.staff.phoneNbrLONG否员工手机号18111111111
body.startDateSTRING否起始日期2021-10-18
body.endDateSTRING否终止日期2021-11-18
body.customFieldValuesARRAY否自定义字段值
body.customFieldValues.customFieldCodeSTRING否字段编号MULTI_INPUT_094518693
body.customFieldValues.customFieldValueLONG否自定义字段填写值123456
body.overdueBOOLEAN否逾期状态false
body.projectStageDtoOBJECT项目状态
body.projectStageDto.projectStageCodeSTRING项目状态编码code
body.projectStageDto.projectStageNameSTRING项目状态名称状态名称
body.applyUserConfigOBJECT适用范围对象
body.applyUserConfig.applyUserSTRING* 全部：“1”
* 部分：“0”		"0"
body.applyUserConfig.departIncludeChildBOOLEAN* 是：true
* 否：false		false
body.applyUserConfig.statisticsTypeSTRING* 并集：UNION_SET
* 交集：INTERSECTION		UNION_SET
body.applyUserConfig.applyUserConfigDtoListARRAY支持 员工（EMPLOYEE），部门（DEPARTMENT），角色（ROLE），职级（RANK），岗位（POST），职位（JOB）配置
body.applyUserConfig.applyUserConfigDtoList.authTypeSTRING支持 如下适用类型的配置：
* 员工：EMPLOYEE
* 部门：DEPARTMENT
* 角色：ROLE
* 职级：RANK
* 岗位：POST
* 职位：JOB		EMPLOYEE
body.applyUserConfig.applyUserConfigDtoList.typeCodeSTRING* 当authType为EMPLOYEE 当前字段传企业自己的员工号（门户维护）
* 当authType为DEPARTMENT当前字段传企业自己的部门号（门户维护）
* 当authType为ROLE当前字段传企业自己的角色号（门户维护）
* 当authType为RANK当前字段传企业自己的职级号（门户维护）
* 当authType为POST当前字段传企业自己的岗位号（门户维护）
* 当authType为JOB当前字段传企业自己的职位号（门户维护）		800000000
参数类型必填说明示例
queryOBJECTquery根节点

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.successBOOLEAN成功标志true
body.messageSTRING错误信息调用成功
