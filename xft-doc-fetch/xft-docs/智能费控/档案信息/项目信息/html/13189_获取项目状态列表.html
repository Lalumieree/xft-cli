获取项目状态列表

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/getProjectStageList

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/getProjectStageList

接口描述

获取企业下所有的项目状态

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是公司标识XFA22222

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.dataARRAY否返回数据
body.data.seqIdLONG否自增ID100
body.data.comIdLONG否公司ID3597
body.data.projectStageCodeLONG否编码00001
body.data.projectStageNameSTRING否名称状态1
body.data.projectStageStsLONG否0停用1启用0
body.data.approvedLONG否0 未立项 1 已立项1
body.successBOOLEAN否true-成功,false-失败true
body.messageSTRING否返回消息调用成功！

接口调用示例

请求报文

{URL}/v1/project/getProjectStageList?OPAUID=XFA232222
