项目信息导入

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/importProject

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/importProject

接口描述

项目信息导入

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
bodyARRAY项目信息导入集合
body.projectInfOBJECT否项目信息对象
body.projectInf.projectIdSTRING否项目编码
body.projectInf.projectNameSTRING否项目名称API导入项目321111
body.projectInf.projectParentIdSTRING否上级项目ID
body.projectInf.projectParentNameSTRING否上级项目名称
body.projectInf.projectLevelSTRING否上限10层
body.projectInf.projectTypeIdSTRING否客户自定义项目类型
body.projectInf.projectTypeNameSTRING否客户自定义项目类型
body.projectInf.projectIndustryCodeSTRING项目行业编码code001
body.projectInf.isTaxFreeBOOLEAN是否免税项目true
body.projectInf.customerInfDtoOBJECT否客户信息
body.projectInf.customerInfDto.customerIdSTRING否客户编码6t783e523
body.projectInf.projectRolesARRAY否项目角色信息
body.projectInf.projectRoles.prjRoleNamSTRING否项目角色名称角色名称
body.projectInf.projectRoles.prjRoleIdSTRING项目角色编号FHG0001
body.projectInf.projectRoles.prjRoleSeqIdSTRING存在历史包袱 后续需要优化删除
body.projectInf.projectRoles.userGroupARRAY否使用人员组
body.projectInf.projectRoles.userGroup.typeSTRING否EMPLOYEE-员工（目前仅支持）EMPLOYEE
body.projectInf.projectRoles.userGroup.staffOBJECT否员工信息
body.projectInf.projectRoles.userGroup.staff.empIdSTRING否员工id、员工花名册编码、员工名称、员工手机号，四选一填写即可U0000
body.projectInf.projectRoles.userGroup.staff.empNumberSTRING员工花名册编码A0001
body.projectInf.projectRoles.userGroup.staff.empNamSTRING员工名称
body.projectInf.projectRoles.userGroup.staff.phoneNbrSTRING员工手机号18111111111
body.projectInf.visibleTypeBOOLEAN否全都可见：true
部分可见：false
部分可见是需要填写使用的用户组		true
body.projectInf.userGroupARRAY否可见用户组
body.projectInf.userGroup.typeSTRING否EMPLOYEE-员工（仅支持）EMPLOYEE
body.projectInf.userGroup.staffOBJECT否同上staff
body.projectInf.projectStageDtoOBJECT项目状态
body.projectInf.projectStageDto.projectStageCodeSTRING项目状态编码code
body.projectInf.projectStageDto.projectStageNameSTRING项目状态名称状态名称
body.projectInf.startDateSTRING否yyyy-MM-dd2021-10-18
body.projectInf.endDateSTRING否yyyy-MM-dd2021-10-18
body.projectInf.customFieldValuesARRAY否自定义字段值
body.projectInf.customFieldValues.customFieldCodeSTRING自定义字段编码
body.projectInf.customFieldValues.customFieldValueSTRING自定义字段填写值
body.projectInf.customFieldValues.multiValuesARRAY多选时的值
body.projectInf.customFieldValues.multiValues.idSTRING多选的值 key
body.projectInf.applyUserConfigOBJECT适用范围对象
body.projectInf.applyUserConfig.applyUserSTRING* 全部：“1”
* 部分：“0”		“0”
body.projectInf.applyUserConfig.departIncludeChildBOOLEAN* 是：true
* 否：false		false
body.projectInf.applyUserConfig.statisticsTypeSTRING* 并集：UNION_SET
* 交集：INTERSECTION		UNION_SET
body.projectInf.applyUserConfig.applyUserConfigDtoListARRAY支持 员工（EMPLOYEE），部门（DEPARTMENT），角色（ROLE），职级（RANK），岗位（POST），职位（JOB）配置
body.projectInf.applyUserConfig.applyUserConfigDtoList.authTypeSTRING支持 如下适用类型的配置：
* 员工：EMPLOYEE
* 部门：DEPARTMENT
* 角色：ROLE
* 职级：RANK
* 岗位：POST
* 职位：JOB		EMPLOYEE
body.projectInf.applyUserConfig.applyUserConfigDtoList.typeCodeSTRING* 当authType为EMPLOYEE 当前字段传企业自己的员工号（门户维护）
* 当authType为DEPARTMENT当前字段传企业自己的部门号（门户维护）
* 当authType为ROLE当前字段传企业自己的角色号（门户维护）
* 当authType为RANK当前字段传企业自己的职级号（门户维护）
* 当authType为POST当前字段传企业自己的岗位号（门户维护）
* 当authType为JOB当前字段传企业自己的职位号（门户维护）		800000000
body.childrenARRAY否子项目集[projectInfDto]
body.children.childrenARRAY否同对象层级嵌套[projectInfDto]
body.children.projectInfOBJECT同对象层级嵌套
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataARRAY否成功数据返回体
root.data.projectIdSTRING否导入成功项目编码202309121847223ff00
root.data.projectNameSTRING否导入成功项目名称API导入项目321111
root.successBOOLEAN否状态标识true
root.messageSTRING否描述调用成功！
