分页查询项目信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/page/query

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/page/query

接口描述

分页查询企业下的项目信息列表

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.globalSearchSTRING否项目编号和名称模糊匹配测试自定义字段
root.startINTEGER是最小支持0，不能为空0
root.limitINTEGER是最大支持100条一页100
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataOBJECT否返回数据
root.data.totalLONG否总数1
root.data.listARRAY否列表数据
root.data.list.projectSeqIdLONG否项目自增id113339
root.data.list.comIdLONG否公司ID3398
root.data.list.projectIdSTRING否项目编码PRJ20211
root.data.list.projectNameSTRING否项目名称项目名称10122
root.data.list.parentProjectNameSTRING否父级项目名称项目名称6
root.data.list.parentProjectIdLONG父级项目编码PRJ20231018110937
root.data.list.projectTypeSTRING否项目类型类型1
root.data.list.startDateDATE否时间戳1662307200000
root.data.list.endDateDATE否时间戳1706630400000
root.data.list.customerIdSTRING项目客户编码CUS20221220110847
root.data.list.customerNameSTRING否项目客户名称刘**
root.data.list.projectAddressSTRING否项目地址名称北京市
root.data.list.projectStageSTRING否项目状态测试项目状态
root.data.list.projectIndustrySTRING否项目行业名称互联网
root.data.list.isTaxFreeSTRING否1:是，
0:否		0
root.data.list.projectRemarkSTRING否项目备注项目备注
root.data.list.customFieldValuesARRAY否自定义字段
root.data.list.customFieldValues.customFieldValueSTRING否自定义字段填写值文本内容
root.data.list.customFieldValues.customFieldCodeSTRING否自定义字段编码1
root.data.list.customFieldValues.multiValuesARRAY否最大支持100个选项[ "选项2","选项3" ]
root.data.list.projectRolesARRAY项目角色
root.data.list.projectRoles.projectRoleIdSTRING项目角色编号000001
root.data.list.projectRoles.projectRoleNameSTRING项目角色名称项目经理
root.data.list.projectRoles.userGroupARRAY项目角色用户组
root.data.list.projectRoles.userGroup.typeSTRING类型范围:

EMPLOYEE("员工"),

职级：职级编码

root.data.list.visibleTypeSTRING1:全部员工
0:部分员工		0
root.data.list.visibleUserGroupARRAY可见用户组
root.data.list.visibleUserGroup.typeSTRING类型范围:

EMPLOYEE("员工"),
DEPARTMENT("部门"),

SUBSIDIARY("合同公司"),

BUSINESS_GROUP("业务分组");		EMPLOYEE
root.data.list.visibleUserGroup.idSTRING与类型匹配的编码：

员工：员工号
部门：部门编码
职位：职位编码
岗位：岗位编码
职级：职级编码
角色：角色编码		U0000
root.successBOOLEAN否该值只会返回true，异常请求参考异常示例true
root.messageSTRING否请求成功提示调用成功！

错误码说明

错误码错误信息解决方案
E0002分页大小不能为空;分页起始位置不能为空;检查分页参数是否漏传
E0002分页起始位置最小为0;检查分页起始值是否小于0
E0002最大支持100条一页;检查分页条数是否大于100
