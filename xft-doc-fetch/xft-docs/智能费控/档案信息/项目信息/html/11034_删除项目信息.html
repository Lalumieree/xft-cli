删除项目信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/delete

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/delete

接口描述

根据项目自增id删除项目信息

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.projectInfOBJECT是项目信息对象
body.projectInf.projectSeqIdLONG是费控端自增ID0
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是公司标识XFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.successBOOLEAN成功标志true
body.messageSTRING错误信息调用成功

接口调用示例

###请求示例

` {BaseURL}/api/v1/project/delete?OPAUID=XFT00001`


{
  "projectSeqId": 0
}

返回示例


{
  "success": true,
  "message": "调用成功！"
}
