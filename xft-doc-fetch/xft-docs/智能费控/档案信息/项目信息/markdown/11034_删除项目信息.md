# 删除项目信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/delete 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/delete 
 ## 接口描述 
根据项目自增id删除项目信息 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.projectInf | OBJECT | 是 | 项目信息对象 |  |
| body.projectInf.projectSeqId | LONG | 是 | 费控端自增ID | 0 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | 公司标识 | XFT00001 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.success | BOOLEAN |  | 成功标志 | true |
| body.message | STRING |  | 错误信息 | 调用成功 |
 
## 接口调用示例 
 ###请求示例
` {BaseURL}/api/v1/project/delete?OPAUID=XFT00001`
```json
{
  "projectSeqId": 0
}
```

### 返回示例
```json
{
  "success": true,
  "message": "调用成功！"
}
```

|-----------|-------------------|-------------------------------|
| 200      | 成功                 | 无                            |
