# 修改项目信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/update 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/update 
 ## 接口描述 
修改项目信息 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
