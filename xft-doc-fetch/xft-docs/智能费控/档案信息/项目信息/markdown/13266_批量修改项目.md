# 批量修改项目

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/batchUpdate 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/batchUpdate 
 ## 接口描述 

批量修改项目信息，一个批次最大支持修改50条项目信息，支持更新项目id、名称、所属上级、项目类型、项目所在地、项目开始时间、项目结束时间、项目所属行业、项目角色、项目可见用户组（目前仅支持用户）、项目状态、项目是否免税。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
