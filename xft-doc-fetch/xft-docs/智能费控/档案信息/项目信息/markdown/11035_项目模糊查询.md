# 项目模糊查询

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/globalSearch 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/globalSearch 
 ## 接口描述 
根据关键字（项目名称/项目编码）模糊查询项目 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json 
## 接口输入参数

|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | 薪福通企业号 | XFT00001 |
| query.keyWord | STRING |  | 搜索关键字 | project1 |
