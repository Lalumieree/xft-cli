# 获取项目状态列表

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/getProjectStageList 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/getProjectStageList 
 ## 接口描述 
获取企业下所有的项目状态 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | 公司标识 | XFA22222 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.data | ARRAY | 否 | 返回数据 |  |
| body.data.seqId | LONG | 否 | 自增ID | 100 |
| body.data.comId | LONG | 否 | 公司ID | 3597 |
| body.data.projectStageCode | LONG | 否 | 编码 | 00001 |
| body.data.projectStageName | STRING | 否 | 名称 | 状态1 |
| body.data.projectStageSts | LONG | 否 | 0停用1启用 | 0 |
| body.data.approved | LONG | 否 | 0 未立项 1 已立项 | 1 |
| body.success | BOOLEAN | 否 | true-成功,false-失败 | true |
| body.message | STRING | 否 | 返回消息 | 调用成功！ |
