# 获取项目详情

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/getProjectInf/{projectSeqId} 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/getProjectInf/{projectSeqId} 
 ## 接口描述 
根据项目自增id获取项目详情 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 | **参数**     | **类型** | **必填** | **位置** | **长度** | **说明**   | **备注**       |
| ------------ | -------- | -------- | -------- | -------- | ---------- | -------------- |
| OPAUID       | String   | 是       | Query    | 8        | 企业标识   | 传薪福通企业号 |
| projectSeqId | Long     | 是       | Path     | 20       | 项目自增id | 费控端自增ID              |
