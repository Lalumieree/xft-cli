# 分页查询项目信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/page/query 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/page/query 
 ## 接口描述 
分页查询企业下的项目信息列表 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数

|---|---|---|---|---|

|---|---|---|---|---|

| root.start | INTEGER | 是 | 最小支持0，不能为空 | 0 |
| root.limit | INTEGER | 是 | 最大支持100条一页 | 100 |

|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |

## 错误码说明 
| 错误码| 错误信息|解决方案| 
  | --- | --- |--- |
  | E0002 | 分页大小不能为空;分页起始位置不能为空; |检查分页参数是否漏传|
| E0002 | 分页起始位置最小为0; |检查分页起始值是否小于0|
| E0002 | 最大支持100条一页; |检查分页条数是否大于100|
