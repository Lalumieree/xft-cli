# 获取项目类型列表

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/project/getProjectTypeInfList 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/project/getProjectTypeInfList 
 ## 接口描述 
获取企业的全部项目类型 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | 公司标识 | XFA23222 |
 
## 接口输出参数

|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.data | ARRAY | 否 | 返回数据 |  |
| body.data.prjTypeSeqId | LONG | 否 | 项目类型自增ID | 253 |
