# 批量查询对账结算单详情

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/corporate/recon-settle/detail 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/corporate/recon-settle/detail 
 ## 接口描述 
批量获取采购付款的对账结算单的单据详情，单次批量最多查询20笔单据
注意：对账结算单为EventType值为RECON_SETTLE的单据 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
