# 采购付款发货明细批量查询

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-pay/core/shipping-detail/batch-query 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-pay/core/shipping-detail/batch-query 
 ## 接口描述 
用于采购付款批量查询发货明细信息，单次查询上限100条 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json

|---|---|---|---|---|

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.shippingDetailCodeList | ARRAY | 否 | 最大查询上限为100 | ["INV202502000001","INV202502000002"] |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |

| 错误码| 错误信息|解决方案|
  | --- | --- | --- |
  | E0002| 查询发货明细数量不能超过100条| 控制单次查询发货明细数量小于100|
  | EROOR1|系统错误| 稍后再试，或联系开发人员|
