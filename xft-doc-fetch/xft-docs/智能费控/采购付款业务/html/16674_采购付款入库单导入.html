采购付款入库单导入

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-pay/import-inventory-order

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-pay/import-inventory-order

接口描述

用于客户在采购付款场景下，将从客户的业务系统中的入库单同步到采购付款系统。

- 当导入入库单需对账时，若入库单数据在系统中不存在，则直接导入；若入库单数据已存在，则校验对账状态和结算单状态，仅支持导入“待对账”并且结算单状态是“未结算”的入库单，导入后做数据更新逻辑；否则会导入失败。

- 当导入入库单无需对账时，若入库单数据在系统中不存在，则直接导入；若入库单数据已存在，则校验结算状态，仅支持导入“未结算”的入库单，导入后做数据更新逻辑；其他结算状态则导入失败。

导入数据不支持部分成功。

调用方式

METHOD: POST

CONTENT-TYPE: application/json
