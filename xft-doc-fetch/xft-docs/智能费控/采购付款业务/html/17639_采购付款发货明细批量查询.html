采购付款发货明细批量查询

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-pay/core/shipping-detail/batch-query

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-pay/core/shipping-detail/batch-query

接口描述

用于采购付款批量查询发货明细信息，单次查询上限100条

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.shippingDetailCodeListARRAY否最大查询上限为100["INV202502000001","INV202502000002"]
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
