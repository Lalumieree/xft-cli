采购付款预入库单导入

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-pay/core/pre-inventory/import

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-pay/core/pre-inventory/import

接口描述

功能描述： 将预入库信息导入至采购付款应用。

接口注意事项：批量导入接口，允许部分成功，导入上限为100。

采购付款与连接器的交互流程图如下：

![预入库单导入 (4).png](https://s3gw.cmbchina.com/lw36.01-public-prd/public-5531217f-9568-47f4-9cdb-65c06fd26856)

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.purchaseCompanyIdLONG是采购法人公司ID65921
root.purchaseCompanyNameSTRING是采购法人公司名称"江西大学附属医学院"
root.supplierArchiveSeqIdLONG是供应商档案ID65924
root.supplierArchiveNameSTRING是供应商档案名称"福建省食品运输公司"
root.inventoryListARRAY是预入库数据
root.inventoryList.inventoryCodeSTRING是预入库单编号"INV2025071804522965"
root.inventoryList.inventoryDateSTRING是yyyy-MM-dd HH:mm:ss"2025-08-28 18:34:00"
root.inventoryList.customFieldValuesARRAY预入库单自定义字段值
root.inventoryList.customFieldValues.fieldCodeSTRING是自定义字段编码"INPUT_100553748"
root.inventoryList.customFieldValues.fieldValueSTRING自定义字段对应的值"DIY"
root.inventoryList.productDetailsARRAY是预入库单产品明细列表
root.inventoryList.productDetails.shippingDetailCodeSTRING是关联的发货明细编号"sn2025071804522965"
root.inventoryList.productDetails.inventoryQuantityBIGDECIMAL是整数不超过15位，小数不超过6位12.56
root.inventoryList.productDetails.productCodeSTRING是产品编码"PRO2025071500001"
root.inventoryList.productDetails.productNameSTRING是产品名称"测试1"
root.inventoryList.productDetails.goodsTypeSTRING是产品种类"测试种类"
root.inventoryList.productDetails.productMeasureUnitSTRING产品计量单位"5"
root.inventoryList.productDetails.unitPriceBIGDECIMAL是整数不超过15位，小数不超过6位22
root.inventoryList.productDetails.productPackagingSTRING包装"4瓶"
root.inventoryList.productDetails.productSpecificationSTRING规格型号"6个"
root.inventoryList.productDetails.productManufacturingEnterpriseSTRING生产企业"杭州娃哈哈公司"
root.inventoryList.productDetails.productBatchNoSTRING生产批号"001"
root.inventoryList.productDetails.expiryDateSTRING格式为：yyyy-MM-dd"2025-07-30"
root.inventoryList.productDetails.invoiceNumberSTRING多张发票建议用逗号隔开，如”1234,2321412“"12457890"
root.inventoryList.productDetails.customFieldValuesARRAY明细自定义字段值
root.inventoryList.productDetails.customFieldValues.fieldCodeSTRING是自定义字段编码"INPUT_100553748"
root.inventoryList.productDetails.customFieldValues.fieldValueSTRING否自定义字段对应的值"DIY"
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN是成功：true，失败：falsetrue
root.messageSTRING否导入全失败时，会返回错误信息"以下发货明细编码重复：shippingDetailCode1"
root.returnCodeSTRING是调用失败：EROOR1
执行失败：EROOR2
执行成功：SUC0000
参数错误：E0002		"SUC0000"
root.bodyARRAY否返回的业务数据
root.body.shippingDetailCodeSTRING是发货明细编号"INV1234567"
root.body.resultBOOLEAN是true - 成功；
false - 失败		true
root.body.errorMessageSTRING错误原因"发货明细编号PRE000001不存在"

错误码错误信息解决方案
EROOR2业务执行错误参考errorMessage错误信息字段
EROOR1系统错误稍后再试，或联系开发人员
E0002单次导入入库单数量不能超过100条控制入库单导入数量小于100
