采购付款-分页查询对账单

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-pay/reconciliation/core/statistic/page

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-pay/reconciliation/core/statistic/page

接口描述

分页查询采购付款对账单

1. 生成对账单起止日期为必填项

2. 查询结果按对账单更新时间降序排列

3. 分页起止行数从0开始；页大小默认10，最大500

4. 使用对账发起时间和对账完成时间进行筛选时，请注意以下事项：

- 对账发起时间：仅在采购付款对账单状态为“对账中”或“已对账”时有值

- 对账完成时间：仅在采购付款对账单状态为“已对账”时有值

调用方式

METHOD: POST

CONTENT-TYPE: application/json

错误码说明

错误码 |错误信息|解决方案

-|-|-

EROOR1|系统错误|请稍后再试，或联系开发人员

EROOR2|对账单生成时间-起始日期不能为空;对账单生成时间-终止日期不能为空|对账单生产时间起止区间必填，日期格式：yyyy-MM-dd

E0002|页大小最大为500|请调整limit参数大小

E0002|对账状态编码不合法|请参照字段备注，输入指定编码

E0002|结算状态编码不合法|请参照字段备注，输入指定编码

E0002|法人公司不存在|请检查法人公司名称或编码

E0002|供应商不存在|请检查供应商名称或编码
