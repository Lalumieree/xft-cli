批量新增数据源选项

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/addMultilevelOption

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/addMultilevelOption

接口描述

批量新增数据源选项

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootARRAY请求体根节点
root.selectOptionNameSTRING是一级选项名称选项1
root.optionSaveDtoListARRAY否树形结构，目前最多支持三级[{"selectOptionName":"选项1"}]
root.optionSaveDtoList.selectOptionNameSTRING二级选项名称选项11
root.optionSaveDtoList.optionSaveDtoListARRAY树形结构，目前最多支持三级[{"selectOptionName":"选项1"}]
参数类型必填说明示例
rootOBJECTquery根节点
root.sourceSeqIdLONG是数据源自增ID，通过分页查询数据源可得1

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN否true:请求成功
false:请求失败		true
root.messageSTRING否成功时：调用成功
失败时：失败的具体信息		调用成功
