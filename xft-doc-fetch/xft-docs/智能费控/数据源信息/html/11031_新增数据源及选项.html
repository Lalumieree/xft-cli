新增数据源及选项

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/addMultilevelSourceAndOption

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/addMultilevelSourceAndOption

接口描述

新增数据源及选项接口

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.idSTRING是数据源IDSOUR01
body.selectSourceNameSTRING是自定义的数据源名称数据源1号
body.optionSaveDtoListARRAY否树形结构，最多支持3层，具体请参考请求示例[ { "seqId": 1214878, "selectOptionName": "选项12", "optionSaveDtoList": null } ]
body.optionSaveDtoList.selectOptionNameSTRING否optionSaveDtoList不为空时，改项必填选项1
body.optionSaveDtoList.optionSaveDtoListARRAY否选项层次 最多3层[ { "seqId": 1214878, "selectOptionName": "选项12", "optionSaveDtoList": null } ]
参数类型必填说明示例
queryOBJECTquery根节点

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
successOBJECT成功标志
success.successBOOLEAN否true：请求成功
false:请求失败		true
success.messageSTRING否返回信息新增数据源及选项成功

	"success":  true,

}
