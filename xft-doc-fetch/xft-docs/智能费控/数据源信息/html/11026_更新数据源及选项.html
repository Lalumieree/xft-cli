更新数据源及选项

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/updateMultilevelSourceAndOption

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/updateMultilevelSourceAndOption

接口描述

**用于费控单据配置下的自定义字段的自定义选项数据源的更新**

更新 费控 单据配置 --> 自定义字段 --> 层级选择 --> 数据源及选项

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.idSTRING是数据源idSOUR02
body.selectSourceNameSTRING是自定义数据源名称数据源1号
body.optionSaveDtoListARRAY否树形结构。选项层级，最多支持三层[{"selectOptionName": "选项12"}]
body.optionSaveDtoList.selectOptionNameSTRING否选项名称选项1
body.optionSaveDtoList.optionSaveDtoListARRAY否子选项集合
层级字段一致（与上一级结构、字段一样）
会存在不固定的多级
最多支持3级		[{"selectOptionName": "选项12"}]
body.optionSaveDtoList.optionSaveDtoList.selectOptionNameSTRING子级选项名称子选项1
参数类型必填说明示例
queryOBJECTquery根节点

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.successBOOLEAN否true: 请求成功
false：请求失败		true
body.messageSTRING否成功时：调用成功
失败时：失败的具体信息		调用成功
