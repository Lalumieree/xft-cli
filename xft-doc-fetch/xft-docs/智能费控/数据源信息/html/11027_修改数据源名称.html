修改数据源名称

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/sourceId/{sourceId}/updateMultilevelSourceName

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/sourceId/{sourceId}/updateMultilevelSourceName

接口描述

费控 单据配置 --> 自定义字段 --> 层级选择 --> 对应选项数据源 ： 根据数据源ID修改数据源名称

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.sourceIdSTRING是数据源Id1000
query.selectSourceNameSTRING是更新后的数据源名称数据源1号改
query.OPAUIDSTRING是公司标识XFT000000

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.successBOOLEAN是成功标志true
body.messageSTRING否返回信息修改数据源名称成功

接口调用示例

###请求示例


{BaseURL}/api/itrip-gateway/v1/dataSource/sourceId/123/updateMultilevelSourceName?OPAUID=XFT000000&selectSourceName=数据源1号改

返回示例


{ 
	   "success":  true,
	   "message":  "修改数据源名称成功"
}
