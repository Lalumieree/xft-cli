查询数据源详细信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/getMultiLevelDetail/{sourceSeqId}

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/getMultiLevelDetail/{sourceSeqId}

接口描述

**用于费控单据配置下的自定义字段的自定义选项根据数据源自增ID查询数据源详细信息**

单据配置 --> 自定义字段 --> 层级选择 --> 对应选项数据源 ： 根据数据源自增ID查询数据源详细信息

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.sourceSeqIdLONG是路径占位符参数1000

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.dataOBJECT否返回数据
body.data.idSTRING否数据源idSOUR02
body.data.selectSourceNameSTRING否自定义数据源名称数据源1号
body.data.optionSaveDtoListARRAY否树形结构，目前最多支持3层
body.data.optionSaveDtoList.seqIdLONG否一级选项唯一键1214876
body.data.optionSaveDtoList.selectOptionNameSTRING否一级选项名称选项1
body.data.optionSaveDtoList.optionSaveDtoListARRAY否* 子选项集合
* 层级字段一致（与上一级结构、字段一样）
* 会存在不固定的多级
* 目前最多支持3层
body.data.optionSaveDtoList.optionSaveDtoList.seqIdLONG否子级选项唯一键1214877
body.data.optionSaveDtoList.optionSaveDtoList.selectOptionNameSTRING否子级选项名称选项11
body.data.comIdLONG公司ID111
body.successBOOLEAN否true:请求成功
false:请求失败		true
body.messageSTRING否成功时：调用成功
失败时：具体的失败信息		调用成功！
