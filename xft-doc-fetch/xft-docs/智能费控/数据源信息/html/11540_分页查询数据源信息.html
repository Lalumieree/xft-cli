分页查询数据源信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/findMultilevelSourcePage

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/findMultilevelSourcePage

接口描述

费控 单据配置 --> 自定义字段 --> 层级选择 --> 对应选项数据源 ：分页查询企业配置数据源信息

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.idsARRAY数据源id精确匹配，建议不超过1000个["SOUR01"]
body.selectSourceNameSTRING自定义的数据源名称数据源1
参数类型必填说明示例
queryOBJECTquery根节点
query.startSTRING不传，默认为00
query.limitSTRING查询行数100

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.listARRAY否数据源列表
body.list.seqIdLONG否数据源自增ID 主键1
body.list.idSTRING否数据源IDSOUR01
body.list.selectSourceNameSTRING否自定义的数据源名称数据源1号
body.list.comIdLONG否公司ID3117
body.totalCountLONG否总数1
body.successSTRING是true：请求成功
false: 请求失败		true
body.messageSTRING是请求成功：调用成功！
失败时：请求失败的消息的异常消息		调用成功！

接口调用示例

###请求示例

{BaseURL}/api/itrip-gateway/v1/dataSource/findMultilevelSourcePage?start=0&limit=100


{
	"ids": ["SOUR01"],
	"selectSourceName": "数据源"
}

返回示例


{
    "success": true,
    "message": "调用成功！",
    "totalCount": 1,
    "list": [
        {
            "seqId": 150,
            "id": "SOUR01",
            "comId": 3648,
            "selectSourceName": "数据源1号"
        }
    ]
}
