# 更新数据源及选项

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/updateMultilevelSourceAndOption 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/updateMultilevelSourceAndOption 
 ## 接口描述 
**用于费控单据配置下的自定义字段的自定义选项数据源的更新**

更新 费控 单据配置 --> 自定义字段 --> 层级选择 --> 数据源及选项 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.id | STRING | 是 | 数据源id | SOUR02 |
| body.selectSourceName | STRING | 是 | 自定义数据源名称 | 数据源1号 |
| body.optionSaveDtoList | ARRAY | 否 | 树形结构。选项层级，最多支持三层 | [{"selectOptionName": "选项12"}] |
| body.optionSaveDtoList.selectOptionName | STRING | 否 | 选项名称 | 选项1 |
| body.optionSaveDtoList.optionSaveDtoList | ARRAY | 否 | 子选项集合<br>层级字段一致（与上一级结构、字段一样）<br>会存在不固定的多级<br>最多支持3级 | [{"selectOptionName": "选项12"}] |
| body.optionSaveDtoList.optionSaveDtoList.selectOptionName | STRING |  | 子级选项名称 | 子选项1 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.success | BOOLEAN | 否 | true: 请求成功<br>false：请求失败 | true |
| body.message | STRING | 否 | 成功时：调用成功<br>失败时：失败的具体信息 | 调用成功 |
 
## 接口调用示例 
 ###请求示例
{BaseURL}/api/itrip-gateway/v1/dataSource/updateMultilevelSourceAndOption
```json
{
        "id": "SOUR02",
        "selectSourceName": "数据源1号",
        "optionSaveDtoList": [
            {
                "selectOptionName": "选项1",
                "optionSaveDtoList": [
                    {
                        "selectOptionName": "选项11",
                        "optionSaveDtoList": [
                            {
                                "selectOptionName": "选项12",
                                "optionSaveDtoList": null
                            }
                        ]
                    }
                ]
            }
        ]
    }
```
### 返回示例
```json
{ 
	   "success":  true,
	   "message":  "调用成功"
}
```
