# 数据源批量删除

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/deleteMultilevelSourceByIds 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/deleteMultilevelSourceByIds 
 ## 接口描述 
数据源批量删除 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | ARRAY | 是 | 要删除的数据源id列表，每个数据源ID长度上限128 | [ "SOUR01"] |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.success | BOOLEAN | 否 | 成功标志 | true |
| body.message | STRING | 否 | 返回信息 | 数据源批量删除成功 |
 
## 接口调用示例 
 ###请求示例
{BaseURL}/api/itrip-gateway/v1/dataSource/deleteMultilevelSourceByIds
```json
  ["SOUR01"]
```
### 返回示例
```json
{ 
	   "success":  true,
	   "message":   "数据源批量删除成功"
}
```
