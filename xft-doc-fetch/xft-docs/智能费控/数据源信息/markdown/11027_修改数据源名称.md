# 修改数据源名称

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/sourceId/{sourceId}/updateMultilevelSourceName 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/sourceId/{sourceId}/updateMultilevelSourceName 
 ## 接口描述 
费控 单据配置 --> 自定义字段 --> 层级选择 --> 对应选项数据源 ： 根据数据源ID修改数据源名称 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.sourceId | STRING | 是 | 数据源Id | 1000 |
| query.selectSourceName | STRING | 是 | 更新后的数据源名称 | 数据源1号改 |
| query.OPAUID | STRING | 是 | 公司标识 | XFT000000 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.success | BOOLEAN | 是 | 成功标志 | true |
| body.message | STRING | 否 | 返回信息 | 修改数据源名称成功 |
 
## 接口调用示例 
 ###请求示例
```json
{BaseURL}/api/itrip-gateway/v1/dataSource/sourceId/123/updateMultilevelSourceName?OPAUID=XFT000000&selectSourceName=数据源1号改
```

### 返回示例
```json
{ 
	   "success":  true,
	   "message":  "修改数据源名称成功"
}
```
