# 批量新增数据源选项

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/addMultilevelOption 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/addMultilevelOption 
 ## 接口描述 
批量新增数据源选项 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | ARRAY |  | 请求体根节点 |  |
| root.selectOptionName | STRING | 是 | 一级选项名称 | 选项1 |
| root.optionSaveDtoList | ARRAY | 否 | 树形结构，目前最多支持三级 | [{"selectOptionName":"选项1"}] |
| root.optionSaveDtoList.selectOptionName | STRING |  | 二级选项名称 | 选项11 |
| root.optionSaveDtoList.optionSaveDtoList | ARRAY |  | 树形结构，目前最多支持三级 | [{"selectOptionName":"选项1"}] |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
| root.sourceSeqId | LONG | 是 | 数据源自增ID，通过分页查询数据源可得 | 1 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.success | BOOLEAN | 否 | true:请求成功<br>false:请求失败 | true |
| root.message | STRING | 否 | 成功时：调用成功<br>失败时：失败的具体信息 | 调用成功 |
 
## 接口调用示例 
 ###请求示例
{BaseURL}/api/itrip-gateway/v1/dataSource/addMultilevelOption?sourceSeqId=12345
```json
[  
  	{  
		"selectOptionName":"选项1",  
		"optionSaveDtoList": [
			{
				"selectOptionName":"选项11",  
				"optionSaveDtoList": [
					{
						"selectOptionName":"选项111",  
						"optionSaveDtoList": []  
					}
				]  
			}	
		]  
  	}  
]  
```
### 返回示例
```json
{ 
	   "success":  true,
	   "message":  "调用成功"
}
```
