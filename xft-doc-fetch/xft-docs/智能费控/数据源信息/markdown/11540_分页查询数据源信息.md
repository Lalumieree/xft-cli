# 分页查询数据源信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/findMultilevelSourcePage 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/findMultilevelSourcePage 
 ## 接口描述 
费控 单据配置 --> 自定义字段 --> 层级选择 --> 对应选项数据源 ：分页查询企业配置数据源信息  
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.ids | ARRAY |  | 数据源id精确匹配，建议不超过1000个 | ["SOUR01"] |
| body.selectSourceName | STRING |  | 自定义的数据源名称 | 数据源1 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.start | STRING |  | 不传，默认为0 | 0 |
| query.limit | STRING |  | 查询行数 | 100 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.list | ARRAY | 否 | 数据源列表 |  |
| body.list.seqId | LONG | 否 | 数据源自增ID 主键 | 1 |
| body.list.id | STRING | 否 | 数据源ID | SOUR01 |
| body.list.selectSourceName | STRING | 否 | 自定义的数据源名称 | 数据源1号 |
| body.list.comId | LONG | 否 | 公司ID | 3117 |
| body.totalCount | LONG | 否 | 总数 | 1 |
| body.success | STRING | 是 | true：请求成功<br>false: 请求失败 | true |
| body.message | STRING | 是 | 请求成功：调用成功！<br>失败时：请求失败的消息的异常消息 | 调用成功！ |
 
## 接口调用示例 
 ###请求示例
{BaseURL}/api/itrip-gateway/v1/dataSource/findMultilevelSourcePage?start=0&limit=100
```json
{
	"ids": ["SOUR01"],
	"selectSourceName": "数据源"
}
```
### 返回示例
```json
{
    "success": true,
    "message": "调用成功！",
    "totalCount": 1,
    "list": [
        {
            "seqId": 150,
            "id": "SOUR01",
            "comId": 3648,
            "selectSourceName": "数据源1号"
        }
    ]
}
```

| ERR0000      | 解密失败 | 无                            |
