# 查询数据源详细信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/getMultiLevelDetail/{sourceSeqId} 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/getMultiLevelDetail/{sourceSeqId} 
 ## 接口描述 
**用于费控单据配置下的自定义字段的自定义选项根据数据源自增ID查询数据源详细信息**

单据配置 --> 自定义字段 --> 层级选择 --> 对应选项数据源 ：  根据数据源自增ID查询数据源详细信息 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json
