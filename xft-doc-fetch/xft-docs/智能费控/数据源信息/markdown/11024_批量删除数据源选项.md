# 批量删除数据源选项

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/deleteMultilevelOptions 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/deleteMultilevelOptions 
 ## 接口描述 
批量删除数据源选项 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 | **参数**     | **类型**   | **必填** | **参数** | **长度** | **说明**     | **备注**                  |
| ------------ | ---------- | -------- | -------- | -------- | ------------ | ------------------------- |
| OPAUID       | String     | 是       | param    | 8        | 公司标识     | 传薪福通企业号            |
| sourceSeqId  | Long       | 是       | param    | 20       | 数据源自增id | -                         |
| optionSeqIds | Array:Long | 是       | body     | -        | 删除选项列表 | 每个optionSeqId最大长度20 | 
## 接口输出参数
 | **参数**                  | **类型**                      | **说明**    | **备注**        |
|----------------|-----------------|----------------------|----------------|
| success      | boolean  | 成功标志 |-
| message      | String   | 返回信息 |- 
## 接口调用示例 
 ###请求示例
{BaseURL}/api/itrip-gateway/v1/dataSource/deleteMultilevelOptions?OPAUID=ITRIP_123&sourceSeqId=123
```json
[1, 2]
```

### 返回示例
```json
{ 
	   "success":  true,
	   "message":  "调用成功"
}
```
