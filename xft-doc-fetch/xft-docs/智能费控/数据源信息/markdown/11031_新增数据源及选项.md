# 新增数据源及选项

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/dataSource/addMultilevelSourceAndOption 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/dataSource/addMultilevelSourceAndOption 
 ## 接口描述 
新增数据源及选项接口 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.id | STRING | 是 | 数据源ID | SOUR01 |
| body.selectSourceName | STRING | 是 | 自定义的数据源名称 | 数据源1号 |
| body.optionSaveDtoList | ARRAY | 否 | 树形结构，最多支持3层，具体请参考请求示例 | [ { "seqId": 1214878, "selectOptionName": "选项12", "optionSaveDtoList": null } ] |
| body.optionSaveDtoList.selectOptionName | STRING | 否 | optionSaveDtoList不为空时，改项必填 | 选项1 |
| body.optionSaveDtoList.optionSaveDtoList | ARRAY | 否 | 选项层次 最多3层 | [ { "seqId": 1214878, "selectOptionName": "选项12", "optionSaveDtoList": null } ] |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| success | OBJECT |  | 成功标志 |  |
| success.success | BOOLEAN | 否 | true：请求成功<br>false:请求失败 | true |
| success.message | STRING | 否 | 返回信息 | 新增数据源及选项成功 |
 
## 接口调用示例 
 ###请求示例
{BaseURL}/api/itrip-gateway/v1/dataSource/addMultilevelSourceAndOption?OPAUID=XFT00001
```json
{

        "id": "SOUR02",
        "selectSourceName": "数据源1号-1",
        "optionSaveDtoList": [
            {
                "selectOptionName": "选项1",
                "optionSaveDtoList": [
                    {
                        "selectOptionName": "选项11",
                        "optionSaveDtoList": [
                            {
                                "selectOptionName": "选项12",
                                "optionSaveDtoList": null
                            }
                        ]
                    }
                ]
            }
        ]
    }
```
### 返回示例
```json
{
	"success":  true,
	"message":  "新增数据源及选项成功"
}
```
