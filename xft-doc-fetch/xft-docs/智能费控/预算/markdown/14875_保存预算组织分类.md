# 保存预算组织分类

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/org-type/save 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/org-type/save 
 ## 接口描述 
本接口适用于单条新增预算组织分类到智能费控
调用本接口后可在“智能费控-系统设置-预算设置-预算组织“中看到添加的预算组织分类。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.opsEmpNumber | STRING | 是 | 操作人花名册编码 | 001 |
| root.budgetOrgTypeId | STRING | 是 | 预算组织编号仅支持50位以内数字和字母 | test0101 |
| root.budgetOrgTypeName | STRING | 是 | 预算组织分类名称 | api分类 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.success | BOOLEAN | 否 | true说明接口调用成功 false说明接口调用失败 | true |
| root.message | STRING | 否 | 接口调用返回信息 | 调用成功！ |
 
## 接口调用示例 
 ### 请求报文
```json
{
  "opsEmpNumber": "001",
  "budgetOrgTypeId": "test0101",
  "budgetOrgTypeName": "api分类"
}
```
### 调用成功的响应报文
```json
{
    "success": true,
    "message": "调用成功！"
}
```
### 调用失败的响应报文，示例仅为其中一部分失败样例
```json
{
    "code": "E0002",
    "message": "预算组织分类编码只允许数字和字母;"
}
```  
## 错误码说明 
sucess为导入成功状态， true为成功 其余都是失败
| 常见报错|
| --- |
| 预算组织分类编码只允许数字和字母;|
