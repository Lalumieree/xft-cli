# 预算方案详情查询

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/get-budget-parent-plan-detail 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/get-budget-parent-plan-detail 
 ## 接口描述 
本接口适用于查询智能费控预算方案详情，包含预算方案基础信息和相关配置项。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
| root.planSeqId | LONG | 是 | 预算方案自增id | 100 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.body | OBJECT | 否 | 预算方案详情 |  |
| root.body.budgetParentPlanSeqId | LONG | 否 | 预算方案自增id | 100 |
| root.body.budgetName | STRING | 否 | 预算方案名称 | 测试预算方案 |
| root.body.budgetStartDate | STRING | 否 | 格式为yyyy-MM-dd | 2024-01-01 |
| root.body.budgetEndDate | STRING | 否 | 格式为yyyy-MM-dd | 2026-12-31 |
| root.body.budgetDimension | STRING | 否 | 1-按预算组织维度<br>2-按预算科目维度<br>3-按预算组织+预算科目维度 | 3 |
| root.body.budgetState | STRING | 否 | 0-停用<br>1-启用 | 1 |
| root.body.eventIds | ARRAY | 否 | 表单自增id列表，最多支持关联100个表单<br>如果不填，则默认为适用所有的表单 | [1,2,3] |
| root.body.budgetPlanId | STRING | 否 | 预算方案编码 | test001 |
| root.body.budgetOrgs | ARRAY | 否 | 预算组织自增id列表，最多支持关联100个预算组织<br>如果不填，则默认为支持所有的预算组织 | [1,2,3] |
| root.body.budgetSubs | ARRAY | 否 | 预算科目自增id列表，最多支持关联100个预算科目<br>如果不填，则默认为支持所有的预算科目 | [1,2,3] |
| root.body.budgetApproveTime | STRING | 否 | NONE-不需要审批<br><br>WHEN_INIT-期初预算编制需审批<br><br>WHEN_MODIFY-预算追加调整需审批<br><br>ANYTIME-期初预算编制/预算追加调整均需审批 | NONE |
| root.body.needPostAccount | STRING | 否 | 0-无需过账<br>1-需要过账 | 0 |
| root.body.allowExternalSystemBill | STRING | 否 | 0-不允许<br>1-允许 | 0 |
| root.body.allowMonthOrderBudgetOccupy | STRING | 否 | 0-不支持<br>1-支持 | 0 |
| root.body.budgetRangeFixed | STRING | 否 | 0=不固定<br>1-固定 | 0 |
| root.returnCode | STRING | 否 | SUC0000-成功<br>E0001-系统错误<br>E0002-参数错误 | SUC0000 |
| root.success | BOOLEAN | 否 | true说明接口调用成功 <br>false说明接口调用失败 | true |
| root.message | STRING | 否 | 接口调用返回信息 | 成功 |
