# 预算方案预算使用情况查询

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/page-query-budget-detail 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/page-query-budget-detail 
 ## 接口描述 
本接口适用于通过预算方案编码、预算方案名称、预算周期类型、预算组织编码、预算科目名称、预算期数查询智能费控预算方案使用详情，包含预算方案基础信息、预算方案总期数、预算方案每期额度等信息。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
