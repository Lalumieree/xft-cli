# 根据预算方案id查询预算子方案详细信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/budget-sub-plan/list-by-parent 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/budget-sub-plan/list-by-parent 
 ## 接口描述 
本接口适用于根据预算方案id查询方案下所有子方案信息的列表，包含子方案的所有详细信息。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.budgetParentPlanSeqId | LONG | 是 | 预算方案id | 1286 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.data | ARRAY | 否 | 接口返回添加结果 |  |
| root.data.budgetPlanSeqId | LONG | 否 | 预算子方案id | 1579 |
| root.data.budgetParentPlanSeqId | LONG | 否 | 预算方案id | 1286 |
| root.data.budgetName | STRING | 否 | 预算方案名称 | 异常1118 |
| root.data.budgetDimension | STRING | 否 | 1按照预算部门,2按照预算科目,3按照预算部门/预算科目 | 1 |
| root.data.budgetEventControl | STRING | 否 | 0部分,1全部 | 0 |
| root.data.budgetState | STRING | 否 | 0未启用,1启用 | 1 |
| root.data.budgetCycle | STRING | 否 | 1月度预算,2季度预算,3年度预算 | 1 |
| root.data.budgetControlType | STRING | 否 | 1不得提交,2不控制 | 0 |
| root.data.budgetWarningType | STRING | 否 | 1预警提示,2不预警 | 2 |
| root.data.budgetWarningRate | STRING | 否 | 预算预警百分比 | null |
| root.data.budgetTurnType | STRING | 否 | 0不顺延,1每个编著周期末自动顺延 | 1 |
| root.data.cycleControl | STRING | 否 | 1月度 2季度 3年度 | 1 |
| root.data.crtId | STRING | 否 | 创建人ID | 42603 |
| root.success | BOOLEAN | 否 | true说明接口调用成功 false说明接口调用失败 | true |
| root.message | STRING | 否 | 接口调用返回信息 | 调用成功！ |
