# 查询预算组织列表

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/org-type/list 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/org-type/list 
 ## 接口描述 
本接口适用于查看智能费控内本公司下所有的预算组织分类+预算组织列表，呈树状结构，与“智能费控-系统设置-预算设置-预算组织“保持一致。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json

## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.data | ARRAY | 否 | 接口返回数据 |  |
| root.data.seqId | LONG | 否 | 预算组织分类自增id | 1 |
| root.data.budgetOrgTypeId | STRING | 否 | 预算组织分类编码 | test0101546 |
| root.data.budgetOrgTypeName | STRING | 否 | 预算组织分类名称 | api分类 |
| root.data.preset | BOOLEAN | 否 | true -预设，目前公司初始化的时候会预设三个组织分类，分别为部门、项目和个人<br>false-非预设，表示人工创建 | false |
| root.data.children | ARRAY | 否 | 预算组织分类下的预算组织列表 |  |
| root.data.children.budgetOrg | OBJECT | 否 | 预算组织 |  |
| root.data.children.budgetOrg.budgetSeqId | LONG | 否 | 预算组织自增id | 10 |
| root.data.children.budgetOrg.budgetId | STRING | 否 | 预算组织编码 | all0101 |
| root.data.children.budgetOrg.budgetParentId | STRING | 否 | 在分类下面，没有上级组织时为null<br>有上级组织时，为上级组织编码 | parent123 |
| root.data.children.budgetOrg.budgetName | STRING | 否 | 预算组织名称 | 全条件组织 |
| root.data.children.budgetOrg.priority | INTEGER | 否 | 决定预算匹配时，默认匹配的次序，例如某项目组织优先级为1，某部门组织优先级为2，某笔费用都能匹配到两者时，默认匹配项目组织。<br>支持1-99的数字 | 1 |
| root.data.children.budgetOrg.typeSeqId | LONG | 否 | 预算组织分类自增id | 1 |
| root.data.children.budgetOrg.orderNum | INTEGER | 否 | 预算组织展示顺序 | 0 |
| root.data.children.children | ARRAY | 否 | 当前预算组织的下级预算组织列表，与分类下的chidren结构一致<br>没有下级组织时，为null | [] |
| root.data.orderNum | INTEGER | 否 | 预算分类展示顺序 | 1 |
| root.success | BOOLEAN | 否 | true说明接口调用成功<br>false说明接口调用失败 | true |
| root.message | STRING | 否 | 接口调用返回信息 | 调用成功！ |
