# 修改预算子方案

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/budget-sub-plan/edit 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/budget-sub-plan/edit 
 ## 接口描述 
本接口提供预算方案下预算子方案的修改功能。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json

## 接口调用示例 
 ### 请求报文
```json
{
  "budgetParentPlanSeqId": "1286",
  "budgetPlanSeqId": 2003,
  "budgetTurnType": "1",
  "budgetWarningType": 2,
  "budgetCycle": 2,
  "budgetControlType": 1,
  "cycleControl":"1",
  "opsEmpNumber":"18351911114",
  "budgetName":"异常1118","budgetStartDate":"2024-10-01","budgetEndDate":"2024-12-31","budgetDimension":"1"
}
```
### 响应报文
```json
{
    "data": {
        "budgetPlanSeqId": 2003,
        "budgetParentPlanSeqId": 1286,
        "budgetName": "异常1118",
        "budgetDimension": "1",
        "budgetEventControl": "0",
        "budgetState": "1",
        "budgetCycle": "2",
        "budgetControlType": "1",
        "budgetWarningType": "2",
        "budgetWarningRate": null,
        "budgetTurnType": "1",
        "cycleControl": "1",
        "crtId": 42603
```

### 更新失败的响应报文，示例仅为其中一部分失败样例
```json
{

    "message": "业务错误: 预算方案不存在，请检查后重试！"  
} 
```

```json
{  
    "code": "E0002",  
    "message": "预算方案自增ID不能为空;"  
}    
```  
## 错误码说明 
sucess为更新成功状态， true为成功 其余都是失败
| 常见报错| 
  | --- | 
  |预算预警百分比不能为空| 
  |控制周期类型不能小于预算周期| 
  |方案周期类型或控制周期类型不能为空| 
  |控制周期力度必须大于方案周期，请检查输入|
