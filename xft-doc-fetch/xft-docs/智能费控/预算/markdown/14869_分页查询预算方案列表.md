# 分页查询预算方案列表

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/page-budget-parent-plan 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/page-budget-parent-plan 
 ## 接口描述 
本接口适用于分页查询智能费控的预算方案列表
单次最多查询100条，支持方案名称、状态、起止时间和方案维度的搜索。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.opsEmpNumber | STRING | 是 | 操作人花名册编码 | 001 |
| root.limit | INTEGER | 是 | 单页数量最小为1，最大为100 | 50 |
| root.query | OBJECT | 否 | 查询条件 |  |
| root.query.budgetDimension | STRING | 否 | 1-按预算组织维度<br>2-按预算科目维度<br>3-按预算组织+预算科目维度 | 3 |
| root.query.budgetEndDate | STRING | 否 | 格式为yyyy-MM-dd<br>筛选方案结束时间早于或等于传入时间的方案 | 2028-06-06 |
| root.query.budgetName | STRING | 否 | 模糊搜索 | 方案 |
| root.query.budgetStartDate | STRING | 否 | 格式为yyyy-MM-dd<br>筛选方案开始时间晚于或等于传入时间的方案 | 2023-06-06 |
| root.query.budgetState | STRING | 否 | 0-停用<br>1-启用 | 1 |
| root.start | INTEGER | 是 | 开始页面最小为0 | 0 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.data | OBJECT | 否 | 查询数据 |  |
| root.data.total | INTEGER | 否 | 查询总数 | 100 |
| root.data.list | ARRAY | 否 | 查询方案列表 |  |
| root.data.list.budgetParentPlanSeqId | LONG | 否 | 预算方案自增id | 100 |
| root.data.list.budgetName | STRING | 否 | 预算方案名称 | 测试预算方案导入 |
| root.data.list.budgetStartDate | STRING | 否 | 格式为yyyy-MM-dd | 2024-01-01 |
| root.data.list.budgetEndDate | STRING | 否 | 格式为yyyy-MM-dd | 2025-12-31 |
| root.data.list.budgetDimension | STRING | 否 | 1-按预算组织维度<br>2-按预算科目维度<br>3-按预算组织+预算科目维度 | 3 |
| root.data.list.budgetState | STRING | 否 | 0-停用<br>1-启用 | 1 |
| root.data.list.crtName | STRING | 否 | 创建人姓名 | 天 |

| root.message | STRING | 否 | 接口调用返回信息 | 调用成功！ |
 
## 接口调用示例 
 ### 请求报文
```json
{
    "opsEmpNumber": "001",
    "limit": 50,
    "query": {
        "budgetDimension": "3",
        "budgetEndDate": "2028-06-06",
        "budgetName": "测试",
        "budgetStartDate": "2023-06-06",
        "budgetState": "1"
    },
    "start": 0
}
```
### 调用成功的响应报文
```json
{
    "data": {
        "total": 1,
        "list": [
            {
                "budgetParentPlanSeqId": 100,
                "budgetName": "测试预算方案导入",
                "budgetStartDate": "2024-01-01",
                "budgetEndDate": "2026-12-31",
                "budgetDimension": "3",
                "budgetState": "1",
                "crtName": "天"
            }
        ]
    },
    "success": true,
    "message": "调用成功！"
}
```
### 调用失败的响应报文，示例仅为其中一部分失败样例
```json
{
    "code": "E0002",
    "message": "单页数量最小为1;"
}
```  
## 错误码说明 
sucess为接口调用成功状态， true为成功 其余都是失败
| 常见报错|
| --- |
| 单页数量最小为1|
| 单页数量最多100|
| 开始页面最小为0|
