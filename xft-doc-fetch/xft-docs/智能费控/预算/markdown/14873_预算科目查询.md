# 预算科目查询

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/subject/type/list 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/subject/type/list 
 ## 接口描述 
本接口适用于全量查询公司下的预算科目分类+科目列表，呈现树状结构，首层为预算科目分类，子级为预算科目分类下的预算科目列表。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json

## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.data | ARRAY | 否 | 预算科目分类+具体科目列表 |  |
| root.data.budgetSubTypeName | STRING | 否 | 科目分类名称 | 分类 |
| root.data.budgetSubTypeId | STRING | 否 | 科目分类编码 | TYPE001 |
| root.data.children | ARRAY | 否 | 预算科目列表 |  |
| root.data.children.subSeqId | LONG | 否 | 预算科目自增id | 10 |
| root.data.children.subName | STRING | 否 | 预算科目名称 | 科目名 |
| root.data.children.expenseTypes | ARRAY | 否 | 预算科目关联的费用类型列表 |  |
| root.data.children.expenseTypes.expSeqId | LONG | 否 | 费用类型自增id | 1 |
| root.data.children.expenseTypes.subjectSeqId | LONG | 否 | 预算科目自增id | 2 |
| root.data.children.expenseTypes.expId | STRING | 否 | 费用类型ID | 01 |
| root.data.children.expenseTypes.expName | STRING | 否 | 费用类型名称 | 机票费 |
| root.data.children.orderNum | INTEGER | 否 | 预算科目顺序 | 0 |
| root.data.children.subTypeSeqId | LONG | 否 | 预算科目所属分类自增id | 20 |
| root.data.children.amountEntryMode | STRING | 否 | byTotalAmount -总金额<br>standPerDay-标准*人天 | standPerDay |
| root.data.children.standardValue | BIGDECIMAL | 否 | 金额录入模式为 standPerDay-标准*人天时<br>输入每人天的金额值；<br>金额录入模式为 byTotalAmount -总金额时，此处没有值 | 30.2 |
| root.success | BOOLEAN | 否 | true说明接口调用成功<br>false说明接口调用失败 | true |
| root.message | STRING | 否 | 接口调用返回信息 | 调用成功！ |
