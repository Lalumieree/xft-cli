# 新增预算父方案

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/import-budget-parent-plan 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/import-budget-parent-plan 
 ## 接口描述 
本接口适用于单条添加预算方案到智能费控
调用本接口后可在“智能费控-系统设置-预算设置-预算方案“中看到添加的预算方案。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.allowExternalSystemBill | STRING | 是 | 0-不允许<br>1-允许 | 1 |
| root.allowMonthOrderBudgetOccupy | STRING | 是 | 0-不支持<br>1-支持 | 1 |
| root.budgetApproveTime | STRING | 是 | NONE-不需要审批<br><br>WHEN_INIT-期初预算编制需审批<br><br>WHEN_MODIFY-预算追加调整需审批<br><br>ANYTIME-期初预算编制/预算追加调整均需审批 | NONE |
| root.budgetDimension | STRING | 是 | 1-按预算组织维度<br>2-按预算科目维度<br>3-按预算组织+预算科目维度 | 3 |
| root.budgetEndDate | STRING | 是 | 格式为yyyy-MM-dd | 2025-12-31 |
| root.budgetName | STRING | 是 | 预算方案名称 | 测试预算方案导入 |
| root.budgetOrgs | ARRAY | 否 | 预算组织自增id列表，最多支持关联100个预算组织<br>如果不填，则默认为支持所有的预算组织 | [1,2,3] |

| root.budgetStartDate | STRING | 是 | 格式为yyyy-MM-dd | 2025-01-01 |
| root.budgetSubs | ARRAY | 否 | 预算科目自增id列表，最多支持关联100个预算科目<br>如果不填，则默认为支持所有的预算科目 | [1,2,3] |
| root.eventIds | ARRAY | 否 | 表单自增id列表，最多支持关联100个表单<br>如果不填，则默认为适用所有的表单 | [1,2,3] |
| root.needPostAccount | STRING | 是 | 0-无需过账<br>1-需要过账<br>若设置过账后生效，预算单审批通过后需点击过账后预算才能生效<br>预算是否需要审批为NONE时，仅支持无需过账-0 | 0 |
| root.opsEmpNumber | STRING | 是 | 操作人花名册编码 | 001 |

## 接口调用示例 
 ### 请求报文
```json
{
    "allowExternalSystemBill": "1",
    "allowMonthOrderBudgetOccupy": "1",
    "budgetApproveTime": "WHEN_MODIFY",
    "budgetDimension": "3",
    "budgetEndDate": "2025-12-31",
    "budgetName": "测试预算方案导入",
    "budgetOrgs": [1,2,3],
    "budgetPlanId": "test001",
    "budgetRangeFixed": "0",
    "budgetStartDate": "2025-01-01",
    "budgetSubs": [1,2,3],
    "eventIds": [1,2,3],
    "needPostAccount": "0",
    "opsEmpNumber": "001"
}
```
### 导入成功的响应报文
```json
{
    "data": {
        "budgetParentPlanSeqId": 755,
        "budgetName": "测试预算方案导入",
        "budgetDimension": "3"
    },
    "success": true,
    "message": "调用成功！"
}
```
### 导入失败的报文，示例仅为其中一部分失败样例
```json
{
    "success": false,
    "message": "业务错误: 预算组织科目在同一时间不能配置于多个方案"
}
```
### 接口调用失败的报文，示例仅为其中一部分失败样例
```json
{
    "code": "E0002",
    "message": "是否允许外部系统单据使用预算标识不能为空;"
}
```  
## 错误码说明 
sucess为导入成功状态， true为成功，其余都是失败

| 常见报错|
| --- |
| 当前预算方案日期设置错误，请检查|
| 预算组织在同一时间不能配置于多个方案|
| 预算科目在同一时间不能配置于多个方案|
| 预算组织科目在同一时间不能配置于多个方案|
