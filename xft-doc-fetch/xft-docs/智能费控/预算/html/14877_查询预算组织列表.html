查询预算组织列表

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/org-type/list

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/org-type/list

接口描述

本接口适用于查看智能费控内本公司下所有的预算组织分类+预算组织列表，呈树状结构，与“智能费控-系统设置-预算设置-预算组织“保持一致。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataARRAY否接口返回数据
root.data.seqIdLONG否预算组织分类自增id1
root.data.budgetOrgTypeIdSTRING否预算组织分类编码test0101546
root.data.budgetOrgTypeNameSTRING否预算组织分类名称api分类
root.data.presetBOOLEAN否true -预设，目前公司初始化的时候会预设三个组织分类，分别为部门、项目和个人
false-非预设，表示人工创建		false
root.data.childrenARRAY否预算组织分类下的预算组织列表
root.data.children.budgetOrgOBJECT否预算组织
root.data.children.budgetOrg.budgetSeqIdLONG否预算组织自增id10
root.data.children.budgetOrg.budgetIdSTRING否预算组织编码all0101
root.data.children.budgetOrg.budgetParentIdSTRING否在分类下面，没有上级组织时为null
有上级组织时，为上级组织编码		parent123
root.data.children.budgetOrg.budgetNameSTRING否预算组织名称全条件组织
root.data.children.budgetOrg.priorityINTEGER否决定预算匹配时，默认匹配的次序，例如某项目组织优先级为1，某部门组织优先级为2，某笔费用都能匹配到两者时，默认匹配项目组织。
支持1-99的数字		1
root.data.children.budgetOrg.typeSeqIdLONG否预算组织分类自增id1
root.data.children.budgetOrg.orderNumINTEGER否预算组织展示顺序0
root.data.children.childrenARRAY否当前预算组织的下级预算组织列表，与分类下的chidren结构一致
没有下级组织时，为null		[]
root.data.orderNumINTEGER否预算分类展示顺序1
root.successBOOLEAN否true说明接口调用成功
false说明接口调用失败		true
root.messageSTRING否接口调用返回信息调用成功！
