分页查询预算方案列表

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/page-budget-parent-plan

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/page-budget-parent-plan

接口描述

本接口适用于分页查询智能费控的预算方案列表

单次最多查询100条，支持方案名称、状态、起止时间和方案维度的搜索。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.opsEmpNumberSTRING是操作人花名册编码001
root.limitINTEGER是单页数量最小为1，最大为10050
root.queryOBJECT否查询条件
root.query.budgetDimensionSTRING否1-按预算组织维度
2-按预算科目维度
3-按预算组织+预算科目维度		3
root.query.budgetEndDateSTRING否格式为yyyy-MM-dd
筛选方案结束时间早于或等于传入时间的方案		2028-06-06
root.query.budgetNameSTRING否模糊搜索方案
root.query.budgetStartDateSTRING否格式为yyyy-MM-dd
筛选方案开始时间晚于或等于传入时间的方案		2023-06-06
root.query.budgetStateSTRING否0-停用
1-启用		1
root.startINTEGER是开始页面最小为00
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataOBJECT否查询数据
root.data.totalINTEGER否查询总数100
root.data.listARRAY否查询方案列表
root.data.list.budgetParentPlanSeqIdLONG否预算方案自增id100
root.data.list.budgetNameSTRING否预算方案名称测试预算方案导入
root.data.list.budgetStartDateSTRING否格式为yyyy-MM-dd2024-01-01
root.data.list.budgetEndDateSTRING否格式为yyyy-MM-dd2025-12-31
root.data.list.budgetDimensionSTRING否1-按预算组织维度
2-按预算科目维度
3-按预算组织+预算科目维度		3
root.data.list.budgetStateSTRING否0-停用
1-启用		1
root.data.list.crtNameSTRING否创建人姓名天
root.successBOOLEAN否true说明接口调用成功 false说明接口调用失败true
root.messageSTRING否接口调用返回信息调用成功！

接口调用示例

请求报文


{
    "opsEmpNumber": "001",
    "limit": 50,
    "query": {
        "budgetDimension": "3",
        "budgetEndDate": "2028-06-06",
        "budgetName": "测试",
        "budgetStartDate": "2023-06-06",
        "budgetState": "1"
    },
    "start": 0
}

调用成功的响应报文


{
    "data": {
        "total": 1,
        "list": [
            {
                "budgetParentPlanSeqId": 100,
                "budgetName": "测试预算方案导入",

                "budgetDimension": "3",

    "code": "E0002",
    "message": "单页数量最小为1;"
}

错误码说明

sucess为接口调用成功状态， true为成功 其余都是失败

常见报错
单页数量最小为1
单页数量最多100
开始页面最小为0
