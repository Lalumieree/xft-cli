预算方案预算使用情况查询

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/page-query-budget-detail

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/page-query-budget-detail

接口描述

本接口适用于通过预算方案编码、预算方案名称、预算周期类型、预算组织编码、预算科目名称、预算期数查询智能费控预算方案使用详情，包含预算方案基础信息、预算方案总期数、预算方案每期额度等信息。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.budgetPlanIdSTRING是预算方案编码不能为空BUDGETPLAN1670331363596
root.budgetPlanNameSTRING非必填，传值则作为查询条件预算方案部门科目
root.budgetCycleSTRING是1:月度预算，2:季度预算，3:年度预算。1
root.budgetOrgIdListARRAY否1，预算组织编码集合不能超过100个
2，查询条件预算组织科目和预算组织不可同时为空		["BUDGETORG0005"]
root.budgetSubjectNameListARRAY1，预算科目名称集合不能超过100个
2，查询条件预算组织科目和预算组织不可同时为空		["测试科目120601"]
root.periodNumbersARRAY1，非必填，传值则作为查询条件
2，预算期数集合不能超过100个		[1,2]
root.startINTEGER是1，分页起始位置不能为空
2，分页起始位置最小从0开始		0
root.limitINTEGER是1，最大支持100条一页
2，查询每页大小最小为1
3，查询每页大小不能为空
4，默认值20		10
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataOBJECT是返回数据
root.data.totalLONG否返回数据总数1
root.data.listARRAY否返回值集合
root.data.list.budgetPlanIdLONG否预算方案编码BUDGETPLAN1670331363596
root.data.list.budgetPlanNameSTRING否预算方案名称预算方案部门科目
root.data.list.budgetCycleSTRING否1:月度预算，2:季度预算，3:年度预算。1
root.data.list.budgetDimensionSTRING否1:按照预算组织，2:按照预算科目，3:按照预算组织/预算科目。3
root.data.list.orgIdSTRING否预算组织编码BUDGETORG0005
root.data.list.orgNameSTRING否预算组织名称开发一组
root.data.list.subjectNameSTRING否预算科目名称测试科目120601
root.data.list.orgOtherStateBOOLEAN否false:否，true:是。false
root.data.list.subjectOtherStateBOOLEAN否false:否，true:是。false
root.data.list.periodINTEGER否总期数2
root.data.list.quotaOfEachPeriodsARRAY否包含每期额度使用的具体信息
root.data.list.quotaOfEachPeriods.periodNumberINTEGER否预算期数1
root.data.list.quotaOfEachPeriods.initialQuotaBIGDECIMAL否17位整数，2位小数400
root.data.list.quotaOfEachPeriods.usedQuotaBIGDECIMAL否17位整数，2位小数0
root.data.list.quotaOfEachPeriods.freezedQuotaBIGDECIMAL否17位整数，2位小数0
root.data.list.quotaOfEachPeriods.remainQuotaBIGDECIMAL否17位整数，2位小数400
root.data.list.quotaOfEachPeriods.reservedQuotaBIGDECIMAL否17位整数，2位小数0
root.data.list.quotaOfEachPeriods.startDateDATE否时间格式为yyyy-MM-dd hh:mm:ss2022-10-01 00:00:00
root.data.list.quotaOfEachPeriods.endDateDATE否时间格式为yyyy-MM-dd hh:mm:ss2022-10-31 00:00:00
root.successBOOLEAN是false:否，true:是。true
root.messageSTRING是接口返回信息调用成功！
