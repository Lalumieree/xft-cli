预算组织详情查询

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/organization/detail

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/organization/detail

接口描述

本接口适用于查询预算组织详情信息，包含预算组织基础信息、承担条件、组织负责人和报表查询人。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.orgSeqIdLONG否预算组织自增id和预算组织编码集合查询条件不可同时为空100
root.organizationIdsARRAY预算组织自增id和预算组织编码集合查询条件不可同时为空，
预算组织编码集合查询条件不可超过100		[ "BUDGETORG20231212084255521eedcd" ]
参数类型必填说明示例
