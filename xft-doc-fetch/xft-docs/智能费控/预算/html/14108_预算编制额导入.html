预算编制额导入

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/import/quota

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/import/quota

接口描述

支持外部系统将预算方案编制额度导入系统。

支持导入编制额进行覆盖和追加两种。

预算导入额度不能超过1000条。

不支持部分成功。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT是请求体根节点
root.budgetPlanNameSTRING是预算方案名称招待预算
root.budgetCycleSTRING是1 月度预算
2 季度预算
3 年度预算		1
root.budgetPlanIdSTRING是预算方案编码XXX0001
root.operationTypeSTRING否0-覆盖编制额
1-追加编制额
不传默认为覆盖编制额		0
root.budgetPlanEstablishQuotasARRAY是最多支持1000条预算编制
root.budgetPlanEstablishQuotas.orgIdSTRING否预算维度为1、3时必填。当预算组织为“其他”时填“#”BUDGETORG20230317154944
root.budgetPlanEstablishQuotas.orgNameSTRING否预算组织名称测试招待
root.budgetPlanEstablishQuotas.subjectNameSTRING否预算维度为2、3时必填测试预算科目
root.budgetPlanEstablishQuotas.quotaOfEachPeriodsARRAY是最多支持60期
root.budgetPlanEstablishQuotas.quotaOfEachPeriods.periodNumberINTEGER是值应在所在方案的预算期数内1
root.budgetPlanEstablishQuotas.quotaOfEachPeriods.initialQuotaBIGDECIMAL否整数位不能超过9位，小数位不能超过2位

操作类型为0-覆盖，或者没有传操作类型时，预算编制初始额不可为空		1000.00
root.budgetPlanEstablishQuotas.quotaOfEachPeriods.addQuotaBIGDECIMAL否整数位不能超过9位，小数位不能超过2位
操作类型为1-追加时，预算编制追加额不可为空		1000.00
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN是否导入成功true
root.messageSTRING明细信息调用成功

接口调用示例

编制额覆盖请求报文


{
  "budgetPlanName": "222",
  "budgetCycle": "1",
  "budgetPlanId": "2222",
  "operationType":"0"
  "budgetPlanEstablishQuotas": [
    {
      "orgId": "BUDGETORG20230317154944",
      "orgName": "测试招待",
      "quotaOfEachPeriods": [
          {
            "periodNumber": 0,
            "initialQuota": 10000000.00
          }
       ]
    }
  ]
}

编制额追加请求报文


{
  "budgetPlanName": "222",
  "budgetCycle": "1",
  "budgetPlanId": "2222",
  "operationType":"1"
  "budgetPlanEstablishQuotas": [
    {
      "orgId": "BUDGETORG20230317154944",
      "orgName": "测试招待",
      "quotaOfEachPeriods": [
          {
            "periodNumber": 0,
            "addQuota": 10000000.00
          }
       ]
    }
  ]
}

响应报文


{
  "success": true,
  "message": "调用成功！"
}
