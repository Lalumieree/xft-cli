新增预算父方案

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/import-budget-parent-plan

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/import-budget-parent-plan

接口描述

本接口适用于单条添加预算方案到智能费控

调用本接口后可在“智能费控-系统设置-预算设置-预算方案“中看到添加的预算方案。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.allowExternalSystemBillSTRING是0-不允许
1-允许		1
root.allowMonthOrderBudgetOccupySTRING是0-不支持
1-支持		1
root.budgetApproveTimeSTRING是NONE-不需要审批

WHEN_INIT-期初预算编制需审批

WHEN_MODIFY-预算追加调整需审批

ANYTIME-期初预算编制/预算追加调整均需审批		NONE
root.budgetDimensionSTRING是1-按预算组织维度
2-按预算科目维度
3-按预算组织+预算科目维度		3
root.budgetEndDateSTRING是格式为yyyy-MM-dd2025-12-31
root.budgetNameSTRING是预算方案名称测试预算方案导入
root.budgetOrgsARRAY否预算组织自增id列表，最多支持关联100个预算组织
如果不填，则默认为支持所有的预算组织		[1,2,3]
root.budgetPlanIdSTRING否预算方案编码必须是50位以内的字母或数字test001
root.budgetRangeFixedSTRING是0=不固定
1-固定
固定区间的方案不支持编辑方案的开始和结束时间		0
root.budgetStartDateSTRING是格式为yyyy-MM-dd2025-01-01
root.budgetSubsARRAY否预算科目自增id列表，最多支持关联100个预算科目
如果不填，则默认为支持所有的预算科目		[1,2,3]
root.eventIdsARRAY否表单自增id列表，最多支持关联100个表单
如果不填，则默认为适用所有的表单		[1,2,3]
root.needPostAccountSTRING是0-无需过账
1-需要过账
若设置过账后生效，预算单审批通过后需点击过账后预算才能生效
预算是否需要审批为NONE时，仅支持无需过账-0		0
root.opsEmpNumberSTRING是操作人花名册编码001
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataOBJECT否接口返回导入结果
root.data.budgetParentPlanSeqIdLONG否预算方案自增id100
root.data.budgetNameSTRING否预算方案名称测试预算方案导入
root.data.budgetDimensionSTRING否1-按预算组织维度
2-按预算科目维度
3-按预算组织+预算科目维度		3
root.successBOOLEAN否true说明接口调用成功
false说明接口调用失败		true
root.messageSTRING否接口调用返回信息调用成功！

接口调用示例

请求报文


{
    "allowExternalSystemBill": "1",
    "allowMonthOrderBudgetOccupy": "1",
    "budgetApproveTime": "WHEN_MODIFY",
    "budgetDimension": "3",
    "budgetEndDate": "2025-12-31",
    "budgetName": "测试预算方案导入",
    "budgetOrgs": [1,2,3],
    "budgetPlanId": "test001",
    "budgetRangeFixed": "0",
    "budgetStartDate": "2025-01-01",
    "budgetSubs": [1,2,3],
    "eventIds": [1,2,3],
    "needPostAccount": "0",
    "opsEmpNumber": "001"
}

导入成功的响应报文


{
    "data": {
        "budgetParentPlanSeqId": 755,
        "budgetName": "测试预算方案导入",
        "budgetDimension": "3"
    },
    "success": true,
    "message": "调用成功！"
}

导入失败的报文，示例仅为其中一部分失败样例


{
    "success": false,
    "message": "业务错误: 预算组织科目在同一时间不能配置于多个方案"
}

接口调用失败的报文，示例仅为其中一部分失败样例


{
    "code": "E0002",
    "message": "是否允许外部系统单据使用预算标识不能为空;"
}

错误码说明

sucess为导入成功状态， true为成功，其余都是失败

常见报错
当前预算方案日期设置错误，请检查
预算组织在同一时间不能配置于多个方案
预算科目在同一时间不能配置于多个方案
预算组织科目在同一时间不能配置于多个方案
