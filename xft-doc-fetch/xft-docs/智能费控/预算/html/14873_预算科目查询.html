预算科目查询

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/subject/type/list

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/subject/type/list

接口描述

本接口适用于全量查询公司下的预算科目分类+科目列表，呈现树状结构，首层为预算科目分类，子级为预算科目分类下的预算科目列表。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataARRAY否预算科目分类+具体科目列表
root.data.budgetSubTypeNameSTRING否科目分类名称分类
root.data.budgetSubTypeIdSTRING否科目分类编码TYPE001
root.data.childrenARRAY否预算科目列表
root.data.children.subSeqIdLONG否预算科目自增id10
root.data.children.subNameSTRING否预算科目名称科目名
root.data.children.expenseTypesARRAY否预算科目关联的费用类型列表
root.data.children.expenseTypes.expSeqIdLONG否费用类型自增id1
root.data.children.expenseTypes.subjectSeqIdLONG否预算科目自增id2
root.data.children.expenseTypes.expIdSTRING否费用类型ID01
root.data.children.expenseTypes.expNameSTRING否费用类型名称机票费
root.data.children.orderNumINTEGER否预算科目顺序0
root.data.children.subTypeSeqIdLONG否预算科目所属分类自增id20
root.data.children.amountEntryModeSTRING否byTotalAmount -总金额
standPerDay-标准*人天		standPerDay
root.data.children.standardValueBIGDECIMAL否金额录入模式为 standPerDay-标准*人天时
输入每人天的金额值；
金额录入模式为 byTotalAmount -总金额时，此处没有值		30.2
root.successBOOLEAN否true说明接口调用成功
false说明接口调用失败		true
root.messageSTRING否接口调用返回信息调用成功！
