保存预算组织分类

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/org-type/save

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/org-type/save

接口描述

本接口适用于单条新增预算组织分类到智能费控

调用本接口后可在“智能费控-系统设置-预算设置-预算组织“中看到添加的预算组织分类。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.opsEmpNumberSTRING是操作人花名册编码001
root.budgetOrgTypeIdSTRING是预算组织编号仅支持50位以内数字和字母test0101
root.budgetOrgTypeNameSTRING是预算组织分类名称api分类
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN否true说明接口调用成功 false说明接口调用失败true
root.messageSTRING否接口调用返回信息调用成功！

接口调用示例

请求报文


{
  "opsEmpNumber": "001",
  "budgetOrgTypeId": "test0101",
  "budgetOrgTypeName": "api分类"
}

调用成功的响应报文


{
    "success": true,
    "message": "调用成功！"
}

调用失败的响应报文，示例仅为其中一部分失败样例


{
    "code": "E0002",
    "message": "预算组织分类编码只允许数字和字母;"
}

错误码说明

sucess为导入成功状态， true为成功 其余都是失败

常见报错
预算组织分类编码只允许数字和字母;
