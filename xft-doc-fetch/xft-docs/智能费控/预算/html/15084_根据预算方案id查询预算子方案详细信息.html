根据预算方案id查询预算子方案详细信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/budget-sub-plan/list-by-parent

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/budget-sub-plan/list-by-parent

接口描述

本接口适用于根据预算方案id查询方案下所有子方案信息的列表，包含子方案的所有详细信息。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.budgetParentPlanSeqIdLONG是预算方案id1286
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataARRAY否接口返回添加结果
root.data.budgetPlanSeqIdLONG否预算子方案id1579
root.data.budgetParentPlanSeqIdLONG否预算方案id1286
root.data.budgetNameSTRING否预算方案名称异常1118
root.data.budgetDimensionSTRING否1按照预算部门,2按照预算科目,3按照预算部门/预算科目1
root.data.budgetEventControlSTRING否0部分,1全部0
root.data.budgetStateSTRING否0未启用,1启用1
root.data.budgetCycleSTRING否1月度预算,2季度预算,3年度预算1
root.data.budgetControlTypeSTRING否1不得提交,2不控制0
root.data.budgetWarningTypeSTRING否1预警提示,2不预警2
root.data.budgetWarningRateSTRING否预算预警百分比null
root.data.budgetTurnTypeSTRING否0不顺延,1每个编著周期末自动顺延1
root.data.cycleControlSTRING否1月度 2季度 3年度1
root.data.crtIdSTRING否创建人ID42603
root.successBOOLEAN否true说明接口调用成功 false说明接口调用失败true
root.messageSTRING否接口调用返回信息调用成功！
