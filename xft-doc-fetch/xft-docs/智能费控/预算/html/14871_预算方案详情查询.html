预算方案详情查询

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/get-budget-parent-plan-detail

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/get-budget-parent-plan-detail

接口描述

本接口适用于查询智能费控预算方案详情，包含预算方案基础信息和相关配置项。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
参数类型必填说明示例
rootOBJECTquery根节点
root.planSeqIdLONG是预算方案自增id100

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.bodyOBJECT否预算方案详情
root.body.budgetParentPlanSeqIdLONG否预算方案自增id100
root.body.budgetNameSTRING否预算方案名称测试预算方案
root.body.budgetStartDateSTRING否格式为yyyy-MM-dd2024-01-01
root.body.budgetEndDateSTRING否格式为yyyy-MM-dd2026-12-31
root.body.budgetDimensionSTRING否1-按预算组织维度
2-按预算科目维度
3-按预算组织+预算科目维度		3
root.body.budgetStateSTRING否0-停用
1-启用		1
root.body.eventIdsARRAY否表单自增id列表，最多支持关联100个表单
如果不填，则默认为适用所有的表单		[1,2,3]
root.body.budgetPlanIdSTRING否预算方案编码test001
root.body.budgetOrgsARRAY否预算组织自增id列表，最多支持关联100个预算组织
如果不填，则默认为支持所有的预算组织		[1,2,3]
root.body.budgetSubsARRAY否预算科目自增id列表，最多支持关联100个预算科目
如果不填，则默认为支持所有的预算科目		[1,2,3]
root.body.budgetApproveTimeSTRING否NONE-不需要审批

WHEN_INIT-期初预算编制需审批

WHEN_MODIFY-预算追加调整需审批

ANYTIME-期初预算编制/预算追加调整均需审批		NONE
root.body.needPostAccountSTRING否0-无需过账
1-需要过账		0
root.body.allowExternalSystemBillSTRING否0-不允许
1-允许		0
root.body.allowMonthOrderBudgetOccupySTRING否0-不支持
1-支持		0
root.body.budgetRangeFixedSTRING否0=不固定
1-固定		0
root.returnCodeSTRING否SUC0000-成功
E0001-系统错误
E0002-参数错误		SUC0000
root.successBOOLEAN否true说明接口调用成功
false说明接口调用失败		true
root.messageSTRING否接口调用返回信息成功
