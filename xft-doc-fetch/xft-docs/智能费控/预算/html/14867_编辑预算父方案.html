编辑预算父方案

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/edit-budget-parent-plan

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/edit-budget-parent-plan

接口描述

本接口适用于单条修改预算方案信息

调用本接口后可在“智能费控-系统设置-预算设置-预算方案“中查看修改后的预算方案信息。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.allowExternalSystemBillSTRING是0-不允许
1-允许		1
root.allowMonthOrderBudgetOccupySTRING是0-不支持
1-支持		1
root.budgetApproveTimeSTRING是NONE-不需要审批

WHEN_INIT-期初预算编制需审批

WHEN_MODIFY-预算追加调整需审批

ANYTIME-期初预算编制/预算追加调整均需审批		NONE
root.budgetEndDateSTRING是格式为yyyy-MM-dd2025-12-31
root.budgetNameSTRING是预算方案名称测试预算方案导入
root.budgetOrgsARRAY否预算组织自增id列表，最多支持关联100个预算组织
如果不填，则默认为支持所有的预算组织		[1,2,3]
root.budgetStartDateSTRING是格式为yyyy-MM-dd2025-01-01
root.budgetSubsARRAY否预算科目自增id列表，最多支持关联100个预算科目
如果不填，则默认为支持所有的预算科目		[1,2,3]
root.eventIdsARRAY否表单自增id列表，最多支持关联100个表单
如果不填，则默认为适用所有的表单		[1,2,3]
root.needPostAccountSTRING是0-无需过账
1-需要过账
若设置过账后生效，预算单审批通过后需点击过账后预算才能生效
预算是否需要审批为NONE时，默认设置无需过账-0		0
root.opsEmpNumberSTRING是操作人花名册编码001
root.budgetParentPlanSeqIdLONG是根据方案自增id匹配预算方案信息进行更新100
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.returnCodeSTRING否SUC0000-成功
E0001-系统错误
E0002-参数错误		SUC0000
root.successBOOLEAN否true说明接口调用成功
false说明接口调用失败		true
root.messageSTRING否接口调用返回信息成功

接口调用示例

请求报文


{
    "budgetParentPlanSeqId":100,
    "allowExternalSystemBill": "0",
    "allowMonthOrderBudgetOccupy": "0",
    "budgetApproveTime": "NONE",
    "budgetEndDate": "2026-12-31",
    "budgetName": "测试预算方案导入1",
    "budgetOrgs": [1],
    "budgetStartDate": "2024-01-01",
    "budgetSubs": [1],
    "eventIds": [1],
    "needPostAccount": "0",
    "opsEmpNumber": "001"
}

更新成功的响应报文


{
    "body": "",
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}

更新失败的响应报文，示例仅为其中一部分失败样例


{
    "returnCode": "E0001",
    "success": false,
    "message": "业务错误: 预算方案不存在，请检查后重试！"
}

接口调用失败的报文，示例仅为其中一部分失败样例


{
    "code": "E0002",
    "message": "预算方案自增ID不能为空;"
}

错误码说明

sucess为更新成功状态， true为成功 其余都是失败

常见报错
正在进行预算额度导入任务，请稍后再试
预算方案不存在，请检查后重试！
预算方案配置了固定区间，不允许修改预算期间，请检查
该方案下还有待审批的预算，暂不能修改审批设置
当前预算方案日期设置错误，请检查
预算组织在同一时间不能配置于多个方案
预算科目在同一时间不能配置于多个方案
预算组织科目在同一时间不能配置于多个方案
预算区间只支持扩大，不允许缩短
已经有预算发生额，预算区间只能向后扩展
