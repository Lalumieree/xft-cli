导入预算组织

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/import/organization

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/import/organization

接口描述

支持外部系统通过接口导入预算组织信息进入费控，支持新增和更新（以组织id为唯一值进行更新），支持导入或条件，不支持部分成功

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.organizationsARRAY是导入的预算组织列表，最多支持100条
root.organizations.adminUsersSTRING否填写员工花名册编码，以英文逗号,分割001,002
root.organizations.budgetIdSTRING否不填会默认生成
仅支持数字和字母
如果填写的组织id与系统内的组织id一致，将按传入信息对组织进行更新		orgId
root.organizations.budgetNameSTRING是预算组织名称api导入测试6
root.organizations.budgetOrgTypeIdSTRING是传入系统已有的预算组织分类idBGTTYP20241129084711
root.organizations.budgetOrgTypeNameSTRING是传入系统已有的预算组织分类名称部门分类
root.organizations.budgetParentIdSTRING否如果传了值，则导入的预算组织为该组织下级组织
如果没传值，则导入的预算组织为第一级		424124
root.organizations.priorityLONG是支持1-99的整数99
root.organizations.reportUsersSTRING否填写员工花名册编码，以英文逗号,分割003,004
root.organizations.conditionsARRAY是1、承担条件不可全为空，即承担部门、项目、项目分类等承担项，必须有一个有值
2、list存在多个条件时，条件之间为或关系
root.organizations.conditions.projectTypesSTRING否填写项目分类编码，以英文逗号,分割PRJTYPe20210,PRJTYPE2024
root.organizations.conditions.projectsSTRING否填写项目编码，以英文逗号,分割PRJ1,PRJ2
root.organizations.conditions.costCentersSTRING否填写成本中心编码，以英文逗号,分割CENTER1,CENTER2
root.organizations.conditions.depWithChildrenSTRING否填写部门编码，以英文逗号,分割00000001234,00000002
root.organizations.conditions.departmentsSTRING否填写部门编码，以英文逗号,分割00000001234,00000002
root.organizations.conditions.matterTypeNameSTRING否填写业务分类名称，以英文逗号,分割天蓝1,天蓝2
root.organizations.conditions.matterMissionNameSTRING否填写业务分类名称，以英文逗号,分割天蓝11,天蓝22
root.organizations.conditions.matterDetailNameSTRING否填写业务分类名称，以英文逗号,分割天蓝111,天蓝222
root.organizations.conditions.personsSTRING否填写员工花名册编码，以英文逗号,分割003,004
root.opsEmpNumberSTRING是填写员工花名册编码001
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataOBJECT否接口返回导入结果
root.data.successBOOLEAN否true说明导入成功
false说明导入失败		false
root.data.errorsARRAY否如果导入失败，这里会返回具体的导入失败原因
如果是导入成功，则为这里为空
root.data.errors.indexLONG否导入失败的组织序号1
root.data.errors.budgetOrgErrorSTRING否具体的导入失败原因校验第1行预算组织失败：预算组织所属上级编码不存在；预算组织承担项目不存在
root.successBOOLEAN否true说明接口调用成功
false说明接口调用失败		true
root.messageSTRING否接口调用返回信息调用成功！

错误码说明

外层success为接口调用状态 true为成功 其余都是失败

data内的sucess为实际导入成功状态， true为成功 其余都是失败

常见报错如下：

未设置预算组织名称，不予上传

预算组织编号仅支持数字和字母，且长度不可超过50位

预算组织名称长度不可超过120位

预算组织名称不可为'其他'

优先级必须为1-99的整数

预算组织编码重复

预算组织所属分类的编码和名称不一致

预算组织所属上级编码不存在

预算组织所属分类必须与所属上级的分类保持一致

xx（表示承担元素）存在重复值

xx（表示承担元素）不存在
