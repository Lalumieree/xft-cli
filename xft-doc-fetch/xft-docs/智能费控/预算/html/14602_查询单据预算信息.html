查询单据预算信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/budget/query-bill-quota

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/budget/query-bill-quota

接口描述

支持外部系统通过单据编号获取费控单据的预算使用情况，包括预算方案、预算编制和预算额度等

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
参数类型必填说明示例
rootOBJECTquery根节点
root.billIdLONG是单据编号2024103104293230

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.dataARRAY否单据预算信息
root.data.budgetNameSTRING否预算方案名称天1028
root.data.budgetPlanStartDateSTRING否yyyy-MM-dd2024-09-01
root.data.budgetPlanEndDateSTRING否yyyy-MM-dd2024-11-30
root.data.budgetCycleSTRING否1-月度预算
2-季度预算
3-年度预算		1
root.data.budgetDimensionSTRING否1-按照预算组织
2-按照预算科目
3-按照预算组织/预算科目		3
root.data.budgetWarningTypeSTRING否1-预警提示
2-不预警		1
root.data.budgetWarningRateBIGDECIMAL否小数点后2位30.00
root.data.periodLONG否单据所在预算期数2
root.data.quotaListARRAY否预算额度列表
root.data.quotaList.budgetOrgIdSTRING否预算组织IDBUDGETORG20241028194945
root.data.quotaList.budgetOrgNamSTRING否预算组织名称天1028
root.data.quotaList.subjectNameSTRING否预算科目名称简单科目1028
root.data.quotaList.budgetExpensesARRAY否当前预算编制下占用的费用信息
root.data.quotaList.budgetExpenses.overBudgetFlagSTRING否0-未超额
1-超额		0
root.data.quotaList.budgetExpenses.overReasonSTRING否预算超支原因额度超额
root.data.quotaList.budgetExpenses.expNamSTRING否费用/补贴名称简单费用
root.data.quotaList.budgetExpenses.dstntOrderINTEGER否所属行程目的地下标0
root.data.quotaList.budgetExpenses.amountBIGDECIMAL否小数点后2位100.00
root.data.quotaList.budgetExpenses.quotaTypeSTRING否0-编制额
1-顺延额		1
root.data.quotaList.currentQuotaOBJECT否预算编制额使用情况
root.data.quotaList.currentQuota.initialQuotaBIGDECIMAL否小数点后2位1000.00
root.data.quotaList.currentQuota.usedQuotaBIGDECIMAL否小数点后2位0.00
root.data.quotaList.currentQuota.frozenQuotaBIGDECIMAL否小数点后2位0.00
root.data.quotaList.currentQuota.remainQuotaBIGDECIMAL否小数点后2位1000.00
root.data.quotaList.currentQuota.reservedQuotaBIGDECIMAL否小数点后2位0.00
root.data.quotaList.remainingQuotaOBJECT否预算顺延额使用情况
root.data.quotaList.remainingQuota.initialQuotaBIGDECIMAL否小数点后2位1000.00
root.data.quotaList.remainingQuota.usedQuotaBIGDECIMAL否小数点后2位0.00
root.data.quotaList.remainingQuota.frozenQuotaBIGDECIMAL否小数点后2位110.00
root.data.quotaList.remainingQuota.remainQuotaBIGDECIMAL否小数点后2位790.30
root.data.quotaList.remainingQuota.reservedQuotaBIGDECIMAL否小数点后2位99.70
root.successBOOLEAN否是否请求成功true
root.messageSTRING否提示信息调用成功！
