# 新建多明细预付款单v2（草稿，审批通过）

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/bills/corporate/supplier/lend/create 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/bills/corporate/supplier/lend/create 
 ## 接口描述 
支持系统调用接口新增草稿、审批通过两种状态的预付款单
