# 新建预付款单（草稿，审批通过）

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/supplier-lend/create 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/supplier-lend/create 
 ## 接口描述 
支持系统调用接口新增审批通过的预付款单

## 错误码说明 
| 错误码| 错误信息|
  | --- | --- |
  | E0001| 单据公司和单据部门需要同时指定或者同时为空 |
| E0001| 请传入企业用户ID|
| E0001| 请传入花名册员工号|
| E0001| 请传入正确的员工匹配规则|
| E0001| 暂不支持除草稿、待审批、审批通过之外的单据状态|
| E0001| 表单不存在|

| E0001| 请填写正确的已核销金额|
