# 批量查询预付款单详情

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/bills/corporate/supplier/lend/detail 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/bills/corporate/supplier/lend/detail 
 ## 接口描述

## 调用方式 
 METHOD: POST
