# 新建对公报账单（草稿，审批中，审批通过）

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/supplier-reimburse/create 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/supplier-reimburse/create 
 ## 接口描述 
支持系统调用接口新增草稿/审批通过的对公报账单
