# 批量查询付款申请单详情

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v2/bills/corporate/pay/apply/detail 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v2/bills/corporate/pay/apply/detail 
 ## 接口描述 
批量获取付款申请单的单据详情 
## 调用方式

 CONTENT-TYPE: application/json
