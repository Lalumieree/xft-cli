费控推送申请单回调

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/bill/push-callback

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/bill/push-callback

接口描述

消费平台推送申请单回调接口

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT是请求体根节点
body.billIdSTRING是单据编号
body.platformSTRING是（ZAI_TU）在途
body.billTypeSTRING是(TRAVEL_APPLY)差旅申请,（INVITATION_APPLY)招待申请
body.requestDataSTRING是申请单推送请求报文
body.responseDataSTRING是申请单推送响应报文
body.codeSTRING是（SUCCESS）成功，（FAIL）失败
body.errorMessageSTRING否错误信息
body.completeTimeSTRING否格式：yyyy-MM-dd HH:mm:ss
参数类型必填说明示例
queryOBJECTquery根节点

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
