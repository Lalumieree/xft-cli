对账结果查询

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/reconciliation/detail/page

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/reconciliation/detail/page

接口描述

分页查询企业对账结果

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.startDateSTRING否yyyy-MM-dd2023-10-01
root.endDateSTRING否yyyy-MM-dd2024-10-01
root.channelListARRAY否第三方消费平台枚举
例CTRIP,DTTRIP,FEIHEAIR		["CTRIP"]
root.statusListARRAY否UN_CHECK：未对账
CHECK_SUCCESS：对账成功
CHECK_FAIL：对账失败		["CHECK_SUCCESS"]
root.startINTEGER是起始页为00
root.limitINTEGER是最大条数为2020
参数类型必填说明示例
rootOBJECTquery根节点
