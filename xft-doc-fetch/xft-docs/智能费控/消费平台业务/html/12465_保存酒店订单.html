保存酒店订单

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/hotel

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/hotel

接口描述

保存酒店订单

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyARRAY请求体根节点
body.orderBaseInfoOBJECT是订单基本信息
body.orderBaseInfo.platformSTRING是订单平台ZAI_TU
body.orderBaseInfo.orderTypeSTRING是N预订 T退票 C改签N
body.orderBaseInfo.thirdOrderIdSTRING是订单编号
body.orderBaseInfo.thirdParentOrderIdSTRING否第三方上级订单编号
body.orderBaseInfo.thirdOriginOrderIdSTRING否第三方原始订单号
body.orderBaseInfo.sourceSTRING否订单来源
OFFICIAL_WEBSITE: 官网
SERVICE_AGENCY_BOOK: 客服代订
SERVICE_MANUAL_RECORD: 客服手工录单		OFFICIAL_WEBSITE
body.orderBaseInfo.relatedBusinessOrderIdLONG否关联业务订单号（酒店系统的业务单号）
body.orderBaseInfo.bookEmpNumberSTRING是系统编号与花名册编号必填一个
body.orderBaseInfo.bookEmpSeqIdLONG是系统编号与花名册编号必填一个
body.orderBaseInfo.bookEmpNameSTRING是预订人名称
body.orderBaseInfo.bookEmpPhoneNumberSTRING否预订人手机号
body.orderBaseInfo.orderCreateTimeDATE是格式yyyy-MM-dd HH:mm:ss
body.orderBaseInfo.orderPayTimeDATE是格式yyyy-MM-dd HH:mm:ss
body.orderBaseInfo.orderCompleteTimeDATE否格式 yyyy-MM-dd HH:mm:ss
body.orderBaseInfo.checkInTimeDATE是格式 yyyy-MM-dd HH:mm:ss
body.orderBaseInfo.checkOutTimeDATE是格式 yyyy-MM-dd HH:mm:ss
body.orderBaseInfo.travelTypeSTRING是COMPANY_TRIP：因公 ，PERSONAL_TRIP：因私COMPANY_TRIP
body.orderBaseInfo.relatedApplyBillIdLONG否申请单号
body.orderPayInfoOBJECT是订单支付信息
body.orderPayInfo.payWaySTRING是COMPANY_PAY： 公司支付 ，PERSONAL_PAY： 个人支付COMPANY_PAY
body.orderPayInfo.amountDOUBLE是精确到两位小数
body.orderPayInfo.excludeServiceFeeDOUBLE是精确到两位小数
body.orderPayInfo.serviceFeeDOUBLE是精确到两位小数
body.orderPayInfo.refundFeeDOUBLE是精确到两位小数
body.orderPayInfo.companyPayAmountDOUBLE是精确到两位小数
body.orderPayInfo.personalPayAmountDOUBLE是精确到两位小数
body.orderPayInfo.exceedStandardSTRING否Y 是 ，N 否Y
body.orderPayInfo.exceedStandardReasonSTRING否超标事由
body.orderPayInfo.refundReasonSTRING否退款事由
body.hotelInfoOBJECT是酒店信息
body.hotelInfo.hotelNameSTRING是酒店名称
body.hotelInfo.hotelAddressSTRING是酒店地址
body.hotelInfo.hotelCityCodeSTRING是城市编码
body.hotelInfo.hotelCityNameSTRING是城市名称
body.hotelInfo.hotelPhoneNumberSTRING否酒店电话
body.hotelInfo.reserveRoomNumINTEGER是预订房间数量
body.hotelInfo.hotelLevelNameSTRING否酒店星级名称
body.hotelInfo.agreementHotelSTRING是Y 是， N 否Y
body.hotelInfo.roomNightNumINTEGER否房晚数
body.hotelInfo.roomTypeSTRING否无类型限制
body.hotelInfo.hasBreakfastSTRING否Y 是， N 否Y
body.hotelInfo.breakfastPriceDOUBLE否精确到两位小数
body.hotelInfo.breakfastNumINTEGER否早餐份数
body.checkInPersonInfoOBJECT是入住人信息
body.checkInPersonInfo.totalTravelStandardDOUBLE否入住人每日总差标
body.checkInPersonInfo.checkInPersonInfosARRAY是入住人列表
body.checkInPersonInfo.checkInPersonInfos.checkInEmpNumberSTRING是入住人编号（花名册编号）
body.checkInPersonInfo.checkInPersonInfos.checkInEmpSeqIdLONG是入住人系统编号（系统编号与花名册编号必填一个）
body.checkInPersonInfo.checkInPersonInfos.checkInEmpNameSTRING是入住人名称
body.checkInPersonInfo.checkInPersonInfos.dayTravelStandardDOUBLE否入住人每日差标
参数类型必填说明示例
queryOBJECTquery根节点

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.messageSTRING接口结果信息
body.bodySTRING响应体
body.returnCodeSTRING返回码
body.errorMsgSTRING错误信息
body.successBOOLEAN是否调用成功
