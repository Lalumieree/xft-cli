月结订单预算释放

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/budget-release

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/budget-release

接口描述

月结订单预算释放接口，释放成功返回释放成功的第三方订单号列表

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootARRAY请求体根节点
root.platformSTRING是订单平台FEIHEAIR
root.thirdOrderIdSTRING是第三方订单号BUDGET-002
root.orderUniqueIdSTRING是订单唯一id2025212002
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN否释放成功标志true
root.messageSTRING否释放失败错误信息null
root.dataARRAY否预算释放结果列表
root.data.orderUniqueIdLONG否释放成功的订单唯一id2025212002

接口调用示例

请求报文


[
    {
        "platform":"FEIHEAIR",
        "thirdOrderId":"BUDGET-002",
        "orderUniqueId":"2025212002"
    }
]

响应报文


{
    "success": true,
    "message": null,
    "data": [
        {
            "orderUniqueId": "2025212002"
        }
    ]
}

错误码说明

错误码错误信息
200成功
