保存餐食订单

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/meal

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/meal

接口描述

消费平台存储连接器导入订单

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyARRAY是请求体根节点
body.orderBaseInfoOBJECT是订单基本信息
body.orderBaseInfo.bookEmpNumberSTRING是预订人编号（花名册编号）
body.orderBaseInfo.bookEmpSeqIdLONG是预订人系统编号（系统编号与花名册编号必填一个）
body.orderBaseInfo.bookEmpNameSTRING是预订人名称
body.orderBaseInfo.platformSTRING是在途 - ZAI_TU， 美团 - MEITUAN_SQT
body.orderBaseInfo.relatedApplyBillIdSTRING否申请单编号
body.orderBaseInfo.orderTypeSTRING是N:正常 T:退订
body.orderBaseInfo.thirdOrderIdSTRING是订单编号
body.orderBaseInfo.thirdOriginOrderIdSTRING否第三方原始订单编号
body.orderBaseInfo.orderCreateTimeSTRING是yyyy-MM-dd HH:mm:ss
body.orderBaseInfo.orderCompleteTimeSTRING否yyyy-MM-dd HH:mm:ss
body.orderPayInfoOBJECT是订单支付信息
body.orderPayInfo.amountSTRING是精确到两位小数
body.orderPayInfo.excludeServiceFeeSTRING是精确到两位小数
body.orderPayInfo.serviceFeeLONG是精确到两位小数
body.orderPayInfo.refundFeeSTRING是精确到两位小数
body.orderPayInfo.payWaySTRING是公司支付 COMPANY_PAY，个人支付 PERSONAL_PAY
body.orderPayInfo.companyPayAmountSTRING是精确到两位小数
body.orderPayInfo.personalPayAmountSTRING是精确到两位小数
body.orderPayInfo.orderPayTimeSTRING是格式：yyyy-MM-dd HH:mm:ss
body.orderPayInfo.refundReasonSTRING否退款事由
body.mealInfoOBJECT是餐食信息
body.mealInfo.shopNameSTRING是餐厅名称
body.mealInfo.diningSceneSTRING否用餐场景
body.mealInfo.mealTypeSTRING是BUSINESS_DINNER：商务宴请，
BUSINESS_TRAVEL：商务差旅，
WORKING：工作餐，
TEAM_BUILDING：团建用餐，
OTHER：其他
body.mealInfo.peopleNumLONG是用餐人数
body.mealInfo.mealCityCodeSTRING是用餐城市编码
body.mealInfo.deliverTimeSTRING否格式：yyyy-MM-dd HH:mm:ss
body.participantsInfoARRAY是用餐人信息
body.participantsInfo.participantEmpNumberSTRING否用餐人编号（花名册编号）
body.participantsInfo.participantEmpSeqIdLONG用餐人系统编号
body.participantsInfo.participantNameSTRING是用餐人名称
body.participantsInfo.dayTravelStandardSTRING否用餐人每日餐标
参数类型必填说明示例
queryOBJECTquery根节点

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.returnCodeSTRING否返回码成功 - SUC0000
body.successBOOLEAN否successtrue
body.messageSTRING否message“成功”
