消费平台开通结果回调

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/account/bind-callback

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/account/bind-callback

接口描述

企业绑定消费平台账号信息后，费控发送消费平台开通申请事件，供应商处理完该事件后需回调本接口用于告知是否开通成功。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.platformSTRING是消费平台平台PLATFORM_CODE
root.bindSuccessBOOLEAN是是否开通成功true
root.failReasonSTRING成功该字段为空
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN调用结果true
root.messageSTRING调用返回信息"成功"
root.returnCodeSTRING返回编码SUC0000
root.errorMsgSTRING异常提示信息，成功为空

接口调用示例

请求报文


{
    "platform":"PLATFORM_CODE",
    "bindSuccess":true
}

响应报文


{
  "success":true,
  "message":"成功",
  "returnCode":"SUC0000"
}

错误码说明

错误码错误信息
200成功
