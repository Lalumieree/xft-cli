保存飞机订单

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/flight

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/flight

接口描述

保存飞机订单

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyARRAY是请求体根节点
body.contactInfoOBJECT否联系人信息
body.contactInfo.contactMailSTRING否联系人邮箱
body.contactInfo.contactNameSTRING否联系人姓名
body.contactInfo.contactPhoneSTRING否联系人手机号
body.flightInfoOBJECT是航班信息
body.flightInfo.airlineCompanyCodeSTRING否航司二字码
body.flightInfo.airlineCompanyNameSTRING否航司名称
body.flightInfo.arrivalTimeDATE是到达时间
body.flightInfo.departAirportCodeSTRING否出发机场三字码
body.flightInfo.departAirportNameSTRING否出发机场名称
body.flightInfo.departCityCodeSTRING是出发地城市编码
body.flightInfo.departTerminalSTRING否出发航站楼名称
body.flightInfo.departureTimeDATE是起飞时间
body.flightInfo.destinationAirportCodeSTRING否到达机场三字码
body.flightInfo.destinationAirportNameSTRING否到达机场名称
body.flightInfo.destinationCityCodeSTRING是目的地城市编码
body.flightInfo.destinationTerminalSTRING否到达航站楼名称
body.flightInfo.flightNumberSTRING否航班号
body.flightInfo.hasFoodSTRING否是否有餐食 Y 是 N否
body.flightInfo.stopNumberINTEGER否经停次数
body.flightTicketInfoOBJECT否机票信息
body.flightTicketInfo.airRangeTypeSTRING否单程：ONE_WAY_TRIPS 来回程：ROUND_TRIPS 环程：CIRCLE_TRIPS
环球程：ROUND_THE_WORLD 缺口程：OPEN_JAW_TRIP
body.flightTicketInfo.cabinLevelSTRING否经济舱：Y，公务舱：C，头等舱：F
body.flightTicketInfo.fuelFeeDOUBLE否燃油附加费
body.flightTicketInfo.insuranceFeeDOUBLE否保险费用
body.flightTicketInfo.itineraryNumberSTRING否行程单号
body.flightTicketInfo.machineryConstructionFeeDOUBLE否机场建设费
body.flightTicketInfo.ticketIssueTimeDATE否出票时间 格式yyyy-MM-dd HH:mm:ss
body.flightTicketInfo.ticketNumberSTRING是票号
body.flightTicketInfo.ticketPriceDOUBLE否票价
body.orderBaseInfoOBJECT是订单基本信息
body.orderBaseInfo.areaTypeSTRING否订单地区类型
INTERNATIONAL: 国际
DOMESTIC: 国内
body.orderBaseInfo.bookEmpNumberSTRING是系统编号与花名册编号必填一个
body.orderBaseInfo.bookEmpSeqIdLONG否系统编号与花名册编号必填一个
body.orderBaseInfo.bookEmpNameSTRING是预订人姓名
body.orderBaseInfo.orderCompleteTimeDATE否订单系统完成时间
格式yyyy-MM-dd HH:mm:ss
body.orderBaseInfo.orderCreateTimeDATE是订单系统创建时间
格式yyyy-MM-dd HH:mm:ss
body.orderBaseInfo.orderTypeSTRING是订单类型
N预订 T退票 C改签
body.orderBaseInfo.passengerEmpNumberSTRING是系统编号与花名册编号必填一个
body.orderBaseInfo.passengerEmpSeqIdLONG否系统编号与花名册编号必填一个
body.orderBaseInfo.passengerNameSTRING是乘机人姓名
body.orderBaseInfo.platformSTRING是ZAI_TU 在途 CTRIP 携程
body.orderBaseInfo.relatedApplyBillIdLONG否申请单号
body.orderBaseInfo.relatedBusinessOrderIdSTRING否关联业务订单号
body.orderBaseInfo.sourceSTRING否订单来源
OFFICIAL_WEBSITE: 官网
SERVICE_AGENCY_BOOK: 客服代订
SERVICE_MANUAL_RECORD: 客服手工录单
body.orderBaseInfo.thirdOrderIdSTRING是第三方订单编号
body.orderBaseInfo.thirdOriginOrderIdSTRING否第三方原始订单编号
退款、改签产生的订单号写入原始预订单的订单编号（退、改单该字段不可为空）
body.orderBaseInfo.thirdParentOrderIdSTRING否第三方上级订单编号
退款、改签产生的订单号写入上一级改签/预订单的订单编号（退、改单该字段不可为空）
body.orderBaseInfo.travelTypeSTRING是差旅类型
"COMPANY_TRIP 因公
PERSONAL_TRIP 因私"
body.orderPayInfoOBJECT是订单支付信息
body.orderPayInfo.amountDOUBLE是交易总额
body.orderPayInfo.baggageAllowanceDOUBLE否行李额度（kg）
body.orderPayInfo.cabinDiscountDOUBLE否舱位折扣
body.orderPayInfo.cabinNameSTRING否舱位名称
body.orderPayInfo.changeFeeDOUBLE是改签手续费
body.orderPayInfo.changeReasonSTRING否改签事由
body.orderPayInfo.companyPayAmountDOUBLE是公司支付金额
body.orderPayInfo.exceedStandardSTRING否是否超标 Y 是 N否
body.orderPayInfo.exceedStandardReasonSTRING否超标事由
body.orderPayInfo.excludeServiceFeeDOUBLE是不含服务费金额
body.orderPayInfo.orderPayTimeDATE是订单支付时间 格式yyyy-MM-dd HH:mm:ss
body.orderPayInfo.payWaySTRING是COMPANY_PAY 公司支付
PERSONAL_PAY 个人支付
body.orderPayInfo.personalPayAmountDOUBLE是个人支付金额
body.orderPayInfo.refundFeeDOUBLE是退票手续费
body.orderPayInfo.refundReasonSTRING否退款事由
body.orderPayInfo.serviceFeeDOUBLE是平台服务费
参数类型必填说明示例
queryOBJECTquery根节点

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.messageSTRING接口结果信息
body.bodySTRING响应体
body.returnCodeSTRING响应码
body.errorMsgSTRING错误信息
body.successBOOLEAN接口是否调用成功
