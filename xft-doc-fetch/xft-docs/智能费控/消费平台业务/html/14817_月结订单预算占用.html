月结订单预算占用

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/budget-occupy

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/budget-occupy

接口描述

月结订单占用预算接口，通过传入订单关键信息进行预算占用，返回占用结果。

往返、多人订票的场景需要进行拆单处理（即与订单保存api拆单逻辑保持一致），每笔数据的orderUniqueId不可重复，且退改签订单数据需要传对应的原单的orderUniqueId（通过parentOrderUniqueId字段传输）

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootARRAY请求体根节点
root.bookEmpNumberSTRING否预订单员工号number001
root.bookEmpNameSTRING是预定人姓名张三
root.orderUseEmpDTOListARRAY否使用人列表
root.orderUseEmpDTOList.useEmpNumberSTRING否使用人员工编号number001
root.orderUseEmpDTOList.useEmpNameSTRING否使用人姓名张三
root.platformSTRING是三方平台枚举FENGYI_SHANGLV
root.productTypeSTRING是FLIGHT：机票；
HOTEL：酒店；
TRAIN：火车；
MEAL：餐食；
CAR：打车		FLIGHT
root.orderTypeSTRING是N：预订单
T：退票单
C：改签单		C
root.orderUniqueIdSTRING是需确保唯一性2025212001
root.parentOrderUniqueIdSTRING否原订单唯一id2025211001
root.thirdOrderIdSTRING是第三方订单号BUDGET-002
root.thirdParentOrderIdSTRING否第三方上级订单号BUDGET-001
root.thirdOriginOrderIdSTRING否第三方原始订单号BUDGET-001
root.relatedApplyBillIdSTRING否关联申请单号2024073003879491
root.orderCreateTimeSTRING是yyyy-MM-dd HH:mm:ss2025-02-11 17:06:00
root.amountBIGDECIMAL是保留两位小数24.00
root.excludeServiceFeeBIGDECIMAL是保留两位小数20.00
root.serviceFeeBIGDECIMAL是保留两位小数4.00
root.refundFeeBIGDECIMAL是保留两位小数0.00
root.changeFeeBIGDECIMAL是保留两位小数0.00
root.sceneNameSTRING否场景名称出差
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.successBOOLEAN否占用成功标志true
root.messageSTRING否占用失败错误信息null
root.dataARRAY否预算占用结果列表
root.data.orderUniqueIdSTRING否订单唯一id2025212002
root.data.budgetOccupyFlagSTRING否0-未占用，1-占用成功，2-占用失败0
root.data.budgetOccupyFailReasonSTRING否占用失败错误信息改订单已成功占用预算，无法重复占用
