# 保存餐食订单

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/meal 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/meal 
 ## 接口描述 
消费平台存储连接器导入订单 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | ARRAY | 是 | 请求体根节点 |  |
| body.orderBaseInfo | OBJECT | 是 | 订单基本信息 |  |
| body.orderBaseInfo.bookEmpNumber | STRING | 是 | 预订人编号（花名册编号） |  |
| body.orderBaseInfo.bookEmpSeqId | LONG | 是 | 预订人系统编号（系统编号与花名册编号必填一个） |  |
| body.orderBaseInfo.bookEmpName | STRING | 是 | 预订人名称 |  |
| body.orderBaseInfo.platform | STRING | 是 | 在途 - ZAI_TU， 美团 - MEITUAN_SQT |  |
| body.orderBaseInfo.relatedApplyBillId | STRING | 否 | 申请单编号 |  |
| body.orderBaseInfo.orderType | STRING | 是 | N:正常 T:退订 |  |
| body.orderBaseInfo.thirdOrderId | STRING | 是 | 订单编号 |  |
| body.orderBaseInfo.thirdOriginOrderId | STRING | 否 | 第三方原始订单编号 |  |
| body.orderBaseInfo.orderCreateTime | STRING | 是 | yyyy-MM-dd HH:mm:ss |  |
| body.orderBaseInfo.orderCompleteTime | STRING | 否 | yyyy-MM-dd HH:mm:ss |  |
| body.orderPayInfo | OBJECT | 是 | 订单支付信息 |  |
| body.orderPayInfo.amount | STRING | 是 | 精确到两位小数 |  |
| body.orderPayInfo.excludeServiceFee | STRING | 是 | 精确到两位小数 |  |
| body.orderPayInfo.serviceFee | LONG | 是 | 精确到两位小数 |  |
| body.orderPayInfo.refundFee | STRING | 是 | 精确到两位小数 |  |
| body.orderPayInfo.payWay | STRING | 是 | 公司支付 COMPANY_PAY，个人支付 PERSONAL_PAY |  |
| body.orderPayInfo.companyPayAmount | STRING | 是 | 精确到两位小数 |  |
| body.orderPayInfo.personalPayAmount | STRING | 是 | 精确到两位小数 |  |
| body.orderPayInfo.orderPayTime | STRING | 是 | 格式：yyyy-MM-dd HH:mm:ss |  |
| body.orderPayInfo.refundReason | STRING | 否 | 退款事由 |  |
| body.mealInfo | OBJECT | 是 | 餐食信息 |  |
| body.mealInfo.shopName | STRING | 是 | 餐厅名称 |  |
| body.mealInfo.diningScene | STRING | 否 | 用餐场景 |  |
| body.mealInfo.mealType | STRING | 是 | BUSINESS_DINNER：商务宴请，<br>BUSINESS_TRAVEL：商务差旅，<br>WORKING：工作餐，<br>TEAM_BUILDING：团建用餐，<br>OTHER：其他 |  |
| body.mealInfo.peopleNum | LONG | 是 | 用餐人数 |  |
| body.mealInfo.mealCityCode | STRING | 是 | 用餐城市编码 |  |
| body.mealInfo.deliverTime | STRING | 否 | 格式：yyyy-MM-dd HH:mm:ss |  |
| body.participantsInfo | ARRAY | 是 | 用餐人信息 |  |
| body.participantsInfo.participantEmpNumber | STRING | 否 | 用餐人编号（花名册编号） |  |
| body.participantsInfo.participantEmpSeqId | LONG |  | 用餐人系统编号 |  |
| body.participantsInfo.participantName | STRING | 是 | 用餐人名称 |  |
| body.participantsInfo.dayTravelStandard | STRING | 否 | 用餐人每日餐标 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.returnCode | STRING | 否 | 返回码 | 成功 - SUC0000 |
