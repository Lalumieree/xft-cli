# 保存火车订单

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/train 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/train 
 ## 接口描述 
消费平台存储连接器导入订单 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
