# 对账结果查询

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/reconciliation/detail/page 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/reconciliation/detail/page 
 ## 接口描述 
分页查询企业对账结果 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
