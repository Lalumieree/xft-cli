# 保存飞机订单

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/flight 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/flight 
 ## 接口描述 
保存飞机订单 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
