# 保存打车订单

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/car 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/car 
 ## 接口描述 
保存打车订单 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | ARRAY | 是 | 请求体根节点 |  |
| body.orderBaseInfo | OBJECT | 是 | 订单基本信息 |  |
| body.orderBaseInfo.bookEmpNumber | STRING | 是 | 预订人编号（花名册编号） |  |

| body.orderBaseInfo.passengerEmpNumber | STRING | 是 | 乘车人编号（花名册编号） |  |

| body.orderPayInfo.payWay | STRING | 是 | COMPANY_PAY：公司支付，PERSONAL_PAY：个人支付 | COMPANY_PAY |
| body.orderPayInfo.companyPayAmount | DOUBLE | 是 | 精确到两位小数 |  |
| body.orderPayInfo.personalPayAmount | DOUBLE | 是 | 精确到两位小数 |  |
| body.orderPayInfo.exceedStandard | STRING | 否 | 是Y，否N | Y |
| body.orderPayInfo.exceedStandardReason | STRING | 否 | 超标事由 |  |
| body.useCarInfo | OBJECT | 是 | 用车信息 |  |
| body.useCarInfo.planDepartTime | STRING | 否 | 格式：yyyy-MM-dd HH:mm:ss |  |
| body.useCarInfo.departTime | DATE | 是 | 格式：yyyy-MM-dd HH:mm:ss |  |
| body.useCarInfo.destinationTime | DATE | 是 | 格式：yyyy-MM-dd HH:mm:ss |  |
| body.useCarInfo.departCityCode | LONG | 是 | 出发城市编码 |  |
| body.useCarInfo.destinationCityCode | LONG | 是 | 目的城市编码 |  |
| body.useCarInfo.departFullAddress | STRING | 否 | 出发地详细地址 |  |
| body.useCarInfo.destinationFullAddress | STRING | 否 | 到达地详细地址 |  |
| body.useCarInfo.useCarSceneName | STRING | 否 | 用车场景名称 |  |
| body.useCarInfo.carModelName | STRING | 否 | 车型名称 |  |
| body.useCarInfo.hasClock | STRING | 否 | 是Y，否N | Y |
| body.useCarInfo.passengerNumber | LONG | 否 | 乘客人数 |  |
| body.useCarInfo.useCarCostMinutes | DOUBLE | 否 | 单位：分钟 |  |
| body.useCarInfo.useCarDistance | DOUBLE | 否 | 单位：千米 |  |
| body.contactInfo | OBJECT | 否 | 联系人信息 |  |
| body.contactInfo.contactName | STRING | 否 | 联系人姓名 |  |
| body.contactInfo.contactPhone | LONG | 否 | 联系手机号 |  |
