# 消费平台订单分页查询

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/page 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/page 
 ## 接口描述 
分页查询订单详情 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.start | INTEGER | 是 | 页数从0开始 | 0 |
| body.limit | INTEGER | 是 | 最大值100 | 20 |
| body.queryDto | OBJECT | 是 | 订单查询条件 |  |
| body.queryDto.orderSeqIdList | ARRAY | 否 | 单次最多查询100条订单 | [123,124] |
| body.queryDto.thirdOrderIdList | ARRAY | 否 | 单次最多查询100条订单 | ["HO123456"，“HO345656”] |
| body.queryDto.platform | STRING | 否 | CTRIP 携程；<br>DTTRIP 同程商旅；<br>FEIHEAIR 飞鹤商旅；<br>MEITUAN_SQT 美团；<br>MBTRIP SCO云旅；<br>E_MEAL E餐通；<br>TRAVEL_ONE_NO 差旅壹号；<br>ZAI_TU 在途商旅；<br>DI_DI 滴滴企业版；<br>SHENG_YI 胜意商旅；<br>GAO_DE 高德打车企业版 | CTRIP |
| body.queryDto.startDate | STRING |  | 按照交易时间查询<br>格式：yyyy-MM-dd HH:mm:ss | 2024-05-01 10:10:10 |
| body.queryDto.endDate | STRING |  | 按照交易时间查询<br>格式：yyyy-MM-dd HH:mm:ss | 2024-05-10 10:10:10 |
| body.queryDto.tripApplyBillIdList | ARRAY |  | 单次最多查询50条事前申请单关联的订单 | [202507171212121212] |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
