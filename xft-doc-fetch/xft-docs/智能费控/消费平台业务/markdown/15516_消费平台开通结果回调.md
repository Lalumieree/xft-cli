# 消费平台开通结果回调

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/account/bind-callback 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/account/bind-callback 
 ## 接口描述 
企业绑定消费平台账号信息后，费控发送消费平台开通申请事件，供应商处理完该事件后需回调本接口用于告知是否开通成功。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.platform | STRING | 是 | 消费平台平台 | PLATFORM_CODE |
| root.bindSuccess | BOOLEAN | 是 | 是否开通成功 | true |
| root.failReason | STRING |  | 成功该字段为空 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.success | BOOLEAN |  | 调用结果 | true |
| root.message | STRING |  | 调用返回信息 | "成功" |
| root.returnCode | STRING |  | 返回编码 | SUC0000 |
| root.errorMsg | STRING |  | 异常提示信息，成功为空 |  |
 
## 接口调用示例 
 ### 请求报文
```json
{
    "platform":"PLATFORM_CODE",
    "bindSuccess":true
}
```
### 响应报文
```json
{
  "success":true,
  "message":"成功",
  "returnCode":"SUC0000"
}
```  
## 错误码说明 
| 错误码| 错误信息|
  | --- | --- | 
  | 200| 成功|
