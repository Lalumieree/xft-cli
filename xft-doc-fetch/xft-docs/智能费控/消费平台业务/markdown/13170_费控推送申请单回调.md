# 费控推送申请单回调

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/bill/push-callback 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/bill/push-callback 
 ## 接口描述 

消费平台推送申请单回调接口 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT | 是 | 请求体根节点 |  |
| body.billId | STRING | 是 | 单据编号 |  |
| body.platform | STRING | 是 | （ZAI_TU）在途 |  |
| body.billType | STRING | 是 | (TRAVEL_APPLY)差旅申请,（INVITATION_APPLY)招待申请 |  |
| body.requestData | STRING | 是 | 申请单推送请求报文 |  |
| body.responseData | STRING | 是 | 申请单推送响应报文 |  |
| body.code | STRING | 是 | （SUCCESS）成功，（FAIL）失败 |  |
| body.errorMessage | STRING | 否 | 错误信息 |  |
| body.completeTime | STRING | 否 | 格式：yyyy-MM-dd HH:mm:ss |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
