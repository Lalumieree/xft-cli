# 月结订单预算释放

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/travel/orders/budget-release 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/travel/orders/budget-release 
 ## 接口描述 
月结订单预算释放接口，释放成功返回释放成功的第三方订单号列表 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | ARRAY |  | 请求体根节点 |  |
| root.platform | STRING | 是 | 订单平台 | FEIHEAIR |
| root.thirdOrderId | STRING | 是 | 第三方订单号 | BUDGET-002 |
| root.orderUniqueId | STRING | 是 | 订单唯一id | 2025212002 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.success | BOOLEAN | 否 | 释放成功标志 | true |
| root.message | STRING | 否 | 释放失败错误信息 | null |
| root.data | ARRAY | 否 | 预算释放结果列表 |  |
| root.data.orderUniqueId | LONG | 否 | 释放成功的订单唯一id | 2025212002 |
 
## 接口调用示例 
 ### 请求报文
```json
[
    {
        "platform":"FEIHEAIR",
        "thirdOrderId":"BUDGET-002",
        "orderUniqueId":"2025212002"
    }
]
```
### 响应报文
```json
{
    "success": true,
    "message": null,
    "data": [
        {
            "orderUniqueId": "2025212002"
        }
    ]
}
```  
## 错误码说明 
| 错误码| 错误信息|
  | --- | --- | 
  | 200 | 成功 |
