查单据审批历史

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/taskHistory/{billId}

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/taskHistory/{billId}

接口描述

根据单据id查询单据审批历史记录

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING否薪福通企业号XFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.dataARRAY否数据
body.data.nodeIdSTRING否节点Id
body.data.nodeNameSTRING否节点名称发起申请
body.data.futureNodeSTRING否未来节点id
body.data.nodeTypeOBJECT否节点类型
body.data.nodeType.typeSTRING否类型START
body.data.nodeType.subTypeSTRING否节点子类型
body.data.nodeType.actionTypeSTRING否操作类型
body.data.nodeType.completeConditionSTRING否完成条件
body.data.startTimeDATE否开始时间
body.data.actionsARRAY否节点操作
body.data.actions.userIdSTRING否用户Id
body.data.actions.userNameSTRING否用户花名册员工号
body.data.actions.userNumberSTRING否用户名
body.data.actions.empThirdPartIdSTRING否部门
body.data.actions.actionSTRING否操作行为
body.data.actions.commentSTRING否备注
body.data.actions.finishTimeDATE否完成时间
body.data.actions.messageSTRING否信息
body.data.actions.taskIdSTRING否任务Id
body.data.actions.idSTRING否ID
body.data.actions.approveSignatureSTRING否审批签名
body.data.commentListARRAY否评论集合
body.successBOOLEAN否true-成功，false-失败true
body.messageSTRING否调用信息调用成功！

接口调用示例

###请求示例

{BaseURL}/api/v1/bills/taskHistory/{billId}?OPAUID=XFT00001
