签收发票删除

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/invoice/deleteInvoice

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/invoice/deleteInvoice

接口描述

将已签收的发票设置为未签收，并改变关联单据的签收状态。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyARRAY是请求体根节点
body.billTypeSTRING是取值参考[发票类型枚举](https://xft.cmbchina.com/open/#/doc/open-document?id=10692&mid=11585)G
body.invoiceNoSTRING否发票类型为1，2，3，4，8，9，A,B,C,D,G,H时有值12002361
body.invoiceCodeSTRING否发票类型为1，2，3，4，8，9，A,B,C,D,G,H时有值21010122
body.trafficNoSTRING发票类型为7,E,F时为票据号码 ，发票类型为5，6时为票号04829364
body.trafficCodeSTRING发票类型为7,E,F时有值044031900111
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是新福通企业号XFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.successBOOLEAN请求结果true
body.messageSTRING返回信息调用成功

接口调用示例

{BaseURL}/api/itrip-gateway/v1/invoice/deleteInvoice?OPAUID=XFT00001

**请求示例**


[
    {
        "billType": "G",
        "invoiceCode": "21010122",
        "invoiceNo": "12002361",
        "trafficCode": null,
        "trafficNo": null
    },
    {
        "billType": "E",
        "invoiceCode": null,
        "invoiceNo": null,
        "trafficCode": 044031900111,
        "trafficNo": 04829364
    }
]

**返回示例**

正常：


{
    "success": true,
    "message": "调用成功！"
}

异常：

如果没有传发票信息，返回样例：


{
    "success": false,
    "message": "发票信息为空或不完整"
}

如果传入了其他公司的发票信息，返回样例：


{
    "success": false,
    "message": "存在其他公司的发票"
}

如果发票完全不属于当前公司，返回样例：


{
    "success": false,
    "message": "发票不在当前公司下"
}

错误码说明

**错误码****错误描述****备注**
