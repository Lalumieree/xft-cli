批量查询单据列表

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/bill-list

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/bill-list

接口描述

根据查询条件分页查询单据基本信息列表

**表单业务类型枚举**

**业务类型编码****表单业务类型含义****对应表单属性**
TRAVEL_APPLY差旅申请申请单
EXPENSE_APPLY费用申请申请单
INVITATION_APPLY招待申请申请单
USE_CAR_APPLY用车申请申请单
COMMON_APPLY其他通用申请申请单
TRAVEL_REIM差旅报销报销单
COMMON_REIM日常报销报销单
EMP_LEND员工借款借还款单
REPAYMENT还款单借还款单
SUPPLIER_LEND预付款对公单据
SUPPLIER_REIM对公报账对公单据
SUPPLIER_SETTLE对公结算对公单据
PAY_APPLY付款申请对公单据
REVERSE反冲单财务处理
REBATE退款单财务处理
CONTRACT合同档案管理
SUPPLIER供应商申请档案管理
CUSTOMER客户档案管理

**单据状态枚举**

**单据状态枚举**单据状态编码**单据状态名称**
DRAFT0草稿
APPRV_PEND1待审批

REJECT4拒绝

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.billCompanyCodesARRAY否单据所属法人公司编码["F000002"]
body.billDepartmentCodesARRAY否单据所属部门编码["00000004"]
body.submitEmpNumbersARRAY否提单人花名册编号["888888"]
body.applyEmpNumbersARRAY否申请人花名册编号["888888"]
body.eventTypesARRAY否见表单业务类型枚举["TRAVEL_APPLY"]
body.billStatusListARRAY否见单据状态枚举["1"]
body.billSubmitBeginTimeSTRING否yyyy-MM-dd hh:mm:ss2023-04-12 10:32:40
body.billSubmitEndTimeSTRING否yyyy-MM-dd hh:mm:ss2025-04-12 10:32:40
body.billFinishBeginTimeSTRINGyyyy-MM-dd hh:mm:ss2023-04-12 10:32:40
body.billFinishEndTimeSTRINGyyyy-MM-dd hh:mm:ss2025-04-12 10:32:40
body.billIdsARRAY否单据编号集合， 最多可传500个[2023042300260455]
body.eventNumbersARRAY否表单编码集合，最多可传50个["01230705110647000002"]
body.eventNamesARRAY否表单名称集合，最多可传50个["出差申请单"]
body.printStatusListARRAY否1:可打印 0：不可打印["0","1"]
body.signStatusListARRAY否N-免签收 W-未签收 F-已签收["N"]
body.startINTEGER否分页开始条数0
body.limitINTEGER否不大于500，不传则默认查前500条数据500
参数类型必填说明示例
queryOBJECTquery根节点

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.bodyOBJECT否返回数据体
body.body.listARRAY否单据列表
body.body.list.billCompanyCodeSTRING否单据所属法人公司编码F000002
body.body.list.billDepartmentCodeSTRING否单据所属部门编码00000004

body.body.list.eventTypeSTRING否见表单业务类型枚举TRAVEL_APPLY
body.body.list.billStatusSTRING否见单据状态枚举1
body.body.list.billSubmitTimeSTRING否yyyy-MM-dd hh:mm:ss2023-04-23 16:00:31
body.body.list.billFinishTimeSTRINGyyyy-MM-dd hh:mm:ss2023-04-23 16:00:31
body.body.list.billIdLONG否单据编号2023042300260455
body.body.list.eventNumberSTRING否表单编码01230705110647000002
body.body.list.eventNameSTRING否表单名称出差
body.body.list.printStatusSTRING否1:可打印 0：不可打印1

body.body.list.billAmountBIGDECIMAL否保留两位小数0
