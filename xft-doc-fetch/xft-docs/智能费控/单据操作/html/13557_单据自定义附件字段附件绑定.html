单据自定义附件字段附件绑定

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/accessory/custom-field/bind

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/accessory/custom-field/bind

接口描述

支持附件绑定至单据附件自定义字段，只支持追加附件至附件自定义字段，不支持更新或删除

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.billIdLONG是所有单据类型、所有单据状态都支持2024022903021009
body.fileInfosARRAY是附件信息
body.fileInfos.fieldNumberSTRING是附件自定义字段编码ACCESSORIES_104631417
body.fileInfos.fileIdsARRAY是单据单个附件字段最大支持附件数为100，
文件编号为上传接口返回的fileId		["c35397d6-5234-4fb2-b874-625951f475c9","c35397d6-5234-4fb2-b874-625951f47517"]
参数类型必填说明示例
queryOBJECTquery根节点

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.successBOOLEAN否绑定成功标志true
body.errorMessageSTRING否绑定失败报错信息单据单个附件字段绑定附件数不可超过100

接口调用示例

请求报文


{
    "billId": 2024022202607003,
    "fileInfos": [
        {
            "fieldNumber": "ACCESSORIES_105702744",
            "fileIds": [
                "c35397d6-5234-4fb2-b874-625951f475c9",
                "c35397d6-5234-4fb2-b874-625951f47517"
            ]
        }
    ]
}

响应报文


{
    "success": true,
    "errorMessage":null
}
