# 批量查询单据列表

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/bill-list 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/bill-list 
 ## 接口描述 
根据查询条件分页查询单据基本信息列表
**表单业务类型枚举**
| **业务类型编码** | **表单业务类型含义** | **对应表单属性** |
| :--------------- | :------------------- | :--------------- |
| TRAVEL_APPLY     | 差旅申请             | 申请单           |
| EXPENSE_APPLY    | 费用申请             | 申请单           |
| INVITATION_APPLY | 招待申请             | 申请单           |
| USE_CAR_APPLY    | 用车申请             | 申请单           |
| COMMON_APPLY     | 其他通用申请         | 申请单           |
| TRAVEL_REIM      | 差旅报销             | 报销单           |
| COMMON_REIM      | 日常报销             | 报销单           |
| EMP_LEND         | 员工借款             | 借还款单         |
| REPAYMENT        | 还款单               | 借还款单         |
| SUPPLIER_LEND    | 预付款               | 对公单据         |
| SUPPLIER_REIM    | 对公报账             | 对公单据         |
| SUPPLIER_SETTLE  | 对公结算             | 对公单据         |
| PAY_APPLY        | 付款申请             | 对公单据         |
| REVERSE          | 反冲单               | 财务处理         |
| REBATE           | 退款单               | 财务处理         |
| CONTRACT         | 合同                 | 档案管理         |
| SUPPLIER         | 供应商申请           | 档案管理         |
| CUSTOMER         | 客户                 | 档案管理         |    

**单据状态枚举**

| **单据状态枚举**    | 单据状态编码 | **单据状态名称**                     |
| :------------------ | ------------ | :----------------------------------- |
| DRAFT               | 0            | 草稿                                 |
| APPRV_PEND          | 1            | 待审批                               |
| APPRV               | 2            | 审批通过                             |
| REJECT              | 4            | 拒绝                                 |
| AUDIT_PEND          | 5            | 待财务审核                           |
| CHECK_PEND          | 6            | 待财务复核                           |
| AUDIT               | 7            | 财务审核通过                         |
| CHECK               | 8            | 财务复核通过                         |
| AUDIT_REJECT        | 9            | 财务审核驳回                         |
| CHECK_REJECT        | 10           | 财务复核驳回                         |
| RECALL              | 11           | 已撤回                               |
| BACK                | 12           | 已退回                               |
| DISCARD             | 15           | 已废弃                               |
| TEMP                | 16           | 暂存                                 |
| DISCARDING          | 17           | 废弃中                               |
| WAIT_OUTSIDE_HANDLE | 18           | 等待外部处理                         |
| CHANGING            | 19           | 变更中                               |
| CLOSED              | 20           | 已停用（目前只用于申请单禁止被关联） |
| DELETED             | 21           | 已删除                               | 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.billCompanyCodes | ARRAY | 否 | 单据所属法人公司编码 | ["F000002"] |
| body.billDepartmentCodes | ARRAY | 否 | 单据所属部门编码 | ["00000004"] |
| body.submitEmpNumbers | ARRAY | 否 | 提单人花名册编号 | ["888888"] |
| body.applyEmpNumbers | ARRAY | 否 | 申请人花名册编号 | ["888888"] |
| body.eventTypes | ARRAY | 否 | 见表单业务类型枚举 | ["TRAVEL_APPLY"] |
| body.billStatusList | ARRAY | 否 | 见单据状态枚举 | ["1"] |
| body.billSubmitBeginTime | STRING | 否 | yyyy-MM-dd hh:mm:ss | 2023-04-12 10:32:40 |
| body.billSubmitEndTime | STRING | 否 | yyyy-MM-dd hh:mm:ss | 2025-04-12 10:32:40 |
| body.billFinishBeginTime | STRING |  | yyyy-MM-dd hh:mm:ss | 2023-04-12 10:32:40 |
| body.billFinishEndTime | STRING |  | yyyy-MM-dd hh:mm:ss | 2025-04-12 10:32:40 |
| body.billIds | ARRAY | 否 | 单据编号集合， 最多可传500个 | [2023042300260455] |
| body.eventNumbers | ARRAY | 否 | 表单编码集合，最多可传50个 | ["01230705110647000002"] |
| body.eventNames | ARRAY | 否 | 表单名称集合，最多可传50个 | ["出差申请单"] |
| body.printStatusList | ARRAY | 否 | 1:可打印 0：不可打印 | ["0","1"] |
| body.signStatusList | ARRAY | 否 | N-免签收 W-未签收 F-已签收 | ["N"] |
| body.start | INTEGER | 否 | 分页开始条数 | 0 |
| body.limit | INTEGER | 否 | 不大于500，不传则默认查前500条数据 | 500 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
