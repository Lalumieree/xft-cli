# 单据自定义附件字段附件上传

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/accessory/custom-field/upload 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/accessory/custom-field/upload 
 ## 接口描述 
支持单据自定义附件字段附件上传，并返回文件唯一编号，可结合单据自定义附件字段附件绑定接口使用。
上传附件单个大小不超过50M。
支持的文件类型为：.csv,.pdf, .zip, .rar, .gif,.bmp, .png, .jpg, .docx, .doc, .xlsx, .xls, .tif, .tiff, .jpg, .jpeg, .jpe, .jfif, .dib, .ppt, .pptx, .pub, .wps, .dps, .et, .pps, .csv, .heif, .heic 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: form-data 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.file | FILE | 是 | 文件流 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.uploaderNumber | STRING | 是 | 取值花名册中的员工号 | 01071828 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.fileId | STRING | 否 | 文件唯一键 | c35397d6-5234-4fb2-b874-625951f475c9 |
| body.success | BOOLEAN | 否 | 上传成功标志 | true |
| body.errorMessage | STRING | 否 | 上传失败报错信息 | 单个附件大小超过50M |
 
## 接口调用示例 
 {BaseURL}/v1/accessory/custom-field/upload?uploaderNumber=01071828
注意文件上传，参数file
### 请求报文
无
### 响应报文
```json
{  
    "fileId": "c35397d6-5234-4fb2-b874-625951f475c9",
    "success": true,
    "errorMessage":null
} 
```
