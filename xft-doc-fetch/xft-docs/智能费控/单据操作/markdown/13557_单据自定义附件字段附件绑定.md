# 单据自定义附件字段附件绑定

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/accessory/custom-field/bind 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/accessory/custom-field/bind 
 ## 接口描述 
支持附件绑定至单据附件自定义字段，只支持追加附件至附件自定义字段，不支持更新或删除 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.billId | LONG | 是 | 所有单据类型、所有单据状态都支持 | 2024022903021009 |
| body.fileInfos | ARRAY | 是 | 附件信息 |  |
| body.fileInfos.fieldNumber | STRING | 是 | 附件自定义字段编码 | ACCESSORIES_104631417 |
| body.fileInfos.fileIds | ARRAY | 是 | 单据单个附件字段最大支持附件数为100，<br>文件编号为上传接口返回的fileId | ["c35397d6-5234-4fb2-b874-625951f475c9","c35397d6-5234-4fb2-b874-625951f47517"] |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.success | BOOLEAN | 否 | 绑定成功标志 | true |
| body.errorMessage | STRING | 否 | 绑定失败报错信息 | 单据单个附件字段绑定附件数不可超过100 |
 
## 接口调用示例 
 #### 请求报文
```json
{
    "billId": 2024022202607003,
    "fileInfos": [
        {
            "fieldNumber": "ACCESSORIES_105702744",
            "fileIds": [
                "c35397d6-5234-4fb2-b874-625951f475c9",
                "c35397d6-5234-4fb2-b874-625951f47517"
            ]
        }
    ]
}
```
#### 响应报文
```json
{
    "success": true,
    "errorMessage":null
} 
```  
## 错误码说明 
| 错误码| 错误信息|
  | --- | --- |
  | 200| 成功|
