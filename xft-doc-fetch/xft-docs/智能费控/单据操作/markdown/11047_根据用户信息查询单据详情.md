# 根据用户信息查询单据详情

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/queryBillByCondition 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/queryBillByCondition 
 ## 接口描述 
根据用户手机号和单据状态查询单据详情 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 | **参数**                       | **类型**                              | **必填** | **位置** | **长度** | **说明**         | **备注**                                                     |
| ------------------------------ | ------------------------------------- | -------- | -------- | -------- | ---------------- | ------------------------------------------------------------ |
| OPAUID                         | String                                | 是       | param    | 8       | 公司标识         | 传薪福通企业号                                               |
| queryCondition                 | Object:BillExtDetailQueryConditionDto | 是       | body     |          | 单据查询条件     | -                                                            |
| &ensp;&ensp;∟&ensp;billApprSts | String                                | 否       | body     |       -   | 单据审批状态     | -                                                            |
| &ensp;&ensp;∟&ensp;billStatus  | String                                | 否       | body     | -        | 单据状态枚举类型 | [参考字典](https://xft.cmbchina.com/open/#/home/open-document?anchor11%E5%8D%95%E6%8D%AE%E7%8A%B6%E6%80%81%E6%9E%9A%E4%B8%BE&id=10692&mid=11585) |
| &ensp;&ensp;∟&ensp;empId       | String                                | 否       | body     | 20       | 操作人员编号     | -                                                            |
| &ensp;&ensp;∟&ensp;empSeqId    | Long                                  | 否       | body     | 20       | 操作人员自增ID   | -                                                            |
| &ensp;&ensp;∟&ensp;phoneNumber | String                                | 否       | body     | -        | 手机号     | -                                                            |
| &ensp;&ensp;∟&ensp;comId       | Long                                  | 否       | body     | 20       | 所属公司ID       | -                                                            |
| &ensp;&ensp;∟&ensp;limit       | int                                   | 否       | body     | -        | 分页数量         | -                                                            |
| &ensp;&ensp;∟&ensp;start       | int                                   | 否       | body     | -        | 分页起始         | -                                                            | 
## 接口输出参数
 | **参数**                  | **类型**                      | **说明**    | **备注**        |
|---------------------------------|-------------------------------|----------------------------------------------------------------|----------------------------------------------------------------|
| list                    | Array: Long           | 单据ID列表|-|
|totalCount                    | String                        | 分页需要的总数                                                     |-| 
## 接口调用示例 
 ###请求示例
```json
{BaseURL}/api/itrip-gateway/v1/bills/queryBillByCondition?OPAUID={OPAUID}
{
  "billApprSts": null,
  "billStatus": "1",
  "comId": null,
  "empId": null,
  "empSeqId": null,
  "phoneNumber": 14900000010,
  "limit": 10,
  "start": 0
}
```

### 返回示例
```json
{
	list: [2021092800903820, 2021092800903821],
	"totalCount": 2
}
```
