入库单修改事件

基本信息

事件名称: 入库单修改事件

事件编号: ITRIPINVC

事件描述

入库单修改及金额发票状态发生变化时，发送该事件，用于通知入库单修改的场景

触发情况

1、入库单手工修改并产生修改记录

2、入库单与发票金额匹配状态发生变更后
