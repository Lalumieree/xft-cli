集采平台耗材结算-产品明细状态变更

基本信息

事件名称: 集采平台耗材结算-产品明细状态变更

事件编号: ITRIPEBPCSHSA

事件描述

医招智付耗材结算确认、退回、无需确认操作成功后，向指定的相关人员或系统发送通知，告知医院入库单产品明细状态的变化情况。

- 注意事项

每批次上限500条数据：系统一次最多能推送500条数据变更。如果需要发送的通知数量超过此限制，会分批发送。这有助于确保每次传输的数据量不至于过大，影响系统的稳定性和效率。

触发情况

在医招智付-集采平台-结算确认-耗材结算（对比模式）-待确认页签：

- 一致/不一致：单笔/批量确认结算、单笔/批量退回操作

- 仅存在医院结算：单笔/批量无需确认操作
