集采平台药品结算明细状态变更

基本信息

事件名称: 集采平台药品结算明细状态变更

事件编号: ITRIPEBPDSDSA

事件描述

医招智付集采平台药品结算明细状态发生变更（由待确认变为已确认或已退回）完成后，向指定的相关人员或系统发送通知，告知明细状态的变化情况。

注意事项

* 每批次上限500条数据：系统一次最多能推送500条数据变更。如果需要发送的通知数量超过此限制，会分批发送。这有助于确保每次传输的数据量不至于过大，影响系统的稳定性和效率。

触发情况

触发场景：医招智付-集采平台-结算确认-药品结算：对待确认结算明细进行确认或退回操作

* 确认成功：明细状态由待确认变为已确认

* 退回成功：明细状态有待确认变更已退回
