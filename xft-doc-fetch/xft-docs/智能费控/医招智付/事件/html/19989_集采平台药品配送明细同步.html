集采平台药品配送明细同步

基本信息

事件名称: 集采平台药品配送明细同步

事件编号: ITRIPEBPSYNC

事件描述

医招智付-集采平台-药品配送明细数据同步后，向指定的相关人员或系统发送通知。

注意事项

每批次上限500条数据：系统一次最多能推送500条数据同步消息。如果需要发送的通知数量超过此限制，会分批发送。这有助于确保每次传输的数据量不至于过大，影响系统的稳定性和效率。

触发情况

触发场景：

1. 使用api接口【导入集采平台药品配送明细】同步相关数据时，会将同步成功的数据发送。

2. 在医招智付-集采平台-规则设置 配置了自动入库确认环节，并且开启了同步定时任务，定时任务成功拉取集采平台数据后会将同步成功的数据发送。

3. 在医招智付-集采平台-自动入库-药品验收、药品收货、药品入库等菜单手动更新数据，会将同步成功的数据发送。
