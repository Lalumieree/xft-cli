# 自动入库预入库单入库事件

## 基本信息 
 
 事件名称: 自动入库预入库单入库事件 
 事件编号: ITRIPPREINVCHG  
## 事件描述 
 医招智付-集采平台-自动入库 预入库单入库后，向指定的相关人员或系统发送通知。

注意事项：

每批次上限500条数据：系统一次最多能推送500条数据同步消息。如果需要发送的通知数量超过此限制，会分批发送。这有助于确保每次传输的数据量不至于过大，影响系统的稳定性和效率。 
## 触发情况
 触发场景：

1. 医招智付-集采平台-自动入库 中手动进行标记入库成功；
2. 医招智付-集采平台-自动入库 中手动推送入库成功。
