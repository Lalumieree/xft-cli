# 单笔查询结算单详情

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/settle/detail 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/settle/detail 
 ## 接口描述 
单笔获取结算单的单据详情
注意：结算单为EventType值为SUPPLIER_SETTLE的单据 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json
