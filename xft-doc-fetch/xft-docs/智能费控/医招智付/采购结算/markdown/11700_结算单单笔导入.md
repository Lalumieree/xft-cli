# 结算单单笔导入

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchaseSettle/importSettleOrder/single 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchaseSettle/importSettleOrder/single 
 ## 接口描述 
结算单单笔导入，用于客户在医招智付场景下，支持导入商品明细维度的结算单。结算单导入成功后返回单据编号。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
