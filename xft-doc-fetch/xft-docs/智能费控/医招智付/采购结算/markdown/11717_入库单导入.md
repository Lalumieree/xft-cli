# 入库单导入

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchaseSettle/importInveOrder 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchaseSettle/importInveOrder 
 ## 接口描述 
入库单导入，用于客户在医招智付场景下，将从医院的业务系统中的数据推送至医招智付系统。
如果入库单在系统已存在且状态为“未结算”，重复导入则为更新操作；如果状态不为“未结算”，导致整批导入失败。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
