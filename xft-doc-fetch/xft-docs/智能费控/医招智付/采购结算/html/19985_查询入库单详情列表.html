查询入库单详情列表

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchaseSettle/pageInveOrderDetails

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchaseSettle/pageInveOrderDetails

接口描述

入库单详情列表查询接口

一页最多不能超过100条

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.billCompanyCodeListARRAY是医招智付-系统设置-基础设置-法人公司-公司编码["F000001"]
body.inveOderApplyDateEndSTRING否yyyy-MM-dd2021-10-11
body.inveOderApplyDateStartSTRING否yyyy-MM-dd2020-10-11
body.inveTypeListARRAY否inOrder-入库 returnOrder-退货 specialReturnOrder-特殊出库["inOrder"]
body.orderStatusListARRAY否noSettle-未结算，settling-结算中， settled-已结算，invalid-作废["noSettle"]
body.supCodeListARRAY否无["SUP00012"]
body.startINTEGER默认为0
body.limitINTEGER默认为100
body.inveOrderIdListARRAY入库单号不得超过100个["CODE1","CODE2"]
body.relateInvoiceStartTimeSTRINGyyyy-MM-dd HH:mm2024-12-12 18:46
body.relateInvoiceEndTimeSTRINGyyyy-MM-dd HH:mm2024-12-15 18:46
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是薪福通企业号XFT00001
