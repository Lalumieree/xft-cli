结算单单笔导入

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchaseSettle/importSettleOrder/single

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchaseSettle/importSettleOrder/single

接口描述

结算单单笔导入，用于客户在医招智付场景下，支持导入商品明细维度的结算单。结算单导入成功后返回单据编号。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.corporateCodeLONG否字段下线中请通过薪福通法人公司编码传值0000
body.billCompanyCodeSTRING是薪福通法人公司编码Fx00001
body.depIdSTRING否费控部门编号和薪福通部门编码只需填写一个1052
body.billDepartmentCodeSTRING费控部门编号和薪福通部门编码只需填写一个
body.eventSeqIdLONG否用于指定费控的表单
如果入参中需要传自定义字段，则该字段必传		10866
body.thirdPartyIdSTRING是第三方流水号5867
body.categorySTRING类别
body.submitTimeSTRING否时间格式为 yyyy-MM-dd HH:mm:ss。2022-09-26 11:00:00
body.supIdSTRING是费控供应商编号SUP20220919111756
body.customFieldValuesARRAY否结算单自定义字段
body.customFieldValues.customFieldSeqIdLONG否结算单自定义字段ID505303
body.customFieldValues.customFieldValueSTRING否结算单自定义字段值{"data": "结算单自定义字段1的值"}
body.invoicesARRAY否发票信息
body.invoices.amountBIGDECIMAL否（最多十五位整数，六位小数）10.26
body.invoices.billingDateSTRING否格式为：yyyy-MM-dd2022-09-17
body.invoices.invoiceCodeSTRING否发票代码365451
body.invoices.invoiceNumberSTRING是发票号码1004273
body.prodDetailsARRAY否商品明细
body.prodDetails.customFieldValuesARRAY否商品明细自定义字段
body.prodDetails.customFieldValues.customFieldSeqIdLONG否商品明细自定义字段ID50489
body.prodDetails.customFieldValues.customFieldValueSTRING否商品明细自定义字段值{"data": "商品明细1自定义字段1的值"}
body.prodDetails.lineNumSTRING否行号45
body.prodDetails.measureUnitSTRING否计量单位箱
body.prodDetails.priceDOUBLE是整数位不能超过15位，小数位不能超过6位不可为负数2.5
body.prodDetails.prodCodeSTRING否商品编码16767629
body.prodDetails.prodMoneyBIGDECIMAL否（最多十五位整数，六位小数）1510.26
body.prodDetails.prodNameSTRING是商品名称矿泉水
body.prodDetails.prodSpecSTRING否规格500ml
body.prodDetails.quantityBIGDECIMAL是（最多十五位整数，六位小数）510.26
body.prodDetails.settleAmountBIGDECIMAL否（最多十五位整数，六位小数）7510.26
body.prodDetails.taxRateBIGDECIMAL否（最多十五位整数，六位小数）010.26
body.prodDetails.isPriceIncludeTaxSTRING是0-否 1-是1
body.prodDetails.prodRemarkSTRING备注
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING薪福通企业号XFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.dataLONG否结算单单据编号2022123000211013
body.successBOOLEAN否是否成功true
body.messageSTRING否提示信息调用成功
