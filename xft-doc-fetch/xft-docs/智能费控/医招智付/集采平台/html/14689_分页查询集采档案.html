分页查询集采档案

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-settle/mapping/archive/page

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-settle/mapping/archive/page

接口描述

用于医招智付场景分页查询集采档案信息，对应页面为医招智付-集采平台-规则设置-集采档案处列表页，单次查询不能超过500条

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.platformCodeSTRING是集采平台编码
江苏省平台：jiangSuProvincePlatform
福建省平台：fuJianProvincePlatform
江西省平台：jiangXiProvincePlatform		fuJianProvincePlatform
root.archiveNameSTRING是归属集采档案名称，对应医招智付-集采平台-规则设置-集采档案的相应集采档案名称特殊药品映射
root.sourceIdListARRAY否规则输入值列表，对应新增集采档案时字段属性为规则输入值的表头字段，最大500条["21934"]
root.startINTEGER是起始页数0
root.limitINTEGER是最大5005
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.bodyOBJECT否返回体
root.body.totalLONG否符合条件总数量1
root.body.listARRAY否集采档案信息列表
root.body.list.sourceIdSTRING否对应新增集采档案时字段属性为规则输入值的表头字段21934
root.body.list.sourceNameSTRING否对应新增集采档案时字段属性为规则输入说明的表头字段注射用甲苯磺酸瑞马唑仑
root.body.list.targetIdSTRING否对应新增集采档案时字段属性为规则产出结果的表头字段是
root.body.list.targetNameSTRING否对应新增集采档案时字段属性为产出结果说明的表头字段null
root.returnCodeSTRING否错误码SUC0000
root.successBOOLEAN否是否调用成功true
root.messageSTRING否错误信息成功

接口调用示例

请求报文


{
  "platformCode": "fuJianProvincePlatform",
  "archiveName":"特殊药品映射",
  "sourceIdList":["21934"],
  "start": 0,
  "limit": 5
}

响应报文


{
    "body": {
        "total": 1,
        "list": [
            {
                "sourceId": "21934",
                "sourceName": "注射用甲苯磺酸瑞马唑仑",
                "targetId": "是",
                "targetName": null
            }
        ]
    },
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}  
## 错误码说明 
| 错误码| 错误信息|
  | --- | --- |
  | EROOR1| 调用失败|
  | EROOR2| 执行失败，分页查出最大限制、当前企业未配置集采平台信息等|
  | SUC0000| 执行成功|
