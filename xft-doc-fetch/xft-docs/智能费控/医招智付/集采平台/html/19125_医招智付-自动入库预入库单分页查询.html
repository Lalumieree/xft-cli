医招智付-自动入库预入库单分页查询

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-settle/pre-inventory/page-query

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-settle/pre-inventory/page-query

接口描述

分页查询医招智付-集采平台-自动入库-药品入库的入库单信息。

注：

分页页面大小不超过200；

查询的生成入库单时间区间不能超过1年；

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT是请求体根节点
root.startINTEGER是分页条件--起始位置0
root.limitINTEGER是分页条件--分页大小20
root.inventoryCodeSTRING模糊搜索"20250923001"
root.orderApplyDateStartSTRING是医招智付-集采平台-自动入库-药品入库中，配送明细转化成入库单的时间；
时间格式：yyyy-MM-dd；
查询时间区间不能超过1年。		"2025-10-23"
root.orderApplyDateEndSTRING是医招智付-集采平台-自动入库-药品入库中，配送明细转化成入库单的时间；
时间格式：yyyy-MM-dd;
查询时间区间不能超过1年。		"2025-10-23"
root.deliverEnterpriseCodeSTRING模糊搜索"NAJING02"
root.deliverEnterpriseNameSTRING模糊搜索"南京华东医药有限公司"
root.downloadStatusSTRING待下载：none；
下载成功：success ；
下载失败：failed		"none"
root.pushDateStartSTRING医招智付-集采平台-自动入库-药品入库中，推送入库单至外部系统的时间；
时间格式：yyyy-MM-dd		"2025-10-23"
root.pushDateEndSTRING医招智付-集采平台-自动入库-药品入库中，推送入库单至外部系统的时间；
时间格式：yyyy-MM-dd		"2025-10-23"
root.pushStatusListARRAY待推送：none；
推送成功：success ；
推送失败：failed。

待推送（none）和推送失败（failed）状态为预入库数据；
推送成功（success ）状态为已入库数据。
如果通过页面”标记入库“操作入库，则以入库状态为准。		["success"]
root.inventoryStatusSTRING预入库单推送至外部系统入库的状态
"已入库"：Y；
”未入库“：N		"Y"
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT是响应体根节点
root.successBOOLEAN是请求成功：true；
请求失败：false		true
root.returnCodeSTRING是调用失败：EROOR1；
执行失败：EROOR2；
执行成功：SUC0000；
参数错误：E0002		"SUC0000"
root.messageSTRING是执行成功时，会返回的信息：”成功“；
执行失败时，会返回错误信息。		"成功"
root.bodyOBJECT是返回的入库单数据
root.body.totalINTEGER是符合条件的数据总数200
root.body.listARRAY是返回的入库单数据
root.body.list.inventoryCodeSTRING是预入库单号"INV202502000001"
root.body.list.pushStatusSTRING是待推送：none；
推送成功：success ；
推送失败：failed		"none"
root.body.list.orderApplyDateSTRING否时间格式为 yyyy-MM-dd"2025-02-25"
root.body.list.deliverEnterpriseCodeSTRING否配送企业代码"DEP001"
root.body.list.deliverEnterpriseNameSTRING否配送企业名称"南京华东医药有限责任公司"
root.body.list.pushTimeSTRING医招智付-集采平台-自动入库-药品入库中，推送入库单至外部系统的时间；
时间格式：yyyy-MM-dd HH:mm:ss		"2025-02-25 17:00:00"
root.body.list.downloadStatusSTRING否待下载：none；
下载成功：success ；
下载失败：failed
root.body.list.pushFailedReasonSTRING医招智付-集采平台-自动入库-药品入库中，推送入库单至外部系统失败的原因"网络异常"
root.body.list.distributionCodeSTRING是入库单关联的配送明细，多个配送明细由英文逗号,隔开"S123456,S123"
root.body.list.inventoryStatusSTRING是预入库单推送至外部系统入库的状态
"已入库"：Y；
”未入库“：N		"Y"
root.body.list.customFieldsARRAY否医招智付-集采平台-规则设置中配置的预入库单自定义字段名称及其值
root.body.list.customFields.customFieldNameSTRING否自定义字段名称"操作人"
root.body.list.customFields.customFieldValueSTRING否自定义字段填写值"lucy"
root.body.list.productDetailsARRAY是产品明细列表
root.body.list.productDetails.productNameSTRING是产品名称"布洛芬片"
root.body.list.productDetails.quantityDOUBLE是整数位不超过15位，小数位不超过6位5
root.body.list.productDetails.priceDOUBLE是整数位不超过15位，小数位不超过6位10.2
root.body.list.productDetails.batchNumberSTRING否批次号"00002"
root.body.list.productDetails.productCodeSTRING否产品编码"XM01AEB173A001010100978"
root.body.list.productDetails.invoiceNumberSTRING否发票号码"57351551"
root.body.list.productDetails.productEnterpriseCodeSTRING否生产企业代码"DEP002"
root.body.list.productDetails.productEnterpriseNameSTRING否生产企业名称"浙江我武生物科技股份有限公司"
root.body.list.productDetails.expiredTimeSTRING否时间格式为 yyyy-MM-dd"2035-10-30"
root.body.list.productDetails.customFieldsARRAY否医招智付-集采平台-规则设置中配置的预入库单-产品明细自定义字段名称及其值
root.body.list.productDetails.customFields.customFieldNameSTRING否自定义字段名称"操作人"
root.body.list.productDetails.customFields.customFieldValueSTRING否自定义字段填写值"jack"
