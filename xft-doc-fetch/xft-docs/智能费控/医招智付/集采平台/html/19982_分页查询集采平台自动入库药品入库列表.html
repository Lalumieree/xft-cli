接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-settle/auto-inventory/inventory/page

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-settle/auto-inventory/inventory/page

接口描述

用于医招智付场景分页查询集采平台自动入库药品入库列表，单次查询上限500条

调用方式

CONTENT-TYPE: application/json
