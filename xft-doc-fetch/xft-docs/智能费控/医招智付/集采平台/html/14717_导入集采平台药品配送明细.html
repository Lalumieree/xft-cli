导入集采平台药品配送明细

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-settle/auto-inventory/drug/distribution-detail/import

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-settle/auto-inventory/drug/distribution-detail/import

接口描述

药品配送明细导入，将集采平台业务系统中的药品配送明细数据推送至医招智付应用。

以药品配送明细编码作为唯一键，重复导入时进行更新操作。

批量接口，允许部分成功。

单次导入上限为1000。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootARRAY请求体根节点
root.approveNoSTRING否批准文号"国药准字H20183260"
root.batchNumberSTRING否批次号"241216005"
root.contactNameSTRING联系人"中医院会计部"

root.deliverEnterpriseNameSTRING否配送企业名称"南京华东医药有限责任公司"

root.deliveryCountBIGDECIMAL否整数位不能超过15位，小数位不能超过6位12
root.deliveryTimeSTRING否"yyyy-MM-dd HH:mm:ss""2024-12-16 19:26:33"
root.distributionCodeSTRING是唯一编码"S1352024121600004"
root.expiredDateSTRING否"yyyy-MM-dd HH:mm:ss""2024-12-27 08:29:19"
root.hospListCodeSTRING院内目录编码"HS12345456"

root.selInvoiceOneSTRING销售发票号（第一票）"24332000000415737631"
root.modSTRING否型号"滴剂"

root.produceEnterpriseCodeSTRING否生产企业代码"91330000742906207U"
root.produceEnterpriseNameSTRING否生产企业名称"浙江我武生物科技股份有限公司"
root.productCodeSTRING否产品编码"XL01XES267E001010178672"
root.productNameSTRING否产品名称"塞瑞替尼胶囊"
root.productPackageSTRING否包装"黄花蒿花粉变应原舌下滴剂3号:总变应原活性640BU/ml,装量为2ml×1瓶/盒"
root.receiveAddressSTRING否收货地址"达道路药品科"
root.receiveCountBIGDECIMAL否整数位不能超过15位，小数位不能超过6位0
root.sendTimeSTRING否"yyyy-MM-dd HH:mm:ss""2024-12-17 08:29:19"
root.specSTRING否规格"黄花蒿花粉变应原舌下滴剂3号:总变应原活性640BU/ml"
root.storageRoomSTRING否库房"中药房"
root.traceCodesSTRING多个追溯码使用英文分号;拼接"123354645656;434354545657"
root.orderCountBIGDECIMAL整数位不能超过15位，小数位不能超过6位12
root.hospPurchasePriceBIGDECIMAL整数位不能超过15位，小数位不能超过6位50
root.highSelLimitPriceBIGDECIMAL整数位不能超过15位，小数位不能超过6位50
root.selInvoiceDateOneDATE"yyyy-MM-dd""2025-08-06"
root.selInvoiceDateTwoDATE"yyyy-MM-dd""2025-08-06"
root.marketingAuthHolderSTRING上市许可人强生公司（Johnson & Johnson）
root.remarkSTRING药品入库列表、详情使用多行文本展示，多行内容请使用换行符进行换行，换行符支持：“
”或“\r
”		采购计划编号：1345600099872238
存储条件：零下5°

参数类型必填说明示例

参数类型必填说明示例
rootOBJECT响应体根节点
root.resultBOOLEAN否true - 导入成功，包含全部成功或者部分成功
false - 导入失败		true
root.succeedSizeINTEGER否成功条数10
root.failedSizeINTEGER否失败条数0
root.errorMessageSTRING否全局错误信息单次导入上限不能超过1000
root.errorListARRAY否部分成功时，这里返回所有失败的数据列表
root.errorList.distributeCodeSTRING否配送明细编码S1352024121600004
root.errorList.errorMessageSTRING否错误信息配送企业代码长度不能超过100位
