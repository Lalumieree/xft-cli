分页查询耗材院内目录

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-settle/hospital-list/consumables/page

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-settle/hospital-list/consumables/page

接口描述

用于医招智付场景分页查询耗材院内目录信息，对应页面为医招智付-目录管理-耗材院内目录-全部院内目录处列表页，单次查询不能超过500条

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.productCodeListARRAY否单次查询最多返回500条["HXQX21157F71"]
root.hospitalListIdListARRAY否单次查询最多返回500条["1854422767814877186"]
root.startINTEGER是起始页数0
root.limitINTEGER是单次查询不能超过500条50
参数类型必填说明示例
rootOBJECTquery根节点
