# 导入集采档案条目

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-settle/mapping/archive/import 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-settle/mapping/archive/import 
 ## 接口描述 
本接口适用于医招智付应用，批量添加集采档案数据，调用本接口后可在医招智付-集采平台-规则设置-集采档案查看导入的数据。

调用本接口时请注意，单批次最多添加500条数据，支持部分成功，部分失败， 其中失败的信息可以在返回报文中查看具体的错误信息。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|

| root.itemList.sourceName | STRING | 否 | 源字段名称 | "阿司匹林片" |
| root.itemList.targetId | STRING | 是 | 映射字段id | "YP001-11" |
| root.itemList.targetName | STRING | 否 | 映射字段名称 | "阿司匹林片（新国标）" |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.returnCode | STRING | 是 | "SUC0000" - 请求成功<br>"EROOR2" -业务执行错误 | "SUC0000" |

| root.body | ARRAY | 否 | 返回体 |  |
| root.body.sourceId | STRING |  | 源字段id |  |
| root.body.errorMessage | STRING |  | 错误原因 | "源字段id长度不能超过128位" |
 
## 接口调用示例 
 ### 请求报文
```json
{
    "platformCode": "fuJianProvincePlatform",
    "archiveName": "药品映射",
    "itemList": [
        {
            "sourceId": "YP001",
            "sourceName": "阿司匹林片",
            "targetId": "YP001-11",
            "targetName": "阿司匹林片（新国标）"
        }
    ]
}
```
### 响应报文
导入成功
```json
{  
    "returnCode":"SUC0000",  
    "success": true,  
    "message": null,  
    "body":  null
}  
```
导入部分成功
```json
{  
    "returnCode":"SUC0000",  
    "success": true,  
    "message": null,  
    "body": [  
        {  
            "result": false,  
            "sourceId": "S20251224434355",  
            "errorMessage": "长度不可超过128位字符"  
        }
    ]  
}  
```
导入全部失败
```json
{  
    "returnCode":"ERROR1",  
    "success": false,  
    "message": "单次导入条目不能超过500条",  
    "body": null  
}  
```  
## 错误码说明 
| 错误码| 错误信息|解决方案|
  | --- | --- | --- |
  | EROOR1	| 系统错误	| 参考message信息字段
  | EROOR2	| 业务执行错误	| 参考message、errorMessage错误信息字段
