# 分页查询集采平台映射规则

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-settle/mapping/rule/page 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-settle/mapping/rule/page 
 ## 接口描述 
用于医招智付场景查询集采平台与院内的字段映射规则，例如药品映射规则与供应商映射规则等，单次查询上限500条 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.mapType | STRING | 是 | 供应商映射规则：supplier、药品映射规则：drug | drug |
| root.platformCode | STRING | 是 | 集采平台编码<br>江苏省平台：jiangSuProvincePlatform <br>福建省平台：fuJianProvincePlatform <br>江西省平台：jiangXiProvincePlatform | fuJianProvincePlatform |
| root.targetIdList | ARRAY | 否 | 若查询药品映射规则，本字段名称为集采院内目录编码；若查询供应商映射规则，本字段名称为集采配送企业代码；单次查询返回结果最多500条 | ["1771065586239942657"] |
| root.sourceIdList | ARRAY |  | 若查询药品映射规则，本字段名称为院内产品编码；若查询供应商映射规则，本字段名称为供应商编码；单次查询返回结果最多500条 | ["ZE0002"] |
| root.start | INTEGER | 是 | 分页起始页数 | 0 |
| root.limit | INTEGER | 是 | 分页大小 | 50 |

## 接口调用示例 
 #### 请求报文
```json
{
  "mapType": "drug",
  "platformCode": "jiangXiProvincePlatform",
  "sourceIdList":["1610830356198977538","1610830356198977539","1610830356198977537"],
  "targetIdList":["1610830356198977538"],
  "start":0,
  "limit":2
}
```
#### 响应报文
```json
{
    "body": {
        "total": 1,
        "list": [
            {
                "sourceId": "1610830356198977538",
                "sourceName": "复合维生素B片",
                "targetId": "1610830356198977538",
                "targetName": "复合维生素B片"
            }
        ]

## 错误码说明 
| 错误码| 错误信息|
  | --- | --- |
  | EROOR1| 调用失败|
  | EROOR2| 执行失败，分页查出最大限制、当前企业未配置集采平台信息等|
  | SUC0000| 执行成功|
