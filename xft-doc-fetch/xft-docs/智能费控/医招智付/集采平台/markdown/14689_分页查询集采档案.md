# 分页查询集采档案

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-settle/mapping/archive/page 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-settle/mapping/archive/page 
 ## 接口描述 
用于医招智付场景分页查询集采档案信息，对应页面为医招智付-集采平台-规则设置-集采档案处列表页，单次查询不能超过500条 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.platformCode | STRING | 是 | 集采平台编码<br>江苏省平台：jiangSuProvincePlatform <br>福建省平台：fuJianProvincePlatform <br>江西省平台：jiangXiProvincePlatform | fuJianProvincePlatform |
| root.archiveName | STRING | 是 | 归属集采档案名称，对应医招智付-集采平台-规则设置-集采档案的相应集采档案名称 | 特殊药品映射 |
| root.sourceIdList | ARRAY | 否 | 规则输入值列表，对应新增集采档案时字段属性为规则输入值的表头字段，最大500条 | ["21934"] |
| root.start | INTEGER | 是 | 起始页数 | 0 |
| root.limit | INTEGER | 是 | 最大500 | 5 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.body | OBJECT | 否 | 返回体 |  |
| root.body.total | LONG | 否 | 符合条件总数量 | 1 |
| root.body.list | ARRAY | 否 | 集采档案信息列表 |  |
| root.body.list.sourceId | STRING | 否 | 对应新增集采档案时字段属性为规则输入值的表头字段 | 21934 |
| root.body.list.sourceName | STRING | 否 | 对应新增集采档案时字段属性为规则输入说明的表头字段 | 注射用甲苯磺酸瑞马唑仑 |
| root.body.list.targetId | STRING | 否 | 对应新增集采档案时字段属性为规则产出结果的表头字段 | 是 |
| root.body.list.targetName | STRING | 否 | 对应新增集采档案时字段属性为产出结果说明的表头字段 | null |
| root.returnCode | STRING | 否 | 错误码 | SUC0000 |
| root.success | BOOLEAN | 否 | 是否调用成功 | true |
| root.message | STRING | 否 | 错误信息 | 成功 |
 
## 接口调用示例 
 #### 请求报文
```json
{
  "platformCode": "fuJianProvincePlatform",
  "archiveName":"特殊药品映射",
  "sourceIdList":["21934"],
  "start": 0,
  "limit": 5
}
```
#### 响应报文
```json
{
    "body": {
        "total": 1,
        "list": [
            {
                "sourceId": "21934",
                "sourceName": "注射用甲苯磺酸瑞马唑仑",
                "targetId": "是",
                "targetName": null
            }
        ]
    },
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}  
## 错误码说明 
| 错误码| 错误信息|
  | --- | --- |
  | EROOR1| 调用失败|
  | EROOR2| 执行失败，分页查出最大限制、当前企业未配置集采平台信息等|
  | SUC0000| 执行成功|
