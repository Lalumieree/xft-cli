# 导入集采平台药品配送明细

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-settle/auto-inventory/drug/distribution-detail/import 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-settle/auto-inventory/drug/distribution-detail/import 
 ## 接口描述 
药品配送明细导入，将集采平台业务系统中的药品配送明细数据推送至医招智付应用。

以药品配送明细编码作为唯一键，重复导入时进行更新操作。

批量接口，允许部分成功。

单次导入上限为1000。
 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
