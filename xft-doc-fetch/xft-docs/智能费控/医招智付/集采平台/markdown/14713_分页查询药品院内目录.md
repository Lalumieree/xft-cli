# 分页查询药品院内目录

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/purchase-settle/hospital-list/drug/page 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/purchase-settle/hospital-list/drug/page 
 ## 接口描述 
用于医招智付场景分页查询药品院内目录信息，对应页面为医招智付-目录管理-药品院内目录-全部院内目录处列表页，单次查询不能超过500条 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.hospitalListIdList | ARRAY | 否 | 列表元素类型为String，单次查询最多返回500条 | ["12202035564056","1819214365319634946"] |
| root.productCodeList | ARRAY | 否 | 列表元素类型为String，单次查询最多返回500条 | ["XA16AAL002N001010178554"] |
| root.start | INTEGER | 是 | 从第0页开始 | 0 |
| root.limit | INTEGER | 是 | 单次查询最大500条 | 50 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
