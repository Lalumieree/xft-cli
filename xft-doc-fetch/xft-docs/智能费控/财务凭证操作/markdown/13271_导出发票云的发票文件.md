# 导出发票云的发票文件

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/invoice/downloadInvoice 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/invoice/downloadInvoice 
 ## 接口描述 
导出发票云的发票文件，交通类发票输入交通类发票代码，普通类发票输入发票号码和发票代码进行查询下载，返回发票的字节流 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.billId | LONG | 是 | 单据编号 | 2022000000000000 |
| body.trafficEcode | STRING |  | invoiceCode 和 invoiceCode 为空时trafficEcode必传 | 252809 |
| body.invoiceCode | STRING |  | 发票类型为"I", "J"时必传 | 950001600111 |
| body.invoiceNumber | STRING |  | 发票类型为"I", "J"时必传 | 20213877 |
| body.invoiceType | STRING | 是 | 发票类型 | 2 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING |  | 新福通企业号 | XFT00001 |

## 接口调用示例

```json
{
  "invoiceCode": "950001600111",
  "invoiceNumber": "20213877",
  "billId": 2022000000000000,
  "invoiceType": "2"
}
```

