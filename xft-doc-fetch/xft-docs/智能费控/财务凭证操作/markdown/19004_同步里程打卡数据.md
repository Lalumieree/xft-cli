# 同步里程打卡数据

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/route/import 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/route/import 
 ## 接口描述 
本接口适用于批量同步里程打卡信息到智能费控，调用本接口后可在智能费控-工作台-个人报销 查看个人打卡里程信息，财务人员可在 智能费控-财务处理-台账管理-里程台账 查看企业员工里程信息。

**调用本接口时请注意**
* 单批次最多添加20条记录，仅支持全部成功，失败的信息在返回报文中查看具体的错误信息。
* 单条里程记录中必须存在一个开始点和一个结束点，途经点最多1000个。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.routeInf | OBJECT | 是 | 里程信息对象 |  |
| root.routeInf.empNumber | STRING | 是 | 打卡人员工号 | USR10000 |
| root.routeInf.mileage | BIGDECIMAL | 是 | 里程还未结束的情况 endFlag=“0”时，此字段公里数默认是0 | 52.09 |
| root.routeInf.startTime | STRING | 是 | yyyy-MM-dd HH:mm:ss | 2024-10-31 14:37:17 |
| root.routeInf.endTime | STRING | 是 | yyyy-MM-dd HH:mm:ss | 2024-10-31 14:40:07 |
| root.routeInf.startAddressName | STRING | 是 | 开始点地址 | xx省xx市xx区xx街道xx路xx号xx小区 |
| root.routeInf.startAddressDetailName | STRING | 是 | 开始点详细地址 | xx省xx市xx区xx街道xx路xx号xx大厦 |
| root.routeInf.startDeviceNumber | STRING | 否 | 开始打卡设备号 | IOS0000000000 |
| root.routeInf.endAddressName | STRING | 是 | 结束点地址 | xx省xx市xx区xx街道xx路xx号 |
| root.routeInf.endAddressDetailName | STRING | 是 | 结束点详细地址 | xx省xx市xx区xx街道xx路xx号 |
| root.routeInf.endDeviceNumber | STRING | 否 | 结束打卡设备号 | IOS0000000000 |
| root.routeInf.routeRemark | STRING | 否 | 里程备注 | 里程备注 |
| root.routeInf.routeNumber | STRING | 是 | 企业自己的里程唯一编码，**同企业下不允许重复**，为后续补充图片做标识 |  |
| root.routeNodeInfList | ARRAY | 是 | 本次里程的所有打卡点 |  |
| root.routeNodeInfList.nodeType | STRING | 是 | “0”-起点<br>“1” -终点<br>“2” -途径点 | “0” |
| root.routeNodeInfList.addressName | STRING | 是 | 当前打卡点地点 |  |
| root.routeNodeInfList.addressDetailName | STRING | 是 | 当前打卡点精确地点 | xx省xx市xx区xx街道xx路xx号xx小区 |
| root.routeNodeInfList.longitude | STRING | 是 | 经度 | 120.19081265920326 |
| root.routeNodeInfList.latitude | STRING | 是 | 纬度 | 30.1924007359704 |
| root.routeNodeInfList.crtTime | STRING | 是 | yyyy-MM-dd HH:mm:ss | 2024-10-31 14:37:17 |
| root.routeNodeInfList.nodeMileage | DOUBLE | 是 | 开始点传0（当nodeType=“0”） | 0 |
| root.routeNodeInfList.nodeNumber | STRING | 是 | 企业自己打卡点的唯一标识，，**同一里程下不允许重复**，为后续补充图片做标识 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |

## 错误码说明 
| 错误码| 错误信息|解决方案|
  | --- | --- | --- |
  | SUC0000|无| 成功|
  | E0001|业务异常| 请求失败根据提示修改入参重新请求|
  | EROOR1|系统异常| 请求失败联系管理员|
