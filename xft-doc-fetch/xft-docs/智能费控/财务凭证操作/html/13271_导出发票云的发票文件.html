导出发票云的发票文件

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/invoice/downloadInvoice

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/invoice/downloadInvoice

接口描述

导出发票云的发票文件，交通类发票输入交通类发票代码，普通类发票输入发票号码和发票代码进行查询下载，返回发票的字节流

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.billIdLONG是单据编号2022000000000000
body.trafficEcodeSTRINGinvoiceCode 和 invoiceCode 为空时trafficEcode必传252809
body.invoiceCodeSTRING发票类型为"I", "J"时必传950001600111
body.invoiceNumberSTRING发票类型为"I", "J"时必传20213877
body.invoiceTypeSTRING是发票类型2
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING新福通企业号XFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点

接口调用示例

请求报文


{
  "invoiceCode": "950001600111",
  "invoiceNumber": "20213877",
  "billId": 2022000000000000,
  "invoiceType": "2"
}

响应报文


下载发票文件的字节流
