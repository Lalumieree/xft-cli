# 单据打印PDF

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/exportPDF 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/exportPDF 
 ## 接口描述 
根据单据打印模板打印单据信息到PDF流中，是否打印发票、图片附件可选。 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |
| body.billIds | ARRAY | 是 | 目前仅允许一个单据编号 | [2023000000000000] |
| body.printPattern | ARRAY |  | img：全部附件（包括发票附件、单据附件、费用附件、审批评论附件、支付回单，类型支持pdf与图片格式）；<br>invoice：发票附件；<br>billAttachment：单据附件；<br>expenseAttachment：费用附件；<br>apprAttachment：审批评论附件；<br>statement：支付回单； | ["invoice","billAttachment", "apprAttachment","expenseAttachment","statement"] |
| body.invoiceScope | ARRAY |  | electronic：电子发票；<br>paper：纸质的 | ["electronic","paper"] |
| body.templateCode | STRING | 是 | 打印模板编码 | D000 |
| body.empSeqId | LONG |  | 当前用户Id | 1 |
| body.printDetail | BOOLEAN |  | 默认为true，若不打印单据详情，此处传入false | true |
| body.invoicePrintLayout | STRING |  | 一页一张:"one"；<br>一页两张:"two"；<br>一页四张:"four"；<br>若不传默认一页一张 | "one" |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | OPAUID | XFT00001 |

| 参数 | 类型 | 必填 | 说明 | 示例 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
 
## 接口调用示例 
 ### **请求示例**
{BaseURL}/api/itrip-gateway/v1/bills/exportPDF?OPAUID=XFT00001
```json
{
    "billIds":[2023000000000000],
    "templateCode":"D000",
    "empSeqId":null,
    "printPattern":["invoice","billAttachment", "apprAttachment","expenseAttachment","statement"],
    "invoiceScope":["electronic","paper"],
    "invoicePrintLayout": "one",
    "printDetail":true
}
```
### **返回示例**
PDF格式的文件流。
