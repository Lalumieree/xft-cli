# 查询批量下载进度

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/invoice/queryProcess 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/invoice/queryProcess 
 ## 接口描述 
1. 此接口用于查询批量下载任务进度，需要先调用接口v1/invoice/batchInvoiceExport发起下载任务，获取返回的批次号；
2. 根据1中返回的批次号，调用此接口可以查询该批次每个单据的下载进度（下载进度包含下载中或者已完成）；
3. 下载完成的单据可以通过接口v1/invoice/downloadHisInvoice 下载对应的压缩文件。
 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | 薪福通企业号 | XFT00001 |
| query.requestId | INTEGER | 是 | 当收不到回调时，根据请求序号查询批量下载进度 | 1234 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | ARRAY |  | 响应体根节点 |  |
| body.billId | LONG | 是 | 单据编号 | 2023110600451801 |
| body.fileList | ARRAY |  | 文件名列表，附件数量多时会有多个 | ["2024030611921094.zip"] |
| body.reason | STRING |  | 失败原因，成功时为空 |  |
| body.status | STRING | 是 | 0-下载中;1-上传中；2-上传完成 | 0-下载中;1-重启中；2-上传完成；9-下载失败 |
| body.crtTim | DATE |  | 创建时间 | 1709691518000 |
| body.revTim | DATE |  | 更新时间 | 1709772009000 |
| body.requestSeqId | LONG | 是 | 批量下载任务编号 | 217474 |

 ###请求示例
