单据打印PDF

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/exportPDF

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/exportPDF

接口描述

根据单据打印模板打印单据信息到PDF流中，是否打印发票、图片附件可选。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.billIdsARRAY是目前仅允许一个单据编号[2023000000000000]
body.printPatternARRAYimg：全部附件（包括发票附件、单据附件、费用附件、审批评论附件、支付回单，类型支持pdf与图片格式）；
invoice：发票附件；
billAttachment：单据附件；
expenseAttachment：费用附件；
apprAttachment：审批评论附件；
statement：支付回单；		["invoice","billAttachment", "apprAttachment","expenseAttachment","statement"]
body.invoiceScopeARRAYelectronic：电子发票；
paper：纸质的		["electronic","paper"]
body.templateCodeSTRING是打印模板编码D000
body.empSeqIdLONG当前用户Id1
body.printDetailBOOLEAN默认为true，若不打印单据详情，此处传入falsetrue
body.invoicePrintLayoutSTRING一页一张:"one"；
一页两张:"two"；
一页四张:"four"；
若不传默认一页一张		"one"
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是OPAUIDXFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点

接口调用示例

**请求示例**

{BaseURL}/api/itrip-gateway/v1/bills/exportPDF?OPAUID=XFT00001


{
    "billIds":[2023000000000000],
    "templateCode":"D000",
    "empSeqId":null,
    "printPattern":["invoice","billAttachment", "apprAttachment","expenseAttachment","statement"],
    "invoiceScope":["electronic","paper"],
    "invoicePrintLayout": "one",
    "printDetail":true
}

**返回示例**

PDF格式的文件流。
