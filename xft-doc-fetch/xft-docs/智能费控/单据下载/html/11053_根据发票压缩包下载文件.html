根据发票压缩包下载文件

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/invoice/downloadHisInvoice

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/invoice/downloadHisInvoice

接口描述

根据发票压缩包下载文件，通过参数可以控制下载发票文件或者下载发票及附件。该接口需要和接口v1/invoice/batchInvoiceExport配合使用

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是薪福通企业号XFT00001
query.billIdLONG是从回调结果中获取2022000000000000
query.fileNameSTRING是从回调结果中获取1121
query.optionINTEGER否1-默认值，仅下载发票；2-下载发票和附件 需要与发起请求时的option相对应1

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点

接口调用示例

###请求示例

`{BaseURL}/api/v1/invoice/downloadHisInvoice?OPAUID=XFT00001&billId=2022000000000000&fileName=1121`

返回示例

文件流
