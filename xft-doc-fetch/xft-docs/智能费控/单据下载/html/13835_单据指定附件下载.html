单据指定附件下载

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/bills/downloadBillFile

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/bills/downloadBillFile

接口描述

根据单据文件唯一id下载单据上的附件

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING否薪福通企业号XFT00001
query.billFileSeqIdSTRING是文件唯一id13526

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点

接口调用示例

请求报文

{BaseUrl}/v1/bills/downloadBillFile?OPAUID=XFT00001&billFileSeqId=13526

响应报文

返回文件内容：ResponseEntity<byte[]>(data, headers, HttpStatus.OK)；data为文件字节数组，

即byte[] bytes = response.body().bytes()就能拿到文件

错误码说明

错误码错误信息
附件不存在
