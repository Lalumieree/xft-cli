单据发票批量下载——发起下载任务

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/invoice/batchInvoiceExport

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/invoice/batchInvoiceExport

接口描述

此接口用于发起下载任务，下载结果通过回调接口获取,最大支持发起50条单据的下载任务。

关联接口使用说明：

1.使用接口/v1/invoice/batchInvoiceExport发起下载任务，返回批次号；

2. 根据改批次号可以调用接口/v1/invoice/queryProcess查询该批次每个单据的下载进度（下载中或者已完成）；

3. 下载完成的单据可以通过接口/v1/invoice/downloadHisInvoice 下载对应的压缩文件。

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyARRAY是请求体根节点
单据编号数组 最多50个
[2021063000806212]
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是薪福通企业号XFT00001
query.restartBOOLEAN发票任务存在问题时再使用，默认请求请勿携带false
query.detailSTRING如果为1，导出的excel会多一个发票明细的sheet1
query.callbackUrlSTRING优先级最高，否则取默认回调地址
query.optionINTEGER默认值为1，仅下载发票附件；值为2时，可以下载发票附件和单据中的附件。1
