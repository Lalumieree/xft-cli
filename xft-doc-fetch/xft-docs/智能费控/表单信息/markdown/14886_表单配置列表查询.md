# 表单配置列表查询

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/form/query-event-list 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/form/query-event-list 
 ## 接口描述 
分页查询表单配置列表，支持表单名称模糊搜索，表单状态，表单类型筛选 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.start | LONG | 是 | 起始值，必须传入大于等于0 | 0 |
| root.limit | LONG | 是 | 每次查询最大数量，大于等于1 ，允许最大值1000 | 10 |
| root.eventStatus | STRING | 否 | 表单状态 "1" 启用，"0"停用 | 0 |
| root.eventName | STRING | 否 | 表单名称，支持模糊搜索 | 出差 |
| root.eventTypeList | ARRAY | 否 | TRAVEL_APPLY 差旅申请<br><br>EXPENSE_APPLY 费用申请<br><br>INVITATION_APPLY 招待申请<br><br>USE_CAR_APPLY 用车申请<br><br>COMMON_APPLY 其他通用申请<br><br>TRAVEL_REIM 差旅报销<br><br>COMMON_REIM 日常报销<br><br>EMP_LEND 员工借款<br><br>REPAYMENT 还款单<br><br>SUPPLIER_LEND 预付款<br><br>SUPPLIER_REIM 对公报账<br><br>SUPPLIER_SETTLE 对公结算<br><br>PAY_APPLY 付款申请<br><br>REVERSE 反冲单<br><br>REBATE 退款单<br><br>CONTRACT 合同<br><br>SUPPLIER 供应商申请<br><br>CUSTOMER 客户 | ["TRAVEL_APPLY"] |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|

## 接口调用示例 
 ### 请求报文
```json
{
    "start": 0,
    "limit": 10,
    "eventStatus": "0",
    "eventName": "出差",
    "eventTypeList": [
        "TRAVEL_APPLY"
    ]
}
```
### 响应报文
```json
{
    "body": {
        "total": 1,
        "list": [
            {
                "eventSeqId": 19390,
                "eventNumber": "01230705110814000001",
                "eventName": "出差申请",
                "eventStatus": "0",
                "eventType": "TRAVEL_APPLY"
            }
        ]
    },
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}
```
