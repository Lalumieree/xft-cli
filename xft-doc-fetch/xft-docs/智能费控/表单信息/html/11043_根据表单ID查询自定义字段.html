根据表单ID查询自定义字段

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/form/customFields/eventSeqId/{eventSeqId}

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/form/customFields/eventSeqId/{eventSeqId}

接口描述

根据表单ID查询自定义字段

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是薪福通企业号XFT00001
