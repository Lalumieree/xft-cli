表单配置列表查询

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/form/query-event-list

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/form/query-event-list

接口描述

分页查询表单配置列表，支持表单名称模糊搜索，表单状态，表单类型筛选

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.startLONG是起始值，必须传入大于等于00
root.limitLONG是每次查询最大数量，大于等于1 ，允许最大值100010
root.eventStatusSTRING否表单状态 "1" 启用，"0"停用0
root.eventNameSTRING否表单名称，支持模糊搜索出差
root.eventTypeListARRAY否TRAVEL_APPLY 差旅申请

EXPENSE_APPLY 费用申请

INVITATION_APPLY 招待申请

USE_CAR_APPLY 用车申请

COMMON_APPLY 其他通用申请

TRAVEL_REIM 差旅报销

COMMON_REIM 日常报销

EMP_LEND 员工借款

REPAYMENT 还款单

SUPPLIER_LEND 预付款

SUPPLIER_REIM 对公报账

SUPPLIER_SETTLE 对公结算

PAY_APPLY 付款申请

REVERSE 反冲单

REBATE 退款单

CONTRACT 合同

SUPPLIER 供应商申请

CUSTOMER 客户		["TRAVEL_APPLY"]
参数类型必填说明示例
rootOBJECTquery根节点

参数类型必填说明示例

参数类型必填说明示例

root.body.list.eventStatusSTRING否表单状态 "1" 启用，"0"停用0
root.body.list.eventTypeSTRING否TRAVEL_APPLY 差旅申请

接口调用示例

请求报文


{
    "start": 0,
    "limit": 10,
    "eventStatus": "0",
    "eventName": "出差",
    "eventTypeList": [
        "TRAVEL_APPLY"
    ]
}

响应报文


{
    "body": {
        "total": 1,
        "list": [
            {
                "eventSeqId": 19390,
                "eventNumber": "01230705110814000001",
                "eventName": "出差申请",
                "eventStatus": "0",
                "eventType": "TRAVEL_APPLY"
            }
        ]
    },
    "returnCode": "SUC0000",
    "success": true,
    "message": "成功"
}

错误码说明

错误码错误信息解决方案
SUC0000成功
E0001失败根据message提示处理
