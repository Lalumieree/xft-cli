# 查询所有城市信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/common/city/getAllCity 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/common/city/getAllCity 
 ## 接口描述 
查询智能费控差旅的所有城市信息 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | 公司标识 | XFT00001 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | ARRAY |  | 响应体根节点 |  |
| body.cityNameEn | STRING | 否 | 城市英文名称 | HangZhou |
| body.countryNamEn | STRING | 否 | 国家英文名称 | China |
| body.cityName | STRING | 否 | 城市名称 | 杭州 |
| body.cityCode | LONG | 否 | 城市编号 | 12 |
| body.countryNam | STRING | 否 | 国家名称 | 中国 |
 
## 接口调用示例 
 ###请求示例
```json
{BaseURL}/api/itrip-gateway/v1/common/city/getAllCity?OPAUID=XFT00001
```

### 返回示例
```json
[
	{
		"cityNameEn":"HangZhou",
		"countryNamEn":"China",
		"cityName":"杭州",
		"cityCode":"12",
		"countryNam":"中国"
	}
]
```
