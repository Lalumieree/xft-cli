# 批量同步月汇率

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/currency/batch-save 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/currency/batch-save 
 ## 接口描述 
根据币种编码批量同步月汇率
1.新增的时候需要传入适用公司相关字段。
2.修改时不需要传适用公司相关字段，仅用作同步月汇率 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |
| root.currencyConfigList | ARRAY | 否 | 币种信息列表（一次最多同步50个币种） |  |
| root.currencyConfigList.currencyCode | STRING | 是 | 币种代码（国家通用的币种代码），可在智能费控的币种汇率维护页面查看 | USD |
| root.currencyConfigList.applyCompanyType | STRING | 否 | ALL-全部公司，PART-部分公司<br>注：该参数只在新增时支持 | ALL |
| root.currencyConfigList.applyCompanyLpcCodes | ARRAY | 否 | 适用的部门编码列表，适用全部公司时候传空数组。<br>注：该字段只在新增时候生效 | [] |
| root.currencyConfigList.currencyBizCode | STRING | 是 | 公司下唯一 | 88888888 |
| root.currencyConfigList.periodRates | ARRAY | 否 | 汇率信息列表（一次最多同步一年的汇率） |  |
| root.currencyConfigList.periodRates.ratePeriod | STRING | 是 | 目前只支持月汇率，格式为：2023-01 | 2025-01 |
| root.currencyConfigList.periodRates.exchangeRate | BIGDECIMAL | 是 | 精确到6位小数，不足补0 | 7.033333 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.body | STRING |  | 成功信息 | 保存成功 |
| root.returnCode | STRING |  | 错误码 | SUC0000 |
| root.success | BOOLEAN |  | 是否成功 | true |
| root.message | STRING |  | 未成功的错误提示 | 成功 |
 
## 接口调用示例 
 ### 请求报文
```json
{
    "currencyConfigList": [
        {
            "currencyCode": "USD",
            "applyCompanyType": "ALL",
            "currencyBizCode": "333",
            "periodRates": [
                {
                    "ratePeriod": "2025-01",
                    "exchangeRate": 7.033
                }
            ]
        }
    ]
}
```
### 响应报文
```json
{  
    "body": "保存成功",    
    "returnCode": "SUC0000",    
    "success": true,    
    "message": "成功"    
}  
```  
## 错误码说明 
| 错误码| 错误信息|解决方案|
  | --- | --- | --- |
  | E0002| 币种代码不能为空|输入币种代码 |
 | E0002| 币种编码不能为空|输入编码代码 |
 | E0002| 汇率周期不能为空|输入汇率代码 |
 | E0002| 汇率不能为空|输入汇率代码 |
 | E0002| 币种代码不可超过50位|缩减币种代码 |
 | E0002| 币种编码不可超过50位|缩减币种编码 |
 | E0002| 不支持的币种代码：XXX|输入正确的币种代码|
 | E0002| 不合法的汇率周期 |输入正确的汇率周期|
 | E0002| 不合法的适用公司类型 |输入正确的适用公司类型|
