# 获取部门列表信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/common/getOrgInfoList 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/common/getOrgInfoList 
 ## 接口描述 
结算单新增时，外部系统需要根据费控的部门查询到相应名称的编码信息，才能新增 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING | 是 | 格式为XFT00001的企业编号 | XFT00001 |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT |  | 响应体根节点 |  |
| body.success | BOOLEAN | 是 | true-成功,false-失败 | true |
| body.message | STRING | 是 | 数据 |  |
| body.data | ARRAY | 是 | 数据 |  |
| body.data.depId | STRING | 是 | 部门ID | 0001 |
| body.data.depNam | STRING | 是 | 部门名称 | 部门1 |
| body.data.depStatus | STRING | 是 | 状态:0停用1启用 | 0 |
 
## 接口调用示例 
 {BaseURL}/api/v1/common/getOrgInfoList?OPAUID=XFT00001
