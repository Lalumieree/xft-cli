# 查询开户行信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/common/bank/query 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/common/bank/query 
 ## 接口描述 
查询开户行信息，可以通过银行名称模糊搜索 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| header | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| body | OBJECT | 是 | 请求体根节点 |  |
| body.start | INTEGER | 是 | 开始页面最小为0 | 0 |
| body.limit | INTEGER | 是 | 单页数量最小为1<br>单页数量最多5000 | 10 |
| body.branchBankName | STRING |  | 银行支行名称（模糊查询） | 杭州支行 |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| query | OBJECT |  | query根节点 |  |
| query.OPAUID | STRING |  | 薪福通企业号 | XFT00001 |
