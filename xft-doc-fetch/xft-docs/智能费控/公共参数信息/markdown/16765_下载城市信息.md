# 下载城市信息

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/common/city 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/common/city 
 ## 接口描述 
根据城市信息类型下载对应的城市信息。 
## 调用方式 
 METHOD: GET  
 CONTENT-TYPE: application/json 
## 接口输入参数 
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 请求体根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | query根节点 |  |
| root.type | STRING | 是 | CITY_LIST_DOMESTIC：国内城市列表；<br>CITY_LIST_INTERNATIONAL：国际城市列表；<br>CITY_TREE_DOMESTIC：国内城市树；<br>CITY_TREE_INTERNATIONAL：国际城市树 | CITY_LIST_DOMESTIC |
 
## 接口输出参数
 
| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应头根节点 |  |

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| root | OBJECT |  | 响应体根节点 |  |
| root.file | FILE |  | 城市json文件 |  |
