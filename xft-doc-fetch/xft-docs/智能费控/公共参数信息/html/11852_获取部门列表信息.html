获取部门列表信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/common/getOrgInfoList

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/common/getOrgInfoList

接口描述

结算单新增时，外部系统需要根据费控的部门查询到相应名称的编码信息，才能新增

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING是格式为XFT00001的企业编号XFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.successBOOLEAN是true-成功,false-失败true
body.messageSTRING是数据
body.dataARRAY是数据
body.data.depIdSTRING是部门ID0001
body.data.depNamSTRING是部门名称部门1
body.data.depStatusSTRING是状态:0停用1启用0

接口调用示例

{BaseURL}/api/v1/common/getOrgInfoList?OPAUID=XFT00001
