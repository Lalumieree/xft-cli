下载城市信息

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/common/city

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/common/city

接口描述

根据城市信息类型下载对应的城市信息。

调用方式

METHOD: GET

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
参数类型必填说明示例
rootOBJECTquery根节点
root.typeSTRING是CITY_LIST_DOMESTIC：国内城市列表；
CITY_LIST_INTERNATIONAL：国际城市列表；
CITY_TREE_DOMESTIC：国内城市树；
CITY_TREE_INTERNATIONAL：国际城市树		CITY_LIST_DOMESTIC

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.fileFILE城市json文件

接口调用示例

请求报文


{BaseUrl}/v1/common/city/?type=CITY_LIST_INTERNATIONAL

响应报文


以字节流文件形式返回，文件名：CITY_LIST_INTERNATIONAL.json

参数说明：

参数类型名称备注
codeString城市编号
nameString城市名称
nameEnString城市英文名
cityPathString城市路径
pathNameString路径名称
typeCityPlaceTypeEnum地址类型COUNTRY("国家"), PROVINCE("省"), PROVINCE_LEVEL_CITY("省级行政区、直辖市"), SPECIAL_ADMINISTRATIVE_REGION("特别行政区"), PREFECTURE_LEVEL_CITY("地级行政区、地级市"), COUNTY_LEVEL_CITY("区/县级行政区，区、县、县级市")
parentCodeString上级编码
countryNameString国家名称
countryNameEnString国家名称英文名
countryCodeString国家编码
enableboolean是否启用中
domesticboolean是否国内城市
childrenArray城市子节点列表

错误码说明

错误码错误信息解决方案
CITY_TYPE_NOT_EXIST城市类型文件信息不存在确认入参城市信息类型是否正确
