批量同步月汇率

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/currency/batch-save

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/currency/batch-save

接口描述

根据币种编码批量同步月汇率

1.新增的时候需要传入适用公司相关字段。

2.修改时不需要传适用公司相关字段，仅用作同步月汇率

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
rootOBJECT请求头根节点
参数类型必填说明示例
rootOBJECT请求体根节点
root.currencyConfigListARRAY否币种信息列表（一次最多同步50个币种）
root.currencyConfigList.currencyCodeSTRING是币种代码（国家通用的币种代码），可在智能费控的币种汇率维护页面查看USD
root.currencyConfigList.applyCompanyTypeSTRING否ALL-全部公司，PART-部分公司
注：该参数只在新增时支持		ALL
root.currencyConfigList.applyCompanyLpcCodesARRAY否适用的部门编码列表，适用全部公司时候传空数组。
注：该字段只在新增时候生效		[]
root.currencyConfigList.currencyBizCodeSTRING是公司下唯一88888888
root.currencyConfigList.periodRatesARRAY否汇率信息列表（一次最多同步一年的汇率）
root.currencyConfigList.periodRates.ratePeriodSTRING是目前只支持月汇率，格式为：2023-012025-01
root.currencyConfigList.periodRates.exchangeRateBIGDECIMAL是精确到6位小数，不足补07.033333
参数类型必填说明示例
rootOBJECTquery根节点

接口输出参数

参数类型必填说明示例
rootOBJECT响应头根节点
参数类型必填说明示例
rootOBJECT响应体根节点
root.bodySTRING成功信息保存成功
root.returnCodeSTRING错误码SUC0000
root.successBOOLEAN是否成功true
root.messageSTRING未成功的错误提示成功

接口调用示例

请求报文


{
    "currencyConfigList": [
        {
            "currencyCode": "USD",
            "applyCompanyType": "ALL",
            "currencyBizCode": "333",
            "periodRates": [
                {
                    "ratePeriod": "2025-01",
                    "exchangeRate": 7.033
                }
            ]
        }
    ]
}

响应报文


{  
    "body": "保存成功",    
    "returnCode": "SUC0000",    
    "success": true,    
    "message": "成功"    
}

错误码说明

错误码错误信息解决方案
E0002币种代码不能为空输入币种代码
E0002币种编码不能为空输入编码代码
E0002汇率周期不能为空输入汇率代码
E0002汇率不能为空输入汇率代码
E0002币种代码不可超过50位缩减币种代码
E0002币种编码不可超过50位缩减币种编码
E0002不支持的币种代码：XXX输入正确的币种代码
E0002不合法的汇率周期输入正确的汇率周期
E0002不合法的适用公司类型输入正确的适用公司类型
