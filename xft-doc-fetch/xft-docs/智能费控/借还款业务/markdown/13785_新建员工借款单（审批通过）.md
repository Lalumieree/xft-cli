# 新建员工借款单（审批通过）

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/emp-lend/bill/create 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/emp-lend/bill/create 
 ## 接口描述 
支持调用此接口新增审批通过的员工借款单
