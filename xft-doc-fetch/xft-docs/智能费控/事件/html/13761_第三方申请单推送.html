第三方申请单推送

基本信息

事件名称: 第三方申请单推送

事件编号: ITRIPBILL

事件描述

费控差旅申请单推送第三方消费平台事件

触发情况

企业在费控开通消费平台后，在设置规则页面配置推送单据类型及状态，当企业下用户所提单据满足配置的单据类型和状态条件时，触发该事件。

事件报文说明

**参数****类型****必填****长度****说明****备注**
applyEmpNameString是申请人姓名
applyEmpNumberString是申请人花名册编号-
billComIdString是-单据公司编号
billComNameString是-单据公司名称
billDepIdString是单据部门编号
billDepNameString是单据部门名称-
billIdString是-单据编号
billRemarkString否-单据备注
billStatusstring是单据状态待审批：1，审批通过：2， 拒绝：4，已撤回：11，已退回：12，已废弃：15，变更中 ：19，
billTypestring是单据类型差旅申请单：TRAVEL_APPLYTRAVEL_APPLY
billProjectIdString否-单据项目编号
billProjectNameString否-单据项目名称
billApplyTimeString是-单据提交时间yyyy-MM-dd hh:mm:ss
businessTripsArray:BillBusinessTripPushDTO是-行程列表
&ensp;&ensp;∟&departCityCodeString是-出发地城市编码
&ensp;&ensp;∟&departCityNameString是-出发地名称
&ensp;&ensp;∟&arriveCityCodeString是-目的地城市编码
&ensp;&ensp;∟&arriveCityNameString是-目的地名称
&ensp;&ensp;∟&departTimeString是-出发时间yyyy-MM-dd hh:mm:ss
&ensp;&ensp;∟&arriveTimeString是-到达时间yyyy-MM-dd hh:mm:ss
&ensp;&ensp;∟&roundTripString是-目的地名称单程：N 往返：Y
&ensp;&ensp;∟&startTimePrecisionString是-开始时间精度值天：DAY，上：AM，下午：PM，时分：MINUTE
&ensp;&ensp;∟&endTimePrecisionString是-结束时间精度值天：DAY，上：AM，下午：PM，时分：MINUTE
peersArray:BillBusinessTripPeerPushDTO否-同行人列表
&ensp;&ensp;∟&empNumberString否-同行人花名册编号
&ensp;&ensp;∟&empNameString否-同行人姓名
