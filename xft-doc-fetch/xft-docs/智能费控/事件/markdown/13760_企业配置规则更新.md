# 企业配置规则更新	

## 基本信息 
 
 事件名称: 企业配置规则更新	 
 事件编号: ITRIPCRU  
## 事件描述 
 企业在费控开通消费平台后，编辑消费平台设置规则，规则内容以事件形式发出，事件内容包括配置推送到第三方平台的单据类型编号和单据推送状态。第三方平台可按需订阅事件，获取需要的信息。 
## 触发情况
 消费平台与费控采用接口方式对接，企业在费控开通消费平台后，编辑设置规则并保存，即会触发改事件。

## 事件报文说明 
  | **参数**                         | **类型**                          | **必填** | **长度** | **说明**     | **备注**            |
| -------------------------------- | --------------------------------- | -------- | -------- | ------------ | ------------------- |
| eventInfs                           | Array：String                            | 否        |        |推送单据类型编号     |       |
| pushSts| String                             | 否        |          | 推送单据状态 | 提交后推送：1，审批通过后推送2 |
