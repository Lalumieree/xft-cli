# 第三方申请单推送

## 基本信息 
 
 事件名称: 第三方申请单推送 
 事件编号: ITRIPBILL  
## 事件描述 
 费控差旅申请单推送第三方消费平台事件 
## 触发情况
 企业在费控开通消费平台后，在设置规则页面配置推送单据类型及状态，当企业下用户所提单据满足配置的单据类型和状态条件时，触发该事件。
