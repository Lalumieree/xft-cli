# 消费平台开通申请事件

## 基本信息 
 
 事件名称: 消费平台开通申请事件 
 事件编号: ITRIPBIND  
## 事件描述 
 企业开通消费平台申请事件，消费平台接订阅该事件，用于校验开通信息的正确性，并回调消费平台开通结果回调接口告知费控开通结果 
## 触发情况
 企业在费控点击开通消费平台，并输入开通信息点击保存后，将向对应消费平台发送该事件
