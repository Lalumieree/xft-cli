新建日常报销单（草稿、审批通过、审批中）

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/reimburse/bill/create

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/reimburse/bill/create

接口描述
