# 新建差旅报销单（草稿、审批通过、审批中）

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/reimburse/bill/travel/create 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/reimburse/bill/travel/create 
 ## 接口描述 
支持调用此接口创建草稿、审批中、审批通过的差旅报销单

自定义字段传值格式和说明如下：
