适用用户差标政策查询

接口调用地址

生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/policy/getTravelPolicies

测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/policy/getTravelPolicies

接口描述

根据用户和表单信息查询当前表单下的差标政策

调用方式

METHOD: POST

CONTENT-TYPE: application/json

接口输入参数

参数类型必填说明示例
headerOBJECT请求头根节点
参数类型必填说明示例
bodyOBJECT请求体根节点
body.aplIdSTRING是申请人IDU0000
body.eventSeqIdsARRAY否单据类型自增ID集合
body.expSeqIdsARRAY否费用类型自增ID集合
参数类型必填说明示例
queryOBJECTquery根节点
query.OPAUIDSTRING公司标识XFT00001

接口输出参数

参数类型必填说明示例
headerOBJECT响应头根节点
参数类型必填说明示例
bodyOBJECT响应体根节点
body.dataARRAY否差标数据
body.data.travelPolicySeqIdLONG否政策自增ID6063
body.data.comIdDOUBLE否公司ID0
body.data.plcNamSTRING否政策名称滚动预算测试政策
body.data.billSeqIdListARRAY否适用单据类型自增Id
body.data.expSeqIdLONG否适用费用自增Id19218
body.data.expNamSTRING否适用费用名称null
body.data.useTypLONG否* 0-费用
* 1-补贴		0
body.data.cityStsSTRING否城市级别自增IdY
body.data.forecTypLONG否* 1-警示提醒
* 2-不得提交		1
body.data.plcCfgSTRING否政策描述 JSON字符串{"txt": "可以住宿500元 / 间夜（ 含） 以下的酒店","fee":"500"}
body.data.plcTypSTRING否* "000" "自定义政策"
* "001" "住宿政策"
* "003" "机票政策"
* "004" "其他政策（按次）"
* "005" "轮船政策"
* "006" "火车政策"
* "007" "其他政策（按天）"
* "008" "用餐政策"		001
body.data.ssdMnySTRING否补贴金额null
body.data.ssdCclTypSTRING否* 次-"1"
* 半天- "2"
* 全天- "3"
* 自定义-"C"
* 里程补贴-"L"
* 时长-"R"
* 时刻-"M"		null
body.data.autoCheckFlgSTRING否* Y - 自动勾选
* N- 不自动勾选		null
body.data.plcGroupSeqIdLONG否绑定的差标等级自增Id18547
body.data.maxTimesConfigStateLONG否* "0" 关
* “1” 开		0
body.data.maxTimesLONG否最大次数1
body.data.awardTypeSTRING否* 1-差额奖励
* 2-折扣奖励		1
body.data.awardCheckFlgBOOLEAN否奖励勾选标志false
body.data.awardRatioSTRING否奖励比例null
body.successBOOLEAN否成功标志位true
body.messageSTRING否返回数据调用成功
