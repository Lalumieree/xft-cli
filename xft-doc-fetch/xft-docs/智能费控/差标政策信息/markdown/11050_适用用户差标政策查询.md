#  适用用户差标政策查询

## 接口调用地址 
 生产环境地址：https://api.cmbchina.com/itrip/xft-api/v1/policy/getTravelPolicies 
 测试环境地址：https://api.cmburl.cn:8065/itrip/xft-api/uat/v1/policy/getTravelPolicies 
 ## 接口描述 
根据用户和表单信息查询当前表单下的差标政策 
## 调用方式 
 METHOD: POST  
 CONTENT-TYPE: application/json
