# 发票有效状态 invoiceCheckStatus
| **枚举名称** | 编码 | **描述** |
|----------|----------|----------|
| VALID | A | 有效 |
| INVALID | B | 作废 |
| REVERSE | C | 已红冲 |
| REVERSE_PART | D | 已部分红冲 |
| REVERSE_WAIT_CHECK | E | 红字申请待确认 |

| **枚举** | 编码 | **描述** |
|----------|----------|----------|
| NOT_CHECK | A | 未查验 |
| REAL_CHECK | T | 真票 |
| FAKE_CHECK | E | 异常 |
| NOT_NEED_CHECK | N | 不需查验 |
| NOW_NOT_CHECK | U | 暂不查验 |
| CANNOT_CHECK | C | 无法验真 |
